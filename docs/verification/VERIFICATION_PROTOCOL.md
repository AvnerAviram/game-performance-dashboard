## ACCEPTANCE CRITERIA

### ✅ VERIFIED (100% confidence)
- All 3 sources found
- All 3 sources agree on primary mechanic
- Core features match across all sources
- Minor terminology differences acceptable if meaning is same

### ⚠️ PARTIALLY VERIFIED (needs review)
- Only 2 sources found
- 2 sources agree, 1 differs
- Core features match but details vary

### ❌ UNVERIFIED (requires manual research)
- Only 1 source found
- Sources contradict each other
- Game not found on any standard source

## RULES TO PREVENT ERRORS

### 1. EXACT NAME MATCHING
- Always verify provider name matches
- Check for similar game names (e.g., "10x Cash" vs "10x Fortune")
- If unsure, mark as "needs review"

### 2. NO ASSUMPTIONS
- If feature is unclear, mark as "unverified" not "probably X"
- If source says "and more", don't guess what "more" means
- If terminology is different, document both versions

### 3. CROSS-REFERENCE EVERYTHING
- Don't accept single-source data
- If sources conflict, research deeper OR mark as "needs review"
- Document discrepancies explicitly

### 4. TERMINOLOGY STANDARDIZATION
- "Cash Collect" = "Money Collect" = same feature
- "Lock & Spin" = "Lock N Spin" = same feature
- "Cascading" = "Avalanche" = same feature
- Document variations but use consistent naming

### 5. WHEN IN DOUBT
- Mark as "needs review"
- Don't proceed with uncertain data
- Flag for manual verification
