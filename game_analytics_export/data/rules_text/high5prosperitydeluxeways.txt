SCATTER LOOT LINK FEATURE The Scatter Loot Link feature is initiated by landing 5 or more Scatter Symbols during a base game spin. These symbols are only awarded at the end of the feature. If the feature is not triggered due to landing less than 5 of these symbols, these symbols are not awarded. The matrix size of the Scatter Loot Link feature is the same as the matrix size of when the feature was initiated. The player has 3 spins to land at least one Scatter Symbol, Super Loot Link Symbol, Mini Jackpot Symbol, Minor Jackpot Symbol, Major Jackpot Symbol, or Grand Jackpot Symbol on an active empty position or else the Scatter Loot Link ends. Any number of new Scatter Symbols, Super Loot Link Symbol, Mini Jackpot Symbol, Minor Jackpot Symbol, Major Jackpot Symbol, or Grand Jackpot Symbols will reset the spin counter to 3 total Scatter Loot Link spins remaining. Once the Scatter Loot Link feature begins, the triggering Scatter Symbols become one of the following scatter types: Teal Scatter Symbol, Blue Scatter Symbol, Green Scatter Symbol, or Purple Scatter Symbol, which are associated with one of 4 multiplier meters. All subsequent landing Scatter Symbols also become one of Teal Scatter Symbol, Blue Scatter Symbol, Green Scatter Symbol, or Purple Scatter Symbol. Matching 6 of the same type of scatter symbol will unlock a multiplier meter. At the end of the Scatter Loot Link feature, any unlocked multiplier associated with each type of scatter will be applied to all scatters of that type. If at any point during the Scatter Loot Link feature, 3 Super Loot Link Symbol symbols are on the reels, the Super Scatter Loot Link feature will trigger. All subsequent spins are played on a 10x6 independent reel matrix with all Scatter Symbols, Mini Jackpot Symbol, Minor Jackpot Symbol, Major Jackpot Symbol, or Grand Jackpot Symbol symbols are carried over. In the Super Scatter Loot Link feature, a new scatter symbol type, [WY1X5], with a corresponding multiplier meter is introduced. Any new Scatter Symbols become one of 5 colors, which are associated with one of 5 multiplier meters. When the feature ends the combined values on all symbols and jackpots will be awarded. When unlocked, the 2x Multiplier Meter awards a 2x multiplier. When unlocked, the 3x Multiplier Meter awards a 3x multiplier. When unlocked, the 5x Multiplier Meter awards a 5x multiplier. When unlocked, the 10x Multiplier Meter awards a 10x multiplier. When unlocked, the 25x Multiplier Meter awards a 25x multiplier.
