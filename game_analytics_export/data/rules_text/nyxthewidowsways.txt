The Widow's Ways RTP: 96.0% VOLATILITY: 14.6 MEDIUM GAME RULES This game is played at 1024 ways. All 5 reels are always in play. Ways wins contain one symbol from each adjacent reel, beginning with the leftmost reel. The same symbol, or its substitute, in a different position on the same reel pays that ways win again. Only the highest win is paid for each ways symbol combination. Ways wins are multiplied by the bet multiplier. Scatter symbols award in any position on the reels. Only the highest win is paid for each scatter symbol combination. Wins are shown in credits unless marked as currency. Values displayed in the Game Rules have not been pre-multiplied, except where stated. Wins subject to verification. Malfunction voids all pays and plays. FREE GAMES BONUS During the base game, 3 Bonus Symbols appearing in any position on reels 2, 3, and 4 starts the Free Games Bonus. Awards 9 free games. Bonus Symbols are scatter symbols. Bonus Symbols appears only on reels 2, 3, and 4. The Free Games Bonus uses different reels. Free games automatically play at the same bet multiplier as the game that initiated the Free Games Bonus. The Free Games Bonus does not award additional free games. The Free Games Bonus ends when 9 free games have been played. Free Games Wild Symbol is wild and substitutes for all symbols. Free Games Wild Symbol appears only on reels 3, 4, and 5. Free Games Wild Symbol expands left and substitutes for all symbols in the same position on each reel from the initial Free Games Expanding Wild Symbol, up to and including reel 2. Wins are evaluated after Free Games Expanding Wild Symbol expands. Last modification date: 10/18/2022
