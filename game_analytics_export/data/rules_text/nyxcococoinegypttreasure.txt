Co Co Coin Egyptian Treasure Game Description CO CO COIN EGYPTIAN TREASURE™, wild keys of fortune possess the power to unlock golden riches. Jackpot, bonus, and credit prizes on gold coins are won when appearing in any position on 3 or more consecutive reels beginning with the leftmost reel. More coins appear on the first three reels during free games, and even more coins appear on all reels in a respin that is triggered randomly when wild appears during bought games. This game has 5 reels, 3 rows and 243 ways to win (EVERY WAY). HOW TO PLAY Select your bet. Press the Spin button to start a game cycle. METERS AND BUTTONS Bet: Displays the current bet. Bet is determined by multiplication of the current coin value with current minimum bet. Coin Value: The coin value determines your bet on the spin. Spin Button: Starts the game. When the reels are spinning, the spin button transforms into the Stop Button. Press the Stop Button to stop the reels immediately. Max: Displays the maximum bet. Autoplay (wherever applicable): Player can set the game to spin without the need to press the spin button every time. You can choose the number of spins to play automatically by selecting through the various options available in the menu. The loss limit (wherever applicable) will avert you from losing above set limit during autoplay session. Autoplay can be manually stopped by pressing Stop Autoplay button. Win: Displays the win for the current or last win paid. Credits: Displays the credit amount of the player. All credit, bet and win meters would be shown in currency.FREE GAMES FEATURE Any 3 or more [FEATURE] trigger the FREE GAMES FEATURE awarding 10 free games. Different reels are used during the free games. Reels 1, 2 and 3 contain extra [BONUS] during the FREE GAMES FEATURE compared to bought game. FREE GAMES FEATURE can be triggered again during the FREE GAMES FEATURE. If the FREE GAMES FEATURE and TREASURE RESPIN are triggered in the same bought spin, the FREE GAMES FEATURE starts after the TREASURE RESPIN. If the FREE GAMES FEATURE is then triggered again in the TREASURE RESPIN, the sum of free games awarded for the two triggered FREE GAMES FEATURE are played in a single FREE GAMES FEATURE that starts after the TREASURE RESPIN ends. All bets and ways played during the FREE GAMES FEATURE are the same as the game that triggered the FREE GAMES FEATURE. TREASURE RESPIN There is a chance that a TREASURE RESPIN is randomly triggered when 1 or more [WILD] appear in bought spins only. After all wins are determined in the bought spin that triggers the TREASURE RESPIN, all reels respin automatically in one free spin and any new wins are added. Different reels with extra [BONUS] on reels 1, 2, 3, 4 and 5 are used in the TREASURE RESPIN. TREASURE RESPIN cannot be triggered again in the TREASURE RESPIN. All bets and ways played during the TREASURE RESPIN are the same as the game that triggered the TREASURE RESPIN.JACKPOTS This game has 2 levels of fixed jackpots [GRAND] and [MEGA]. Jackpot can be won during base game and free game. Jackpot fixed prizes: GRAND JACKPOT 250,000 MEGA JACKPOT 50,000 Currency unit and amount vary according to jurisdiction. Jackpot wins will be multiplied by the coin value. GAMBLE FEATURE To enter the gamble game after a win, press the GAMBLE button. Select RED/BLACK or a SUIT to gamble the total win. Win is quadrupled if suit selection is correct. Win is doubled if red/black selection is correct. Gamble feature is not available for any feature win. Wins may be gambled up to 5 times. Gamble will not be offered for the wins awarded during the feature trigger. Gamble will not be offered during Autoplay. All wins cannot be gambled if the Jackpot is triggered. GAME RULES This game requires a minimum of 50 credits multiplied by coin value to play. All wins on consecutive reels beginning with the leftmost reel except scatter. Each symbol appearing on a reel is counted once per winning combination. Coinciding wins are added. [EVERYWAY] wins are multiplied by the coin value. [Scatter] wins are added to [EVERY WAY] wins. All wins are multiplied by the coin value except [scatter]. [scatter] wins are multiplied by the total bet. [scatter] and [BONUS] wins are added to [EVERY WAY] wins. Only highest win is paid on each line. Malfunction voids all pays and plays. All open game rounds (pending state) will be automatically settled after 30 days.[EVERY WAY] [EVERY WAY] provides for 243 ways to win. [EVERY WAY] wins are winning combinations that appear in any position on consecutive reels beginning with the leftmost reel, except [BONUS] and [SCATTER]. Each symbol appearing on a reel is counted once per winning combination. AUTOPLAY SETTINGS Number of spins: Select number of Auto spins as per available choices: 5,10,20,30,40,50. Autoplay Advanced Settings (wherever applicable): Loss Limit: Stops autoplay spins if selected limit reached. Win Limit: Stops autoplay spins if selected limit reached. Single win limit: Stops autoplay spins if selected limit reached. Stop on any win: Stops on any wins. NET POSITION Net position (wherever applicable): The Net position will display customer’s net position, in the currency of their account or product (e.g., pounds sterling, dollar, Euro) since the session started. Net position is defined as the total of all winnings minus the sum of all losses since the start of the session. REALITY CHECK Reality check (wherever applicable): The Reality check feature may display as a periodic in-game notifications for the real money players. Game gets suspended whenever the check appears during game play and only resumes once the player has acknowledged the check and has taken displayed actions. “Back to game”:Notifying player to resume the game. “To Lobby”:Quits the current game (after a final prompt) and back to the Operators Main Game Menu Screen. MALFUNCTION In case of any malfunction of hardware or software during the game play, all current game bets and payouts are considered void and all bets affected are refunded.
