Africa X Up About this Game l This is a 5 reel 3 row video slot game. l The objective of the game is to spin the reels so the symbols land in different combinations. l Winning combinations are highlighted on the reels and a payout is awarded. l This game includes 243 ways. X UP Feature l 2 landed Scatter symbols award an X UP token. No token is awarded for 3 landed Scatter symbols. l Awarded X UP tokens are collected and highlighted in the X UP Multiplier Trail device. l Greater Free Spins win multipliers are activated as more X UP tokens are collected. l When Free Spins is started, any active Free Spins win multiplier will be in effect from the beginning of the feature. l When Free Spins is started any collected X UP tokens will be carried into the feature. l Collecting 1 X UP token will activate the 2x Free Spins Multiplier. l Collecting 3 X UP tokens will activate the 3x Free Spins Multiplier. l Collecting 5 X UP tokens will activate the 4x Free Spins Multiplier. l Collecting 7 X UP tokens will activate the 5x Free Spins Multiplier. l Collecting 10 X UP tokens will activate the 7x Free Spins Multiplier. l Collecting 13 X UP tokens will activate the 10x Free Spins Multiplier. l Collecting 18 X UP tokens will activate the 15x Free Spins Multiplier. l Collecting 23 X UP tokens will activate the 25x Free Spins Multiplier and start the Free Spins Feature. l Collected X UP tokens and any activated Free Spins Multipliers are stored and used per bet amount. l Collected X UP tokens are removed on completion of Free Spins for the current bet amount. Free Spins l 3 Scatter symbols anywhere on the reels 2, 3, and 4 trigger 8 Free Spins. l Any activated X UP Free Spins win multiplier will be in effect from the start of the game. l During Free Spins, landing 2 Scatters will award 2 additional Free Spins and an X UP token. l Additional X UP tokens can be collected during Free Spins which can increase the active X UP Free Spins win multiplier. l All bets played are the same as the spin that triggered this feature. l All ways played are the same as the spin that triggered this feature. l A maximum of 25x X UP Free Spins win multiplier can be achieved. l Wins awarded during the feature are added to base game wins. l Free Spins cannot be retriggered. l Free Spins play out automatically. Bet Rules l Each spin played costs a total bet amount. The initial bet is the current total bet divided by the way cost of 30. Way Rules l This game includes 243 ways. l A way is a pattern of adjacent symbol positions that runs across the reels. l A win is awarded when matching symbols land adjacent to each other in this pattern. l Way wins run from left to right, starting from the first slot reel. Payout Rules l Way win payouts are based on the initial bet placed. l The total payout is the sum of all way wins and Scatter wins resulting from a spin. l Only the highest winning combination is awarded per symbol combination. l The win up to value displayed is rounded down, and is approximately the highest possible win achievable on a single spin. l The Paytable displays the payout for each symbol combination based on the current bet. l If multiple winning combinations land on a single way, only the highest win is awarded. Wild Symbols l Wild symbols can substitute for other symbols to complete winning combinations. l Wild symbols expand to fill the reel. l Wild symbols can substitute for all symbols except Scatters. l Wild symbols can create their own winning combination when landing in a way combination. Scatter Symbols l In base game this symbol appears on reels 2, 3 and 4 only. l In free spins this symbol appears on reels 2 and 4 only. l Scatters can create their own winning combination when landing anywhere on the reels. l Scatter wins are multiplied by the total bet.
