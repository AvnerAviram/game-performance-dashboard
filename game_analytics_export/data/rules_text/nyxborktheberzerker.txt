Bork The Berzerker Hack N Slash Edition Game Rules Note: The Børk the Berzerker ® video slot is referred to as Børk the Berzerker ®The Game or Game. Børk the Berzerker® is a video slot with 5 reels and 25 fixed pay lines. The game features expanding wilds and bonus game. The theoretical return to player is 97,3%. The game offers up to 15 bet levels for the selected currency. A game round is started with the action button and plays the game with the selected bet level. Auto play plays the game automatically for the number of game rounds selected in the auto play page. All game payout and win combinations are paid out according to the pay table. All pay line wins during a game round are added and presented at the end of the game round. All wins are presented and paid out in the selected currency. Only the highest win per pay line is paid out, if in sequence from far left to right in combinations according to the pay table. The bonus symbol is a scatter symbol that grants the bonus game if 3 or more bonus symbols are won. The number of bonus symbols won correlates to the number of free spins granted in the bonus game according to the pay table. The expanding wild is a wild symbol that expands vertically if included in a pay line win. The expanding wild only appears on reels 2, 3 and 4 and substitutes for all symbols except the bonus symbol. The bonus game is played with the same bet as the spin that triggered the bonus game. No bets can be altered during a game round. In the event of game malfunction all affected bets and pays are discharged. Last modification date: 3/26/2018
