Book of Sheba 1. Overview The goal is to obtain a winning combination on a winning line spread across the reels. Game specifications: Game specifications: Type Slots Number of reels 5 Number of rows 3 Number of lines 10 (cannot be changed) Number of coins you can bet 1- per line (cannot not be changed) Progressive No Double feature No Auto spin feature Yes Wild symbols Yes Multipliers symbols No Scatter symbols Yes (Wild) Bonus Round No Free s pins Yes Instant win No RTP 96.00% The game disconnects if you do not take any actions for a specific number of consecutive minutes (this number varies depending on jurisdiction); the session recovery feature will save the game progress so that the next time the game is loaded, you can continue where you left off. In the event of technical difficulties resulting in a dropped connection to the game server, the game you were playing is saved automatically and will load the next time you establish a connection to the game server. You will not be able to reconnect to the game server for the first 30 seconds after the disconnection. Note : In case of malfunction all pays and play are voided; any bet that was not yet accepted will be voided, any bet that was unsettled will be refunded. The maximum win from a single game is £250,000 2. How to Play Note : If you do not have enough money to play a game, the bet can be set to the minimum selectable amount through a dialog window informing about this. All winning lines pay from left to right. Wins on different lines are all added. Only the highest winning combination on each line is paid. Winnings from Free Spins are added to the total win. The auto spin feature does not automatically start free spins - player input is required. Book symbol Landing 3 or more Book symbols anywhere in view will trigger Free Spins. Book symbols are also Wild symbols and can replace any other symbols to form winning combinations. 3.2. Free Spins Landing 3 or more Book symbols anywhere in view will trigger 10 Free Spins. At the start of Free Spins, a special expanding symbol is randomly chosen to help you win more. Accept the first symbol to keep it, or Reject to receive a new symbol which must then be played. After initial pays, the special symbol can expand to form wins on all 10 lines, even on non-adjacent reels. Free Spins can be retriggered. 3.3. Golden Book symbol Golden Book symbols are available only on reels 2, 3 and 4 during Free Spins. If a Golden Book symbol lands on any of these it will transform the reel gold for the remainder of Free Spins. Golden Book symbols are scatters and also Wild symbols. When a reel is transformed gold it will transform any symbols on the reel into the special expanding symbol if a win with that chosen symbol is possible. Normal Book symbols are available in all the reels. Note : expanded reel= expanded symbol. fs_trigger= free spins. free_spins_triggered= free spins. bonus_triggered= bonus won. special_symbol= special symbol. lost= lost. won= won. 4. Auto spin feature The loss limit setting allows stopping the auto spin feature automatically when the selected limit, plus your cumulative winnings, and minus your cumulative bets, is less than 0. 4.1. Desktop version Click the AUTO SPIN button to open the AUTO SPIN settings window where you can set up the following: Number of auto spins: click + or - to select the number of auto spins to play. Loss limit: click + or - to select the value of the loss limit; Stop if a win reaches: click + or - to select the value of the maximum win; Stop if JACKPOT is won (only for games with Progressive Jackpots): to enable this setting, tick the checkbox; to disable it, remove the tick. Click the ACTIVATE button to close the AUTO SPIN settings window and to start the auto spin feature with the currently selected bet and auto spin setup. Click the button to close the window without starting the auto spin feature. Click the STOP AUTO SPIN button to stop the auto spin feature at any time. 4.2.
