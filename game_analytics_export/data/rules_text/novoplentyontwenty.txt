Plenty on Twenty Rules 7 substitutes for all symbols except scattered 'Star'. All pays are for wins of left to right except Scatters, which pay any. All prizes are on selected lines excluding Scatters. Highest win only paid per selected line. Line wins are multiplied by credits bet per line. Scatter prizes are multiplied by total bet. Scatter wins are added to line wins. Main Game You have 20 paylines and 5 reels at your disposal with Plenty on Twenty. The aim of the game is to get the same symbol on all 5 reels. All winning combinations run across the reels from left to right. The '7' symbol is a wildcard and can replace any symbol, apart from the 'Star' Scatter symbol. Game Theme and Graphics Let the cheeky fruits tumble into the right combinations and they will bring you a hefty harvest. That way you can pleasantly rake in big wins in a very short time. RTP The Return to player for this game is 95.61%. The maximum win from a single game is £250,000. Last modification date: 6/17/2020
