Super Fruits Wild WINNING LINES All wins pay left to right across adjacent reels Wins can occur across any or all of the win lines All games are played with 10 win lines FORTUNE SPINS Selecting the Fortune Spins button activates the Fortune Spins menu. Selecting PLAY will then activate Fortune Spins. The RETURN TO GAME button will deactivate the Fortune Spins menu. During Fortune Spins, all expanding wilds transfer across to subsequent reel sets. Each game of Fortune Spins is played across 4 sets of reels. This feature uses the selected stake split 4 ways across each set of reels. Therefore, the payouts for each set of reels will reflect a stake of 1/4 that of the selected stake when a winning combination lands. MAIN GAME CONTROLS Button Function SPIN: Press to start playing at the current stake value. AUTOPLAY: In the Autoplay options panel, select the number of autoplay and loss limit amounts. Autoplay countdown is automatic. STAKE: Pressing once will open stake window for you to adjust your stake amount. FORTUNE SPINS: Press to enable Fortune Spins mode on/off. SETTINGS: Press to access game settings and game information. SOUND: Press to enable sound on/off. GAME HISTORY: Press to see your play history. TURBO: Speeds up reel spins. Press to enable Turbo mode on/off. SLAM STOP: During a regular spin, pressing the SPIN button whilst the reels are spinning will cause them to land quicker. This does not impact the outcome of the game. GENERAL INFORMATION GAME RECOVERY In the event of an unexpected interruption / termination of a real play game, the game will recover to the last known state. PAYBACK INFORMATION Your odds of getting any particular outcome are always the same. This reflects the theoretical return across a very large number of plays over an extended period of time. This game has been independently fairness tested. The expected payback for this game is 94.98-94.99%. Malfunction voids all pays and plays. Last modification date: 2/25/2021
