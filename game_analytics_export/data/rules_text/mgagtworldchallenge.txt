GT World Challenge by Andy Soucek Untitled Document PRIZE PLAN MAIN GAME 10 Fixed bet lines. Bet line wins pay when in succession from both leftmost to rigth and rightmost to left WILD REEL + EXTRA SPIN Andy Soucek wild symbol substitute all symbols except bonus symbol. One or more Andy Soucek wild symbol appering anywhere on reels 2,3 or 4. Activates the extra spins to a maximum of 3 re-spins. BONUS GAME MINIGAME SELECTION Three 'Bonus' figures, or more, give access to the mini-game selection screen. Several entries to the mini-game selection screen will unlock, one after the other, accesses to bonus games. GRID BONUS GAME If the first game is chosen, the grid game will start. We have six cars to select, each car have a prize. The game ends when the exit symbols appears or the six cars have been chosen. RACE BONUS GAME If the second game is chosen, the race game will start. There are prizes hidden, you have to choose one of them. The player chooses a possibility by obtaining the prize shown. BOXES BONUS GAME If the third mini-game is chosen, the boxes game will start. It's a raffle among 5 prizes. The player has to choose the best prize by stopping the selector. The final prize will be the corresponding prize that is illuminated. PODIUM BONUS GAME If the fourth minigame is chosen, the podium game will start. A raffle start between prizes or up arrow. If the raffle stops in an up arrow goes to the upper level. If stops in a prize the game ends. PRIZE LINES Last modification date: 8/26/2020
