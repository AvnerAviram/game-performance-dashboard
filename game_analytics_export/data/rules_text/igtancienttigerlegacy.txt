Ancient Tiger's Legacy Ancient Tiger's Legacy Game Rules (1.0 - 04-July-2025) 1. Overview The goal is to obtain a winning combination on a winning line spread across the reels. Game specifications : Type Slots Number of reels 3 Number of rows 3 Number of lines 5 (cannot be changed) Number of coins you can bet 10 (cannot be changed) Progressive No Double feature No Auto spin feature Yes Wild symbols No Multiplier symbols No Scatter symbols No Bonus round No Bonus s pins No Instant win No RTP 94.10 % The game disconnects if you do not take any actions for a specific number of consecutive minutes (this number varies depending on jurisdiction); the session recovery feature will save the game progress so that the next time the game is loaded, you can continue where you left off. In the event of technical difficulties resulting in a dropped connection to the game server, the game you were playing is saved automatically and will load the next time you establish a connection to the game server. You will not be able to reconnect to the game server for the first 30 seconds after the disconnection. Note : In case of malfunction all pays and play are voided; any bet that was not yet accepted will be voided, any bet that was unsettled will be refunded. 2. How to Play Note: If you do not have enough money to play a game, the bet can be set to the minimum selectable amount through a dialog window informing about this. All winning lines pay from left to right. Wins on different lines are all added. Only the highest winning combination on each line is paid. On the info bar, you can view the current BET, your BALANCE and your current WIN, displayed in either credit or your real currency, depending on the current user's choice (Available only for specific jurisdictions). To toggle between credit or your real currency, click in the BALANCE area (Available only for specific jurisdictions). 2.1. Desktop version To play, follow these steps: Click on the main screen, and then use the + and - buttons to select a coin value. Click to save the settings and return to the main screen. Note: The total BET is displayed in either credit or your real currency, depending on the current user's choice (Available only for specific jurisdictions). To toggle between credit or your real currency, click in the BALANCE area (Available only for specific jurisdictions). Click to place the bet and spin the reels one time. The total amount won will be shown in the WIN field from the info bar, in either credit or your real currency, depending on the current user's choice (Available only for specific jurisdictions). For each winning combination, the amount won is the amount shown in the paytable for that winning combination. To toggle between credit or your real currency, click in the BALANCE area (Available only for specific jurisdictions). 2.2. Auto spin feature The loss limit setting allows stopping the auto spin feature automatically when the selected limit, plus your cumulative winnings, and minus your cumulative bets, is less than 0. 5.1. Desktop version Click the AUTO SPIN button to open the AUTO SPIN settings window where you can set up the following: Number of auto spins : click + or - to select the number of auto spins to play. Loss limit : click + or - to select the value of the loss limit; Stop if a win reaches : click + or - to select the value of the maximum win; Stop if JACKPOT is won (only for games with Progressive Jackpots): to enable this setting, tick the checkbox ; to disable it, remove the tick. Click the ACTIVATE button to close the AUTO SPIN settings window and to start the auto spin feature with the currently selected bet and auto spin setup. Click the CLOSE button to close the window without starting the auto spin feature. Click the STOP AUTO SPIN button to stop the auto spin feature at any time. 5.2. Mobile version Tap the AUTO SPIN button to open the AUTO SPIN settings window where you can set up the following: Number of auto spins : tap + or - to select the number of auto spins to play.
