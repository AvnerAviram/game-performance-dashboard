Creepy Christmas Basic Rules Creepy Christmas is a 5x4 video Slot played on 40 paylines. Wins are formed by landing identical symbols on adjacent reels starting from left to right. All wins are shown in currency. Malfunction voids all pays and plays. Only the highest win is paid per winning combination. Wins on multiple lines are added together. Special reels are in play during the Free Games Feature. Wild substitutes for all symbols except Scatters. Lady Santa wild symbol appears only on reel 3 and acts as a wild in the hot zone. HOT ZONE 2 Symbol placed on the middle of reel 3 highlighted by a special framed Hot Zone. Lady Santa's Skulls feature is awarded, when the Lady Santa Wild symbol lands inside the Hot Zone. Free Spins feature is awarded, when 1 snow globe Scatter lands inside the Hot Zone. Hot Zone is present during both base game and Free Spins. Quick start Game starts by setting a selected Bet amount and by pressing the Spin button. Round ends when reels stop and there are no wins, or after the win presentation if there are wins. Turbo mode (where available) Turbo mode is speeding up reels spinning animation when enabled. This option does not influence the game round result but only visual presentation. Game menu Game menu contains options and settings that control some of the game features: History - Displays previous rounds from the current session Close Game - Closes the game client Sound - Controls the Sound FXs in the game (on/off) Music - Controls the Music in the game (on/off) Turbo mode (where available) - Controls the Turbo mode feature (on/off) Game is paused while the menu is visible and can be closed by pressing the Close button or anywhere on the screen. Bet selection Game starts with a default Bet amount set for the game. Player can change the Bet amount by either: Using Up and Down arrow buttons next to the Bet Amount Pressing the Bet amount where Bet Amount selector will open up Bet amount cannot be changed during a game round and if there is a win presentation happening, changing bet Amount will reset the presentation. Autoplay mode (where available) Autoplay mode enables automatic play of game. It can be enabled or disabled at any time during the Base game. Autoplay mode is enabled by pressing the Autoplay button which allows the player to select: Number of rounds to be played automatically. Loss Limit (where available) - Disables the Autoplay mode if Balance is reduced by more than Loss Limit amount. Stop on Bonus Feature (where available) - Disables the Autoplay mode when Bonus game or Free games are triggered. Stop on Win above (where available)- Disables the Autoplay mode when there is a single round win exceeding the amount entered. Spin button will display the remaining number of spins to be played automatically. Pressing on the Autoplay button while Autoplay mode is enabled will reset the number of Rounds and disable the Autoplay mode. Malfunction A malfunction of the game voids all pays. In case the game round is interrupted, it can be restored by simply starting the game again. If it is not restored, the round is voided within the jurisdictional requirements or operator's policy and their terms and conditions. Malfunction voids all pays and plays.
