Tumbling Together RTP: 94.9% Game Rules Malfunction voids all pays and play This game is played at 243 ways and all 5 reels are always in play. Select Bet Multiplier Click the Spin button to spin the reels Payouts are made according to the Paytable amounts for the Credits stated Highest win is paid for each ways symbol combination. All 5 reels are always in play. Ways wins contain one symbol from each adjacent reel, beginning with the leftmost reel, or ending with the rightmost reel. The same symbol, or its substitute, in a different position on the same reel pays that ways win again. Only the highest win is paid for each ways symbol combination. Ways wins are multiplied by the bet multiplier. Bonus Symbol only appears on reels 2, 3, and 4. Bonus Symbol is also a Scatter Symbol. Scatter symbols award in any position on the reels. Only the highest win is paid for each scatter symbol combination. Wilds symbols substitute for all symbols except the Free Games Bonus symbol Wild symbols only appear on reels 2, 3, and 4. All symbols in winning combinations disappear. Reels continue to tumble until no winning symbol combinations are created as a result of the tumbling reels. The empty symbol positions are filled as symbols tumble from above. Afterwards, symbols are re-evaluated for new winning combinations. Free Games bonus Awards 10 free games.
