Delorean Lucky Tap User Interface ‘+’ / ‘-’ buttons increase / decrease the bet value — the bet value is the total bet per game round. Starts a game round at the current bet value. Displays the autoplay options. Stops the autoplay session. Starts the autoplay session. The numerical value displayed within the button shows the number of autoplay game rounds remaining Bet Deﬁnitions Current Bet: The current bet is the value that appears above . Minimum Bet: The minimum bet is the lowest possible bet value, which is selected by pressing until the bet value no longer decreases. Bet Multiplier: The bet multiplier is defined as the current bet divided by the minimum bet. Autoplay Options Autoplay will allow game rounds to be played automatically. Click on the Autoplay button and select the number of Autoplay game rounds. Additional Information The overall return to player is 93.82%. Malfunction voids all pays and plays. Multi plier Feature If 10 or fewer prizes are awarded on any winning play, the Multiplier Feature may randomly be activated. If the Multi plier Feature is activated, a Random Multiplier of x2, x3, x4, x5, x6, or x7 will appear. All Currency Wins will be multi plied by the Random Multiplier. Wins A winning play occurs when the meter fills up completely during the game round. On a winning bet, players can be awarded 1 - 16 symbols which each award a credit prize, and/or a chance to trigger the Free Games feature if the symbol is collected. Free Games 10 Free Games are awarded if the symbol is awarded on a winning play. 10 additional Free Games can be retriggered if the symbol is awarded on a winning play during Free games. The Multiplier Feature is always active for all wins in Free Games. Free Games end when 0 Free Games remain or after 1000 Free Games have been played. The bet and pays during the Free Games feature are the same as the initiating base game. Last modification date: 5/23/2024
