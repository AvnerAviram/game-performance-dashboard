QUICK START GAME STARTS BY SETTING A SELECTED BET AMOUNT AND BY PRESSING THE SPIN BUTTON/AUTOPLAY. ROUND ENDS WHEN REELS STOP AND THERE ARE NO WINS, OR AFTER THE WIN PRESENTATION IF THERE ARE WINS. TURBO MODE TURBO MODE IS SPEEDING UP REELS SPINNING ANIMATION WHEN ENABLED. THIS OPTION DOES NOT INFLUENCE THE GAME ROUND RESULT BUT ONLY VISUAL PRESENTATION. GAME MENU GAME MENU CONTAINS THE FOLLOWING OPTION(S): LOBBY – CLOSES THE GAME AND RETURNS TO LOBBY. SOUND- CONTROLS THE SOUND IN THE GAME (ON/OFF). TURBO MODE- CONTROLS THE TURBO MODE FEATURE ( ON/OFF). GAME RECALL – DISPLAYS PREVIOUS ROUNDS FROM THE CURRENT SESSION. HELP/PAY – DISPLAYS THE PAYTABLE AND GAME RULES. GAME MENU CAN BE CLOSED BY PRESSING ANYWHERE ON THE GAME SCREEN. AUTOPLAY AUTOPLAY MODE ENABLES AUTOMATIC PLAY OF GAME. IT CAN BE ENABLED OR DISABLED AT ANY TIME DURING THE BASE GAME. AUTOPLAY MODE IS ENABLED BY PRESSING THE AUTOPLAY BUTTON WHICH ALLOWS THE PLAYER TO SELECT THE NUMBER OF ROUNDS TO BE PLAYED AUTOMATICALLY.
