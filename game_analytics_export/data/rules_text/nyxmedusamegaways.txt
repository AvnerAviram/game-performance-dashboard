The theoretical average return to player (RTP) is Standard: 96.286% BUY PASS: 97.630% MEGAWAYS™ The number of symbol positions appearing on each reel of the MAIN REEL ARRAY is determined randomly at the beginning of each Game Round. Reels 1 and 6 can contain 2, 3, 4, 5, 6 or 7 symbol positions each Game Round. Reels 2, 3, 4 and 5 can contain 2, 3, 4, 5 or 6 symbol positions each Game Round. Number of symbol positions is determined independently for each reel. Including EXTRA REEL, each Game Round has between 324 and 117,649 total ways to win. Number of symbol positions on each reel is fixed throughout the Game Round, including both the initial reel spin and any subsequent REACTIONS. If FREE GAMES FEATURE is triggered during the Game Round, number of symbol positions on each reel is re-evaluated at the commencement of the first FREE GAME and for each new FREE GAME Round thereafter. EXTRA REEL EXTRA REEL adds a symbol position to each reel it appears above. EXTRA REEL appears above Reels 2, 3, 4 and 5 only. Symbols appearing on EXTRA REEL are determined independently of the MAIN REEL ARRAY. Any symbol can appear on EXTRA REEL during Bought Games except SCATTER and SOLDIER. Any symbol can appear on EXTRA REEL during FREE GAMES FEATURE. SCATTER can only appear on EXTRA REEL during FREE GAMES FEATURE if revealed via SNAKE STRIKE FEATURE. REACTIONS A REACTION is triggered whenever one or more winning combinations occur. After all applicable wins have been paid, every symbol appearing in a winning combination is removed from its respective reel. Any remaining symbols cascade downwards on the MAIN REEL ARRAY and shift left on EXTRA REEL (if applicable). Empty reel positions are populated by new symbols appearing from above each respective reel on the MAIN REEL ARRAY and from the right of EXTRA REEL. Wins are evaluated again when all symbol positions have been filled. REACTIONS can be retriggered indefinitely. REACTIONS can be triggered during FREE GAMES FEATURE. MEDUSA’S GAZE FEATURE May be triggered whenever one or more WILD symbols appear on EXTRA REEL. Reels directly below WILD symbol(s) are covered by MEDUSA. MEDUSA replaces one or more symbols on covered reels with WILD symbols. MEDUSA can replace any symbol with WILD except SCATTER. MEDUSA'S GAZE FEATURE can be triggered again after each REACTION. MEDUSA'S GAZE FEATURE can be triggered once per spin and once per REACTION only. Wins are paid once only after each MEDUSA'S GAZE FEATURE has completed. WILDS added to the MAIN REEL ARRAY that do not immediately appear in a winning combination remain on their respective reel until they appear in a subsequent winning combination or until the end of the Game Round. MEDUSA'S GAZE FEATURE can be triggered during FREE GAMES FEATURE. FREE GAMES FEATURE 3 SCATTER symbols anywhere on screen award 10 FREE GAMES. Each additional SCATTER appearing thereafter adds an extra 5 FREE GAMES to the trigger award. SCATTERS are evaluated once only at the conclusion of the game round. Win Multiplier starts at x2. Win Multiplier increases by 1 before each REACTION. Win Multiplier remains at its current value at the commencement of the next Free Game Round. Win Multiplier value can increase indefinitely. Win Multiplier applies to all winning combinations. Win Multiplier returns to x1 at the completion of FREE GAMES FEATURE. SNAKE STRIKE FEATURE SNAKES appear to the left and right of EXTRA REEL throughout FREE GAMES FEATURE. SNAKE STRIKE FEATURE is triggered whenever 1 or more SOLDIER symbols appear on EXTRA REEL during FREE GAMES. Each SOLDIER that appears on EXTRA REEL is ATTACKED by one of the two SNAKES. If SOLDIER is attacked by LEFT SNAKE, symbol position turns into WILD. If SOLDIER is attacked by RIGHT SNAKE, symbol position turns into SCATTER. SNAKE STRIKE FEATURE can be retriggered during REACTIONS if subsequent SOLDIER symbols appear on EXTRA REEL. MEDUSA'S GAZE FEATURE is triggered once only per spin and per REACTION after any applicable SNAKE STRIKE FEATURES have completed Wins are paid once only per spin and per REACTION after all FEATURES have completed FREE GAMES FEATURE can be retriggered during FREE GAMES FEATURE FREE GAMES are played at the coin value of the trigger game TEMPTATION FEATURE™ Triggered once all FREE GAMES have completed. Two OPTIONS are presented. Choose whether to COLLECT the current FREE GAMES win or REPLAY the FREE GAMES FEATURE. If player COLLECTS the current FREE GAMES win, FREE GAMES FEATURE completes. If REPLAY OPTION is selected, winnings from the original FREE GAMES FEATURE are forfeited. Any wins accrued during the Bought Game that triggered FREE GAMES FEATURE are BANKED and are not forfeited if REPLAY OPTION is selected. Number of REPLAYED FREE GAMES awarded is equal to the number of FREE GAMES awarded in the original FEATURE trigger. Win Multiplier resets to x2 at the beginning of the REPLAYED FREE GAMES FEATURE. REPLAYED FREE GAMES are played at the coin value of the trigger game. FREE GAMES FEATURE can be REPLAYED once only. TEMPTATION FEATURE is not triggered again after REPLAYED FREE GAMES FEATURE completes.
