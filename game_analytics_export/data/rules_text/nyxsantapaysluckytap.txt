Santa Pays Lucky Tap User Interface ‘+’ / ‘-’ buttons increase / decrease the bet value — the bet value is the total bet per game round. Starts a game round at the current bet value. Displays the autoplay options. Stops the autoplay session. Exits the autoplay menu. Starts the autoplay session. The numerical value displayed within the button shows the number of autoplay game rounds remaining Bet Deﬁnitions Current Bet: The current bet is the value that appears above . Minimum Bet: The minimum bet is the lowest possible bet value, which is selected by pressing until the bet value no longer decreases. Bet Multiplier: The bet multiplier is defined as the current bet divided by the minimum bet. Autoplay Options Autoplay will allow game rounds to be played automatically. Click on the Autoplay button and select the number of Autoplay game rounds. Autoplay session — The number of Autoplay game rounds selected. Loss Limit — set the maximum loss for the Autoplay session — if this value is met or exceeded, the Autoplay session will stop. Single Win Limit — set a win limit for the Autoplay session — if this value is met or exceeded, the Autoplay session will stop. Stop On A Progressive Jackpot (if enabled) — choose if the Autoplay session should stop / not stop if a Progressive Jackpot is won. This option will only be displayed on games oﬀering Progressive Jackpot(s). Bet — amend the Bet value, if required. The Start Button will populate with the selected number of Autoplays when a permissible combination of settings has been selected. Certain Autoplay settings are mandatory for some jurisdictions. If you are disconnected while playing, Autoplay settings will have to be re-set when you re- load the game. Additional Information The overall return to player is 96.79%. The game outcomes are determined by a random number generator that has been certified by an approved test house. Incomplete games will be stored for a minimum of 15 minutes. Returning to an incomplete game allows the player to complete the incomplete game round. If a game is incomplete for longer than the stated time period, the game will be auto resolved and any winnings credited to the player’s account. Misuse and malfunction voids all pays and plays. The above 2 points may be subject to further terms, conditions and processes of the gaming site operator. Wins A winning play occurs when lights up during the game round. A Currency Win, the Dice Game, or both are awarded on a winning play. Multiplier Feature The Multiplier Feature may randomly be activated on a winning Play. If the Multiplier Feature is activated, a random multiplier of x2, x3, x4, x5, x6, or x7 will appear. The Currency Win will be multiplied by the random multiplier. Dice Game The Dice Game is initiated if 3 are awarded on a winning play. During the Dice Game, the player rolls two dice and collects Currency Wins. At minimum bet, Currency Wins during the Dice Game are awarded according to the following table: Sum of Dice Currency Win 6 or 8 0.40 5 or 9 0.60 4 or 10 0.80 3 or 11 1.20 2 or 12 2.00 7 on Roll 1 1.40 + Collect 7 on Roll 2+ Collect When a 7 is rolled, the accumulated Currency Wins are awarded. The Multiplier Feature is not active during the Dice Game. Last modification date: 12/6/2023
