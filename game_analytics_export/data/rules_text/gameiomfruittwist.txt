Fruit Twist Game Features Fruit Twist is a video slot game in 5x3 format. It has 5 paylines and 8 symbols. Main game You can select your stake in the following manner: by setting LINES, COIN SIZE and COINS parameters (in the desktop version) by clicking on the bet in the bottom left corner and using up and down arrows (in the mobile version) by opening hamburger menu in the bottom right corner, clicking on the coins icon and then setting LINES, COIN SIZE and COINS parameters (in the mobile version) Press SPIN button to start the game. The reels will spin and present the result. The result is then evaluated against the paytable. Your winnings for the game in progress are displayed in the info line in the bottom middle part of the user interface and on slot face. Paylines There are 5 paylines. The following rules apply: All winning combinations must appear from left to right and start on the 1st reel, unless explicitly stated otherwise. Only the highest win is paid for each active payline. Payline wins are multiplied by the active line points. Scatter wins are multiplied by total bet and added to the payline wins. All displayed prizes are prize totals with all applicable multipliers already taken into account. Payouts in the pay table are shown in credits. Balance, bet and win are shown in player's currency. Malfunction voids all pays and plays. RTP of this game is 95%. Unfinished Games Any games that are unfinished will be recreated once the player returns to game after the interruption. The maximum win from a single game is £250,000. Last modification date: 6/4/2020
