Electric Charge About this Game This is a 5 reel 3 row video slot game. The objective of the game is to spin the reels so the symbols land in different combinations. Winning combinations are highlighted on the reels and a payout is awarded. This game includes 20 fixed paylines. This game features Cash Spree™ which can be awarded in the base game and during the Free Spins feature. Payout Rules If multiple winning combinations land on a single payline, only the highest win is awarded. Only the highest winning combination is awarded per symbol combination. The payout value is based on the winning combination formed. The total payout is the sum of all payline wins, Cash Spree™ and Free Spin symbol wins resulting from a spin. The Paytable displays the payout for each symbol combination based on the current bet. Payline Rules A payline is a pattern of adjacent symbol positions that runs in a line across the reels. Payline win payouts are based on the initial bet placed. A win is awarded when matching symbols land adjacent to each other on a payline. Payline wins run from left to right, starting from the first slot reel. This game includes 20 fixed paylines. Bet Rules Each spin played costs a total bet amount. The initial bet is the current total bet pided by the payline cost of 20. The total bet is based on the initial bet placed, multiplied by the cost of the paylines played. Wild Symbols Wild symbols can substitute for other symbols to complete winning combinations. Wild symbols can create their own winning combination when landing in a payline combination. Wild symbols can substitute for all symbols except Cash Spree™ and Free Spin Symbols. Mystery Symbol There is a designated section on the reels that selects a symbol each spin. The Mystery symbol will change to any other symbol, except Free Spins symbol. This symbol can appear stacked on the reels. Cash Spree™ Symbols Land Cash Spree™ symbols, starting on reels 1 and 2, to trigger the feature. Cash Spree™ symbols award Mega, Major, Minor, Mini, or cash values. Mega, Major, Minor, and Mini only appear on reels 4 and 5. Cash values appear on all reels. Higher cash values appear on reels 4 and 5. All Cash Spree™ symbols pay left to right on adjacent reels. Cash Spree™ symbol wins are added to payline wins. Free Spins Wheel 3 Free Spin symbols scattered anywhere on the reels triggers Free Spins Wheel. Each wedge of the Free Spins Wheel contains a number of free spins and a multiplier. The multiplier applies to all wins in the Free Spins except Mega. Select the wheel to stop spinning. When the wheel stops, the wedge that the pointer points to is awarded. Select the Spin button to start the Free Spins. 3 Free Spin symbols scattered anywhere on the reels retriggers Free Spins. The number of free spins awarded on a retrigger is equal to the initial trigger. Free Spin symbols appear on reels 2, 3, and 4. Wins awarded during this feature are added to base game wins. All bets played are the same as the spin that triggered this feature. All lines played are the same as the spin that triggered this feature. Game Controls Credits: Displays the current balances. Bet: Sets the current bet value. Win: Displays the most recent win. Spin button: Starts a reel spin. Spacebar: Starts a reel spin. Spins are played at the current selected bet. Spacebar during reel spin stops the current reel spin. This does not influence the results of the spin. Selecting the game screen can stop the spin quicker. This does not influence the results of the spin. Spacebar during wheel spin stops the current wheel spin. This does not influence the results of the spin. Autoplay button: Displays Autoplay options. Autoplay stop button: Ends the current Autoplay session. Bet button: Displays options to change the current bet. Menu button: Displays additional game options. Settings button: Displays the game settings. Paytable button: Displays payout tables and game rules. Home button: Exits the game. Help button: Displays help and additional information. Autoplay Autoplay allows a selected number of spins to play out automatically. Loss Limit: Sets a loss limit for the Autoplay session. Autoplay ends if the balance decreases by this amount. Spins are played at the current selected bet. Spins: Sets the number of spins for the Autoplay session. Autoplay ends when these spins have played out. The number of spins remaining in the Autoplay session is displayed in the game. This feature is available in base game only. Game Settings Left-Handed Mode button: Enables left-handed mode. Music: Enables game music. Sounds: Enables game sounds. Startup Screen button: Enables game advertisement on game load. Bet Settings Bet Value buttons: Sets the current bet to the selected value. Max Bet button: Sets the current bet to the maximum value. Free Games A Free Games offer is a number of free game spins awarded to a player. Each offer has a unique reference number. The bet configuration is set by the game managers and cannot be changed. Wins are added to the cash or bonus balance, depending on the offer. An offer can be played now, played later, or discarded. Select Play Now to begin playing a Free Games offer. Only one Free Game offer can be played at a time. Select Play Later to postpone the offer. A postponed offer will be available again when re-entering the game. Select the Bin icon to discard the Free Games. A discarded offer can only be re-opened by contacting the game managers and providing the reference number. When an offer is complete, discarded, or expired the next available offer is displayed. Each offer has an expiry date and time; if an offer is not played before this time, it will no longer be available. Select the Information icon to view bet information and the offer expiry date and time. Additional Information Some settings and features may not be available in this game. Malfunction voids all pays and plays. Any changes to game rules will be conducted in accordance with regulatory requirements. In the event of a disconnection, the final game state is displayed on return to the game. The balance is updated with the amount won or lost. The results of the last game played are displayed. If a request does not reach the server before disconnection, the results of the previous game played are displayed. Any features in progress that require interaction or selection can be continued. Return to Player The Return to Player (RTP) is the theoretical statistical percentage of the total money bet in a game that is paid out as winnings over time. Values are calculated initially by piding the total possible payouts by the total possible outcomes that are available within the game. Values reflect the maximum expected return and do not change.
