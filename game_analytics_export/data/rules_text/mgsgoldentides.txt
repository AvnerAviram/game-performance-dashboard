Autoplay You can also spin a set number of times in succession without interacting with the game, using Autoplay. You just select a bet amount before you start Autoplay, as it plays with the last bet value. Tap/Click Autoplay On. Tap/Click the number of spins you want to play. The Autoplay session automatically starts playing. To stop Autoplay, Tap/Click Stop. Custom Mode Autoplay is also available with Custom Mode where some Autoplay Settings are available. Autoplay Settings Coin Size You can select a coin size before you start Autoplay, as it plays with the last bet value. Coins Bet Coins Bet is the number of coins allowed per line. Coins Bet differ between games. Total Bet Total Bet displays the Total Bet value. Total Bet = (Coin Size * Coins Bet * Number of paylines). Spins You can set the number of times the slot reels spin during your Autoplay session. Win Limit Stop spins if the number of credits won on any spin exceeds or equals the amount selected. Loss Limit Stop spins if Autoplay exceeds the loss limit value with the next bet. Until Stop Continue Autoplay until you Tap/Click the Stop button or stops if Free Spins or another bonus feature is triggered.
