Musketeers Overview The goal is to obtain a winning combination on a winning line spread across the reels. Game specifications : Type Slots Number of reels 5 Number of rows 3 Number of lines 1 to 20 Number of coins you can bet per line 1 to 5 Progressive No Double feature No Autospin feature Yes Wild symbols Yes Multiplier symbols Yes Scatter symbols Yes Bonus round No Free s pins Yes Instant win No RTP 96.69% Jackpot 100,000 coins, at max bet The game disconnects if you do not take any actions for a specific number of consecutive minutes (this number varies depending on jurisdiction); the session recovery feature will save the game progress so that the next time the game is loaded, you can continue where you left off. In the event of technical difficulties resulting a dropped connection to the game server, the game you were playing is saved automatically and will load the next time you establish a connection to the game server. You will not be able to reconnect to the game server for the first 30 seconds after the disconnection. Note : In case of malfunction all pays and play are voided; any bet that was not yet accepted will be voided, any bet that was unsettled will be refunded. How to Play Note: If you do not have enough money to play a game, the bet can be set to the minimum selectable amount through a dialog window informing about this. All winning lines pay from left to right. Wins on different selected lines are all added. Only the highest winning combination on each selected line is paid. Winnings from free spins are added to the total win. Note : On the info bar located under the main game buttons, you can view the current BET, your BALANCE and your current WINNINGS, in your currency. To play, follow these steps: Use the coin value select button to choose a coin value. The coin value select button is located in the bottom-left side of the main game screen and shows the currently selected coin value. Use the LINE BET button to choose the amount of coins you want to bet on each active line. Use the LINES button to choose the number of active lines. Note : The current bet amount, in your currency, is displayed in the TOTAL BET box. Click SPIN to place the bet and spin the reels once. You can click the BET MAX button to place the maximum bet (maximum number of coins per line and maximum number of active lines) for the currently selected coin value and spin the reels once. The total amount won, in your currency, according to the paytable, is shown in the WINNINGS box. For each winning combination, the amount won is the amount shown in the paytable for that winning combination. 3. Special Symbols and Features 3.1. Extended Wild During normal spins, 1 to 5 reels can turn into an extended Wild (a Wild reel), at random. During free spins, extended Wilds remain in place until the free spins end. The Extended Wild replaces any other symbol, except Scatter . 3.2. Scatter Scatters pay anywhere on the reels, according to the paytable. 3.3. Free Spins Three or more Scatters anywhere on the reels award a specific number of free spins, according to the paytable. Additional free spins can be won during free spins. At the end of the free spins, you may also win a multiplier (between X2 and X10). The multiplier applies to total free spins winnings. 4. Autospin feature The loss limit setting allows stopping the autospin feature automatically when the selected limit plus your cumulative winnings minus your cumulative bets is less than 0. Note : The autospin feature does not automatically start free spins – player input is required. Click the AUTO SPIN button to open the AUTO SPIN SETTINGS window, where you can set up the autospin feature. On the AUTO SPIN SETTINGS window, you can set up the following: Loss limit – enter a number for the loss limit value or use the left and the right arrow buttons to select a number for the loss limit value Stop if WIN reaches – to enable this setting, click the button on its left side and then enter a number for the WIN value or use the left and the right arrow buttons to select a number for the WIN value; click the button on its left side again to disable it Stop if JACKPOT is won - (only for games with Progressive Jackpots or with a specific Jackpot value displayed) - to enable this setting, click the button on its left side; click the button on its left side again to disable it Number of AUTO SPINS to play – use the left and the right arrow buttons to select the number of autospins Click START to close the AUTO SPIN SETTINGS window and start the autospin feature with the currently selected bet and number of autospins or click CANCEL to close the AUTO SPIN SETTINGS window without starting the autospin feature. Click the STOP xx/yy button to stop the autospin feature at any time; xx represents the remaining number of autospins and yy represents the number of autospins to play that you selected on the AUTO SPINS SETTINGS window. 5.
