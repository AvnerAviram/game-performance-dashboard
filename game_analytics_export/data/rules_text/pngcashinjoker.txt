Cashin' Joker Unfinished Games If your game round is disrupted, all game information and placed bets are stored until you reopen the game. You may continue your unfinished round from the point of interruption. Unresolved bets placed but remaining undecided in incomplete games will become void after 7 days. Malfunction voids all pays and plays. Information Payline wins occur on the number of selected paylines, according to the information in the paytable and game rules. When winning on multiple paylines in a single game round, all winnings are added together. All winnings from Features (such as Free Spins), Bonus Games and/or Scatters (if applicable) are also added to payline wins. All winning combinations are paid out at the end of a game round. Free Spin features are played with the same bet and same number of selected lines as the game round that triggered the feature – unless otherwise stated. The bet cannot be changed during a currently running game round. Please refer to the game rules for more information. All wins pay left to right, beginning with the left-most reel. Symbol wins pay for any winning payline combination without any gaps. Only the highest win is paid per active win line. Game Rules Cashin' Joker is a 3-reel game that expands into a 4-reel game when the X-TRA CASHIN' REEL feature is triggered. This game includes Multiplier Number symbols and a JOKER symbol. The Multiplier Number symbols act as a Multiplier that multiplies the total bet. The JOKER symbol is a SCATTER that can land on any reel, but only one JOKER symbol can appear at a time. When the JOKER symbol lands, the X-TRA CASHIN' REEL feature is triggered. To win in this game, the Multiplier Number symbols are combined and will be the total Multiplier for that game round. It is also possible to land a blank (no symbol), which has no value. During the base game, each reel has corresponding Multiplier Number symbols: Reel 1 contains 8, 3, 2, 1, and 0; Reel 2 contains 8, 2, 1, and 0; and Reel 3 contains only 0. The RE-SPIN feature can be randomly triggered when specific conditions are met. If reel 1 displays a non-zero Multiplier number and reel 2 displays any Multiplier number while reel 3 is blank (no symbol). Reel 3 will perform a single RE-SPIN. During the RE-SPIN, it is possible to land a JOKER symbol, which can trigger the X-TRA CASHIN' REEL feature. JOKER is a SCATTER that can land on any reel, triggering the X-TRA CASHIN' REEL feature. This feature reveals X-TRA reel added to the leftmost position, expanding the reel size to 4x1. It is also possible to land a blank (no symbol) on any reel except reel 1, blanks have no value. During this feature, each reel has corresponding Multiplier Number symbols: Reel 1 contains 10, 8, 3, 2, and 1; Reel 2 contains 8, 3, 2, 1, and 0; Reel 3 contains only 0; and Reel 4 also contains only 0. During the X-TRA CASHIN’ REEL, RE-SPIN cannot be triggered, JOKER symbols cannot land, and the X-TRA CASHIN’ JOKER cannot be re-triggered. All possible Multiplier Number combinations for the entire game are: 0, 1, 2, 3, 8, 10, 11, 12, 13, 18, 20, 21, 22, 23, 28, 30, 31, 32, 33, 38, 80, 81, 82, 83, 88, 100, 101, 102, 103, 108, 110, 120, 130, 180, 200, 210, 220, 230, 280, 300, 310, 320, 330, 380, 800, 810, 820, 830, 880, 1000, 1010, 1020, 1030, 1080, 1100, 1200, 1300, 1800, 2000, 2100, 2200, 2300, 2800, 3000, 3100, 3200, 3300, 3800, 8000, 8100, 8200, 8300, 8800, 10000, 10100, 10200, 10300, and 10800 X the total bet. Bets are selected using the bet buttons. Click the plus and minus buttons to change the bet one step at a time. To start the round, click SPIN. When the reels stop, the symbols displayed determine your prize according to the paytable. Any combination of wins resulting from a single game cycle, including the results of any possible triggered bonuses, is capped at a maximum of 10888 times total bet. Auto adjust bet - Automatically reduces the total bet when starting a new spin, if the balance is too low to initiate a spin at the current bet level. The reduced total bet selected will be the highest possible bet but will still be lower than the selected bet size when autospins were initiated.
