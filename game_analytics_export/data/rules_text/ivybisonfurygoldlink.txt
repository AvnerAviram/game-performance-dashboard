GAME RULES: BISON FURY GOLD LINK is a 5x4 reel slot game with 1024 WAYS to win. The BET value can be changed by using the ‘+’ and ‘-’ buttons. Click on the PLAY button to begin the game with the currently selected or default BET amount. All payouts are made according to the paytable. The paytable will always display the winning amounts for the currently selected BET. All symbols pay from LEFT TO RIGHT on consecutive reels, starting with the left-most reel, excluding the SUNSET (SCATTER) and GOLD LINK symbols. Only the highest win in each possible combination will be paid. All possible wins are added to the player BALANCE. The SUNSET symbol acts as a SCATTER. The WILD COIN symbol is WILD and substitutes for all symbols except the SUNSET (SCATTER) and GOLD LINK symbols. The WILD symbol will appear only on reels 2, 3, and 4 in the base game and FREE SPINS feature. The SUNSET (SCATTER) and WILD symbols do not have any payout in the base game or FREE SPINS feature. The GOLD LINK symbol does not have any payout in the base game. When 3 or more SUNSET (SCATTER) symbols land anywhere in view, the FREE SPINS feature will be triggered. Any GOLD LINK symbol that lands in view will be locked in position and will reveal either a CASH VALUE, a JACKPOT (MINI, MAJOR and MEGA) or COLLECT. A GOLDEN SPIN will then be awarded. COLLECT symbol only appears during the GOLD LINK feature. GOLD LINK symbols that land in base game or GOLDEN SPIN will not reveal any COLLECT. Any GOLD LINK symbols in view will be locked in position and a GOLDEN SPIN will be awarded. 6 or more GOLD LINK symbols landing anywhere in view will trigger the GOLD LINK feature with the chance to win the JACKPOTS (MINI, MAJOR and MEGA). The values displayed on the GOLD LINK symbols will carry forward to the GOLD LINK feature. The GOLD LINK symbol will appear in the base game and the GOLD LINK feature only and is available on all reels. The JACKPOTS (MINI, MAJOR and MEGA) are active during the GOLD LINK feature. The SUNSET (SCATTER) symbol will appear only in the base game and FREE SPINS. Logging into your account on the desktop website and selecting the option “HISTORY” inside any game will allow you to access your gameplay history. All payouts are made in USD. GOLDEN SPIN: A GOLDEN SPIN is awarded to the player when each GOLD LINK symbol lands in view and is locked in place. GOLDEN SPINS will re-trigger if further GOLD LINK symbols land on subsequent spins. The GOLD LINK symbols lock and hold until no further GOLD LINK symbols are added. If 6 or more GOLD LINK symbols land during the GOLDEN SPINS, the GOLD LINK feature will be awarded. The values displayed on the GOLD LINK symbols will carry forward to the GOLD LINK feature. During GOLDEN SPINS the values displayed on the GOLD LINK symbols will not be awarded as a payout. GOLDEN SPINS are played at the same BET as triggering game. FREE SPINS FEATURE: The FREE SPINS feature is triggered when 3 or more SUNSET (SCATTER) symbols land anywhere in view during the base game. Players will be awarded 8, 20 or 50 FREE SPINS when 3, 4 or 5 SUNSET (SCATTER) symbols land anywhere in view during the base game. During FREE SPINS the WILD COIN symbol acts as a STICKY WILD. When a WILD COIN symbol lands in view it will award between 1 and 5 RESPINS randomly. The number of RESPINS for each WILD COIN is shown on the symbol. RESPINS do not count as FREE SPINS. For each RESPIN completed the WILD COIN count is decreased by 1. When the RESPINS count reaches zero for each WILD COIN the symbol will disappear on the next spin. Further WILD COIN symbols which land during RESPINS will trigger additional RESPINS up to a maximum of 5 for each new WILD COIN symbol which lands. FREE SPINS commence once all the RESPINS are completed. The FREE SPINS feature will be re-triggered when 3 or more SUNSET (SCATTER) symbols land in view during FREE SPINS. FREE SPINS will end when all RESPINS and FREE SPINS have been completed. FREE SPINS and RESPINS are played at the same BET as the triggering FREE SPINS game. GOLD LINK FEATURE: The GOLD LINK feature is triggered when 6 or more GOLD LINK symbols land anywhere in view during the base game or GOLDEN SPINS. The GOLD LINK feature starts with the same number of GOLD LINK symbols seeded in their landing positions. The GOLD LINK feature starts with 3 spins awarded to the player. Each additional GOLD LINK symbol that lands in view resets the spins counter to 3. Each GOLD LINK symbol that lands in view will have a random BET MULTIPLIER applied to it from the following range - 0.5x, 1x, 1.5x, 2x, 2.5x, 3x, 4x, 4.5x, 5x, 7.5x, 15x, 20x, 60x, 75x, 100x, 500x (MINI JACKPOT), 5000x (MAJOR JACKPOT) and 10000x (MEGA JACKPOT). If a COLLECT symbol lands during the GOLD LINK feature it will collect all the values from the GOLD LINK symbols currently locked in view. Once all the values on the GOLD LINK symbols have been collected, the COLLECT symbol will transform into a standard GOLD LINK symbol displaying the collected accumulated value. The reel positions of the collected GOLD LINK symbols will be cleared on the next spin and the spins count will be reset to 3. The GOLD LINK feature will continue until the player reaches zero spins or has filled all 20 GOLD LINK reel positions. Once the GOLD LINK feature is completed all the GOLD LINK symbol values are accumulated and awarded as winnings to the player. The JACKPOTS (MINI, MAJOR and MEGA) can be won only during the GOLD LINK feature: If a GOLD LINK symbol labeled with MINI lands anywhere in view, the MINI JACKPOT (500x) will be awarded to the player. If a GOLD LINK symbol labeled with MAJOR lands anywhere in view, the MAJOR JACKPOT (5000x) will be awarded to the player. If a GOLD LINK symbol labeled with MEGA lands anywhere in view, the MEGA JACKPOT (10000x) will be awarded to the player. More than 1 JACKPOT can be won during the GOLD LINK feature. The JACKPOT values are in ratio with the BET selected and wagered. FEATURE ENHANCER: The FEATURE ENHANCER increases the players BET by 1.25x. The FEATURE ENHANCER increases the chances of triggering the GOLD LINK feature. The Theoretical return to the player using FEATURE ENHANCER is 95.72%.
