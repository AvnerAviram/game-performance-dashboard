Celtic Goddess About Celtic Goddess Celtic Goddess is a 5x3 reel, thirty fixed payline slot game. General Game has fixed 30 lines. If you have a winning combination lined up on the payline, the win is highlighted on the reels. Regular wins are created when the same symbols appear next to each other on a payline. Winning combinations must start from the leftmost reel and the symbols have to be consecutive along a payline. One regular win is paid per line. If you have more than one combination, you are paid the win that has the highest value. Regular wins depend on your bet per line except Scatter. Bet per line = (Coin size * Coins Bet). Regular wins in paytable are displayed in credits, so if your total bet is 0.30 credits per spin, your bet per line = Total Bet / number of paylines i.e. 0.30 credits / 30 paylines =0.01 credits. Your total payout is calculated as { (bet per line * credits won on minimum bet) * 100 }. For example a 5 of a kind win of Diamond symbol which is 0.50 credits, the total payout amount will come as { (0.01 bet per line * 0.50 credits won on minimum bet) * 100 } = 0.50 credits. The Win up to value is displayed in coins in the game paytable which is the maximum value that can be won by the player while playing on maximum bet. This value is achieved using the combination of regular wins itself. So the conversion from coins to credits would be (Max win value in coins * Maximum bet per line) = Max win value in credits. For example, the Max win value is 1000 coins and you play on maximum bet which is 30 credits per spin, your Maximum bet per line = maximum bet / number of paylines i.e. 30 credits / 30 paylines = 1 credits. So your Max win value in credits is calculated as (1000 coins * 1 credits) = 1000 credits. For providing more information to the player in calculating the payouts, this game has a dynamic paytable. The payouts displayed in the paytable for all the symbol combinations are the final amounts in credits which is already multiplied by active bet per line and paid to the player. You are paid out the win amount displayed in the Paid box. If you do not have a winning combination, the game is over and you can play again. The theoretical percentage return to player is 96.02%. Scatter symbol is a Scatter symbol. Scatter symbols can appear during base game only. 5, 4, 3 Scatter symbols scattered anywhere award 16, 12 or 8 Free Spins respectively. Scatter symbol cannot be substituted by the Wild symbol. Scatter wins are multiplied by total bet and added to payline wins. Only highest Scatter win is paid per spin. Base pay of the Scatter symbol for the particular combination can be calculated as (credits won on minimum bet / number of paylines) * 100. So total payouts for the Scatter win can be calculated as (base pay of the Scatter symbol combination * total bet). For providing more information to the player in calculating the payouts, this game has a dynamic paytable. Scatter payouts displayed in the paytable are the final amounts in credits which are already multiplied by the total bet and paid to the player. Wild symbol Danu Goddess symbol is a Wild symbol. Wild appears on reels 2 and 4. Wild substitutes for all symbols except Scatter symbol. Any win with 1 wild symbol subtituting is multiplied by 2x and with 2 wild symbols subtituting is multiplied by 4x. Goddess Feature Goddess feature may trigger randomly on any spin. During Goddess feature, Any suit symbols on reels will be replaced by any single high paying symbol except wild and scatter. Free Spin Bonus is a Scatter symbol. 5, 4 or 3 scatter symbols scattered anywhere awards 16, 12 or 8 free spins respectively. During free spin bonus, Goddess feature occurs after every spin. Free Spin Bonus cannot be re-triggered. All bets and lines played are the same as those that started the Free Spin Bonus round. Autoplay You can also spin a set number of times in succession without interacting with the game, using Autoplay. You must select a bet amount before you start Autoplay, as it plays with the last bet value. Tap/Click Autoplay On . Tap/Click the number of spins you want to play. The Autoplay session automatically starts playing. To stop Autoplay, Tap/Click Stop . Custom Mode Autoplay is also available with Custom Mode where some Autoplay Settings are available. Autoplay Settings Coin Size You can select a coin size before you start Autoplay, as it plays with the last bet value. Coins Bet Coins Bet is the number of coins allowed per line. Coins Bet differ between games. Total Bet Total Bet displays the Total Bet value. Total Bet = (Coin Size * Coins Bet * Number of paylines). Spins You can set the number of times the slot reels spin during your Autoplay session. Win Limit Stop spins if the number of credits won on any spin exceeds or equals the amount selected. Loss Limit Stop spins if Autoplay exceeds the loss limit value with the next bet. Until Stop Continue Autoplay until you Tap/Click the Stop button or stops if Free Spins or another bonus feature is triggered. Important: This feature may not be available in the game you are playing. Keyboard Shortcut Spin/Stop Button can be selected by pressing Space bar on Keyboard.
