Action Bank Plus HOW TO PLAY Base Game The slot features a 6x4 reel set, where the 6th column is made up of four individually spinning reels, which can contribute to winning combinations. Winning combinations may contain only one symbol from each adjacent reel, beginning with the far left reel. Only highest winner paid per winning combination. Like symbols or their substitute, on a single reel are not part of the same winning combination but may create separate winning combinations. Wins on different ways are added together. Paytable reflects current bet configuration. Reel Set If all four reel positions in the 6th column show an O symbol, there is a chance for all the O symbols in this column to transform into vault symbols. When all four O symbols in the 6th column transform into vault symbols, the Vault Feature will be triggered. Choose a Vault to reveal a base number of Free Spins to play with. Vaults contain 4, 8, 12 or 16 Free Spins as an award. Extra Free Spins can be awarded. One Free Spin will be added to the current number of Free Spins for every O symbol which landed on the first 5 reels. The Free Spins Feature will begin once the total number of Free Spins has been calculated. The Wild symbol appears in Free Spins only, on reels 2, 3, 4, and 5. The Wild symbol substitutes for all symbols except the O symbol and the vault symbol. LEGAL NOTICES Malfunction voids all pays and plays.
