YoYos Wild Game Description “Walk the Dog” or go “Two-Handed” with the world’s first YoYo-themed Slot Game. The innovative trick feature creates a pattern of substitute symbols based on four classic YoYo tricks which then creates additional pay lines. All you need to trigger the trick-selector is spin 3 scatters on the reels and the feature will present you with between 2-3 tricks.There is nothing quite like this game, so take a nostalgic trip down memory lane and play to see what all the excitement is about! How to play Take out your favourite YoYo and let’s start playing. First, you need to choose your initial bet amount and then hit ‘Spin’ to get the reels rolling! If you want a more relaxed experience, then you can select the Autoplay function. Look for the YoYo symbol, which appears on reels 2,3 or 4 and is the game’s scatter. If three YoYos appear on the screen then the trick selector will present you with up to three amazing YoYo tricks. These tricks will turn all the surrounding symbols into wilds and help you to win big. Yoyo’s Tricks The feature is triggered when 3 YoYo symbols appear on reels 2,3,4. Once this happens you will see the trick selector, which will provide you with up to 3 tricks. Each trick will turn the reels on the direction of the YoYo trick into wild symbols giving a number of new winning combinations. Return To Player The Return to Player for this game is 96.5%.
