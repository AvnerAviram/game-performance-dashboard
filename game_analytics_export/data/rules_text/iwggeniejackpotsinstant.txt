Genie Jackpots Instant Introduction Reveal the 25 symbols. Find 3, 4 or 5 like symbols in a vertical, horizontal or diagonal line to win a prize! If a carpet turns red you unlock the RED CARPET BONUS game. Find 3 magic lamp symbols anywhere to unlock the LUCKY LAMP BONUS game. Start the game Select the stake amount, then to start the game press the PLAY button. Once the PLAY button has been selected the stake is deducted from the current balance and the game starts. End of the game At the end of the game, any winnings are displayed. To play again press the PLAY button. Once the PLAY button has been selected the stake is deducted from the current balance and the game starts. How to play At the start of the game there are 25 yellow carpets. Select each carpet to turn to reveal a symbol. Or press the REVEAL ALL button to reveal all symbols. If a carpet turns red unlock the RED CARPET BONUS game. Find 3 lamp symbols anywhere to unlock the LUCKY LAMP BONUS GAME. RED CARPET BONUS GAME Pick chests to open. Chests opened reveal a prize and adds to your prize. Keep going until you find the COLLECT card. At this point all prizes revealed during the game are win. LUCKY LAMPS BONUS GAME Pick 1 magic lamp. Win the prize that is revealed Winning Wins are achieved by: Finding 3 or more like symbols in a vertical, horizontal or diagonal line. Win the corresponding prize on the prize table. RED CARPET BONUS game - win all prizes revealed during the game. LUCKY LAMPS BONUS GAME - win the prize that is revealed General Information System malfunction voids all pays and all plays.
