Mr. Gold 1. Overview The goal is to obtain a winning combination on a winning line spread across the reels.. Game specifications : Type Slots Number of reels 5 Number of rows 3 Number of lines 10 (cannot be changed) Number of coins you can bet 10 (cannot be changed) Progressive No Double feature No Auto spin feature Yes Wild symbols Yes Multiplier symbols No Scatter symbols Yes (Vault symbol) Bonus round Yes Free s pins No Instant win No RTP Base game: 94.00 % Fever Spins: 100.00 % The game disconnects if you do not take any actions for a specific number of consecutive minutes (this number varies depending on jurisdiction); the session recovery feature will save the game progress so that the next time the game is loaded, you can continue where you left off. In the event of technical difficulties resulting in a dropped connection to the game server, the game you were playing is saved automatically and will load the next time you establish a connection to the game server. You will not be able to reconnect to the game server for the first 30 seconds after the disconnection. Note : In case of malfunction all pays and play are voided; any bet that was not yet accepted will be voided, any bet that was unsettled will be refunded. 2.1. Wild The Wild symbol replaces any other symbols, except the Vault symbols, to form winning combinations. 3.2. Stacked Wilds Stacked Wilds appear on reel 3 in Base game and on reels 2, 3 or 4 in Fever Spins . Landing a full stack of high symbols will convert that stack into a stack of Wilds . 3.3. Vault Bonus Vault Bonus is triggered by landing Vault symbols on reels 2, 3 and 4. When 3 Vault symbols land all cash symbols become collectables. The Vault on reel 3 opens and collects the cash values. Only 1 Vault symbol can land per reel. Vault Bonus can be triggered in Base Game or during Fever Spins . Cash Values in total bet: Blue (20) Notes: 1x, 2x, 3x Green (100) Notes: 3x, 5x, 7x, 10x Purple (500) Notes: 7x, 10x, 15x, 20x, 25x, 50x 3.4. Fever Spins Fever Spins feature is triggered by any base win of 4 x Bet or higher. Fever Spins initially cost the same as the base spin that triggered them. Player can increase the Fever Spins bet up to 2x the original base bet. Player can freely increase or decrease the bet between Fever Spins . Player can play Fever Spins as long as their Fever Spins /Win balance is available, up to 100 spins maximum. Player can cash out at any point and add the Fever Spins win to their balance. If player runs out of funds in Fever Spins , it is not possible to fund Fever Spins from the main balance. Note: Increasing the bet during Fever Spins will also increase the payouts in proportion. 4. Auto spin feature The loss limit setting allows stopping the auto spin feature automatically when the selected limit, plus your cumulative winnings, and minus your cumulative bets, is less than 0. 5.1. Desktop version Click the AUTO SPIN button to open the AUTO SPIN settings window where you can set up the following: Number of auto spins : click + or - to select the number of auto spins to play. Loss limit : click + or - to select the value of the loss limit; Stop if a win reaches : click + or - to select the value of the maximum win; Stop if JACKPOT is won (only for games with Progressive Jackpots): to enable this setting, tick the checkbox ; to disable it, remove the tick. Click the ACTIVATE button to close the AUTO SPIN settings window and to start the auto spin feature with the currently selected bet and auto spin setup. Click the CLOSE button to close the window without starting the auto spin feature. Click the STOP AUTO SPIN button to stop the auto spin feature at any time. 5.2.
