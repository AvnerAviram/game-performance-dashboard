Whirl Win BASIC RULES The game can be played on either 5 lines for 5 credits or 9 lines for 9 credits. Wins on multiple lines are added together. Only the highest line win is paid on each active pay line. Player is responsible to confirm credits registered before game start. Bet per line is the total bet divided by the number of lines played. If 1 or 2 DOUBLE WILD appear on an active payline, these symbols match any other symbols except BONUS on the payline. 1 DOUBLE WILD in a winning combination multiplies the pay by 2. 2 DOUBLE WILD in a winning combination multiplies the pay by 4. 3 DOUBLE WILD in a winning combination pays 4000x bet per line only. 2 DOUBLE WILD and a blank symbol is not a win. DOUBLE WILD can only substitute for a if there is a on the active payline on another reel. 1DOUBLE WILD without a and without matching non-blank symbols is not a win. 1 appearing on the payline is an "Any One Win", regardless of the reel on which appears. 2 appearing on the payline is an "Any Two Win", regardless of the 2 reels on which these symbols appear. All payline wins must occur on adjacent reels beginning with the leftmost reel except . Symbols must align centered on the paylines to qualify as winning symbols. If symbols are only partially on the payline then it is not in a winning position. All wins are on an active payline. WHEEL BONUS RULES If 1 or 2 BONUS appearing on or within 1 position of line 1 may award 1 spin of the wheel. 3 BONUS appearing on or within 1 position of line 1 awards 1 spin of the wheel. After the wheel stops, the value indicated by the pointer is won. When choosing 5 paylines, the possible wedge values are: 50, 250, 20, 500, 30, 125, 200, 100, 40, 1000, 75, 150, 50, 400, 125, 200, 40, 300, 30, 2000, 75, 100. When choosing 9 paylines, the possible wedge values are: 100, 450, 40, 1000, 50, 200, 400, 150, 75, 2000, 125, 250, 100, 700, 150, 400, 75, 500, 50, 4000, 125, 200. Values are multiplied by bet per line. Paytable Winnings per symbol are calculated for Bet amount of $2.70. RTP of this game is 94.10%. Quick start Game starts by setting a selected Bet amount and by pressing the Spin button. Round ends when reels stop and there are no wins, or after the win presentation if there are wins. Turbo mode (where available) Turbo mode is speeding up reels spinning animation when enabled. This option does not influence the game round result but only visual presentation. Game menu Game menu contains options and settings that control some of the game features: History - Displays previous rounds from the current session Close Game - Closes the game client Sound - Controls the Sound FXs in the game (on/off) Turbo mode (where available) - Controls the Turbo mode feature (on/off) Left Hand Mode (where available) - Controls the position of the UI on Mobile devices Vibrate (where available) - Controls the vibration on the Mobile devices (on/off) Game is paused while the menu is visible and can be closed by pressing the Close button or anywhere on the screen. Bet selection Game starts with a default Bet amount set for the game. Player can change the Bet amount by either: Using Plus and Minus buttons next to the Bet Amount Pressing the Bet amount where Bet Amount selector will open up Bet amount cannot be changed during a game round and if there is a win presentation happening, changing bet Amount will reset the presentation. LINES SELECTION Change the lines for the round to be played. Changing the number of lines may adjust the current bet. WIN PAID Game play win amount displayed. Autoplay mode (where available) Autoplay mode enables automatic play of game. It can be enabled or disabled at any time during the Base game. Autoplay mode is enabled by pressing the Autoplay button which allows the player to select: Number of rounds to be played automatically. Loss Limit (where available) - Disables the Autoplay mode if Balance is reduced by more than Loss Limit amount. Stop on Bonus Feature (where available) - Disables the Autoplay mode when Bonus game or Free games are triggered. Stop on Win above (where available)- Disables the Autoplay mode when there is a single round win exceeding the amount entered. Autoplay button will display the remaining number of spins to be played automatically. Pressing on the Autoplay button while Autoplay mode is enabled will reset the number of Rounds and disable the Autoplay mode. Malfunction A malfunction of the game voids all pays. In case the game round is interrupted, it can be restored by simply starting the game again.
