[H4] GENERAL INFO :GENERAL RULES
• This game is played at 243 ways.
• All 5 reels are always in play.
• Ways wins contain one symbol from each adjacent reel, beginning with the leftmost reel.
• The same symbol, or its substitute, in a different position on the same reel pays that ways win again.
• Only the highest win is paid for each ways symbol combination.
• Ways wins are multiplied by the bet multiplier.
• Scatter symbols award in any position on the reels.
• Only the highest win is paid for each scatter symbol combination.
• Scatter wins are multiplied by the total amount bet.
• Scatter wins are awarded in addition to ways wins.
• Wins are shown in credits unless marked as currency.
• Values displayed in the Game Rules/Help Screens have not been pre-multiplied, except where stated.
• Wins subject to verification.
• Malfunction voids all pays and plays.
