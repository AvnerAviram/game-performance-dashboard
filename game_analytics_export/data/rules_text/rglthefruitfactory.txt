The Fruit Factory Game overview The Fruit Factory is a five reel, 20 win line slot game featuring a new innovative ‘Reel Runner’ Bonus Round and a Super Game mode consisting of 5 linked spins. If 3 or 4 Reel Runner scatter symbols appear anywhere on the reels, the Reel Runner Bonus Round is awarded. If 5 Reel Runner symbols appear anywhere on the reels the jackpot is awarded. The Fruit Factory features a maximum prize of 5000 x the Stake per Line value. Formats This document provides information on The Fruit Factory for the following game formats: Flash. HTML5 Tablet. HTML5 Phone. The game Playing the game Bet value Select the total Bet value. (Bet = Lines in Play x Stake per Line). Start button Flash game: Press the Start button to spin the reels. Tablet and Phone games: Press the Start button to spin the reels. The tablet game has the additional option to use thumb control Start buttons – these buttons can be turned on / off by pressing the tool button and selecting the on / off toggle button. Auto button Flash game: Pressing the Auto button will provide various auto play options. Once the desired options have been selected, press the Tick button to start auto play. Press the Cross button to exit the Auto Play set-up. During Auto Play, pressing the Stop button at any time cancels auto play. Tablet game: Pressing the Auto button will provide the option to select 10, 25 or 50 auto spins. Pressing the Stop button at any time cancels auto play. Phone game: Pressing the auto button will start 10 auto spins. Pressing the stop button at any time cancels auto play. Displays Various displays are used to show important information to the player during the game. On phone, some of these displays appear as pop-ups, only when needed. Information display At different points during game play, this will show information about the game, including payouts per win line, scatter awards and Auto Play status. Lines in Play display 20 lines are always in play. Stake per Line display This displays the Stake per Line value. This may be changed by pressing the Up and Down buttons. Bet display This displays the total bet value charged per game. Win display At different points during game play this will display: Total win line and Bonus Round payouts. Total paid for a complete bet. Game views The game can be played in different views, as follows: Flash: Close-up view of the reels only. . Tablet and Phone: Both Tablet and Phone games can be played in landscape and portrait view. Payouts Win line payouts When a winning line is obtained: The winning symbols will be highlighted. The total amount won will be shown. When not in Auto Play, the payout per win line will be displayed one at a time. Standard Game payout values The Fruit Factory has a maximum payout of 5000 x Stake per Line value. All payouts are multiplied by the Stake per Line value, except Scatter awards which are multiplied by the total Bet value. The Fruit Selector screen will be displayed. Press the Stop button to randomly select a fruit to use in the Bonus Round. The Bonus Round consist of 9 reels in a 3x3 grid. Each reel is populated with the selected fruit, a jackpot symbol and blanks. There are 3 steps in the Reel Runner Bonus Round: Step 1: Press the Spin button to spin all 9 reels. There are 8 win lines. 3 horizontal, 3 vertical and 2 diagonal. Keep spinning all 9 reels until a win is awarded. Step 2: Press the Spin button to randomly select one of the 9 reels. This reel will respin and any wins on the 8 win lines will be paid. Step 3: The last reel to spin will now flash red. All others will flash green. Press the Spin button to randomly select one of the 9 reels. If a green reel is selected, it will respin and any wins will be paid. If the red reel is selected, the Bonus Round is complete. Keep repeating Step 3 until the Jackpot is won or the red reel is selected. Any Full House, including Blanks, awards the Jackpot prize. The maximum payout in any one game is 250 x the Total Bet value. An information page will be displayed, outlining the basic rules and cost. Press the Tick button to agree to the rules and switch the game into Super Game mode, or press the Cross button to return to the Standard Game. The game will remain in Super Game mode until switched back to the Standard Game. Press the Super Game button to switch back to the Standard Game. Whilst in Super Game mode, press the Info button to see the full game rules and payouts. Super Game cost: The Super Game costs ten times the Standard Game whilst playing for the same maximum payout. This is done automatically on switching, but can be increased or decreased as required. On switching to Super Game mode, the following changes are made to the staking set up: The Stake per Line is doubled. The Total Bet is multiplied by ten. The maximum payout remains the same. On switching to Standard Game mode, the following changes are made to the staking set up: The Stake per Line is halved. The Total Bet is divided by ten. The maximum payout remains the same. Super Game rules: The Super Game consists of 5 linked spins, played out automatically. During the Super Game, all Reel Runner Symbols are WILD and remain in place. The Reel Runner Bonus Round is awarded (if won) after the last spin. The Super Game uses a different set of reels from the standard Game. The Super Game Bonus Round uses an enhanced Fruit Selector. The maximum payout from the combined 5 spins is 25 x the Total Bet (2500 x Stake per Line). If the maximum payout is reached, any further spins or features are cancelled. Super Game payout values The Super Game has a maximum payout of 2500 x Stake per Line value. All payouts are multiplied by the Stake per Line value, except Scatter awards which are multiplied by the total Bet value. The RTP is expressed as the percentage returned to players in winnings of the total amount staked, e.g. if £1,000,000 was staked and £960,000 was returned, the RTP of the game for that period of play would be 96%. The expected RTP for the Standard Game is no less than 95.46%. The expected RTP for the Super Game is no less than 95.86% The actual RTP during a playing session can vary widely, in either direction, from the theoretical return.
