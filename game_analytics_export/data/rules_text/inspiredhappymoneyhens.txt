Happy Money Hens WIN LINES ALL GAMES ARE PLAYED WITH 20 WIN LINES. WINS CAN OCCUR ACROSS ANY OR ALL OF THE WIN LINES. ONLY THE HIGHEST AWARD IS PAID PER WINNING COMBINATION. ALL WINS PAY LEFT TO RIGHT ON ADJACENT REELS STARTING WITH THE LEFTMOST REEL. WINS ON DIFFERENT PLAYED LINES ARE ADDED. WILD SYMBOLS SUBSTITUTES FOR ALL SYMBOLS FOR LINE WINS. WILDS ONLY APPEAR ON REELS 2, 3 & 4 IN THE BASE GAME AND WHEN FORTUNE BET IS ACTIVE. FEATURE SYMBOLS EVERY TIME A FEATURE SYMBOL LANDS IN VIEW ON THE REELS IT WILL BE ADDED TO THE FEATURE POT. WHEN A FEATURE SYMBOL IS ADDED, THERE IS A CHANCE THAT THE FEATURE POT WILL BE TRIGGERED AND THE BONUS AWARDED WITH THIS FEATURE ACTIVE. ADDITIONALLY, WHEN A FEATURE SYMBOL IS ADDED, THE FEATURE POT MAY INCREASE IN SIZE UP TO LEVEL 4. THE LEVEL OF THE FEATURE POT DOES NOT INCREASE THE CHANCE OF TRIGGERING THE BONUS. WHENEVER A BONUS IS AWARDED DURING BASE GAME AND FORTUNE BET, THE FEATURE POT LEVEL RESETS TO 1. IF A BLUE FEATURE SYMBOL LANDS IN VIEW, THERE IS A CHANCE THE BONUS MAY BE TRIGGERED WITH DOUBLE COINS FEATURE ACTIVE. IF A RED FEATURE SYMBOL LANDS IN VIEW, THERE IS A CHANCE THE BONUS MAY BE TRIGGERED WITH POT FILL FEATURE ACTIVE. IF A GOLD FEATURE SYMBOL LANDS IN VIEW, THERE IS A CHANCE THE BONUS MAY BE TRIGGERED WITH HAPPY BOOST FEATURE ACTIVE. LANDING MORE THAN ONE FEATURE SYMBOL INCREASES THE CHANCE THAT A BONUS WILL BE TRIGGERED. IF A SINGLE FEATURE POT IS TRIGGERED THE BONUS IS AWARDED WITH THAT FEATURE ACTIVE. IF MORE THAN ONE FEATURE POT IS TRIGGERED, THE BONUS IS AWARDED WITH THAT COMBINATION OF FEATURES ACTIVE. IF THE BONUS IS ACTIVATED WITH THE DOUBLE COINS FEATURE ACTIVE, THEN DURING THE BONUS SOME COINS WILL LAND AS DOUBLE VERSIONS. THESE ARE REPRESENTED BY 2 COINS HAVING THE SAME CASH VALUE OR 2 POT FILL COINS FOR A POT FILL IF THAT FEATURE IS ACTIVE. DOUBLE COINS COUNT AS 2 COINS WHEN DECREMENTING THE ROW ACTIVATION COUNTS. IF THE BONUS IS ACTIVATED WITH THE POT FILL FEATURE ACTIVE, THEN SOME COINS MAY LAND AS POT FILL COINS. IF POT FILL COINS LAND ON AN ACTIVE ROW, THESE ADD TO THE RELEVANT POT NAME, AND WHEN THAT NAME IS COMPLETE, THE VALUE OF THAT POT IS ASSIGNED TO THE TRIGGERING COIN. THE BONUS BEGINS WITH ALL POT NAMES EMPTY, AND IF A POT IS WON THEN ITS NAME IS RESET TO EMPTY. AFTER A POT FILL COIN HAS BEEN PROCESSED IT CHANGES TO A CASH COIN WITH A VALUE ON IT IF THE POT WASN'T AWARDED. THE POT VALUES ARE M-I-N-I, A FIXED VALUE OF 10X M-I-N-O-R, A FIXED VALUE OF 20X M-A-J-O-R, A FIXED VALUE OF 50X G-R-A-N-D, A FIXED VALUE OF 1000X IF THE DOUBLE COINS FEATURE IS ALSO ACTIVE THEN SOMETIMES A DOUBLE POT FILL COIN WILL LAND AND THIS WILL ADD TWO LETTERS INTO THE RELEVANT POT NAME. POT FILL COINS ON INACTIVE ROWS WILL BE AWARDED IF THAT ROW IS ACTIVATED. IF THE BONUS IS TRIGGERED WITH THE HAPPY BOOST ACTIVE, THEN AT RANDOM, THE VALUE OF ONE OR 00 MORE OF THE COINS CURRENTLY ON THE MINI REELS MAY BE BOOSTED. THIS BOOST CAN BE BETWEEN 2X AND 100X BASE GAME BET. ONLY COINS ON ACTIVE ROWS CAN BE BOOSTED. IF THE BONUS IS TRIGGERED WITH THE HAPPY BOOST ACTIVE, ONE OF THE OTHER FEATURES MAY ALSO BE TRIGGERED. POSSIBLE BOOST VALUES ON THE COINS ARE: 2X, 3X, 4X, 5X, 6X, 8X, 10X, 12X, 15X, 18X, 20X, 25X, 30X, 40X, 50X, 100X BASE GAME BET. BONUS GAME WHEN THE BONUS IS TRIGGERED THERE ARE INITIALLY 15 MINI REELS SHOWN IN A 5 BY 3 CONFIGURATION. ADDITIONALLY THERE ARE A FURTHER 4 ROWS OF 5 MINI REELS ABOVE THESE. THESE EXTRA ROWS ARE INITIALLY INACTIVE AND DISPLAY A COUNT INDICATING WHEN THEY WILL BE ACTIVATED. WHEN THIS COUNT REACHES ZERO, THE ROW IS ACTIVATED AND ANY CASH COINS PRESENT ON THIS ROW WILL BE AWARDED AT THE END OF THE BONUS. POT FILL COINS ARE AWARDED IMMEDIATELY. EVERY COIN THAT LANDS IN THE BONUS ON AN ACTIVE ROW DECREMENTS THE COUNT OF EACH INACTIVE ROW. WHEN THE COUNT OF A ROW REACHES ZERO, THAT ROW BECOMES ACTIVE. DOUBLE COINS DECREMENT THE COUNTS BY 2. THE INITIAL COUNTS FOR THE ROWS ARE, FROM THE BOTTOM, 10, 14, 18 AND 22. COINS WILL HAVE A CASH VALUE ON THEM, OR SOMETIMES POT FILL COINS IF THAT FEATURE IS ACTIVE. ALL 35 MINI REELS WITHOUT STUCK COINS SPIN IN EACH ROUND OF THE BONUS WHEN THE PLAYER PRESSES START. COINS WILL LAND AND STICK ON ALL REELS, BUT ONLY COINS ON ACTIVE ROWS ARE PROCESSED. POT FILL COINS LANDING ON AN INACTIVE ROW WILL STICK BUT WILL NOT BE PROCESSED. WHEN AN INACTIVE ROW IS ACTIVATED, ANY POT FILL COINS ON THIS ROW ARE THEN PROCESSED. WHEN COINS LAND, THEY STICK FOR THE REMAINDER OF THE BONUS. A POT FILL COIN WILL CHANGE TO A CASH VALUE ONCE THE LETTER HAS BEEN ADDED TO THE RELEVANT POT NAME IF THE POT IS NOT AWARDED. POSSIBLE CASH VALUES ON THE COINS ARE: 0.5X, 1X, 1.5X, 2X, 2.5X, 3X, 4X, 5X, 10X, 25X, 50X, 100X, 250X THE BONUS STARTS WITH 3 LIVES. FOR EVERY ROUND IN THE BONUS THIS COUNT DECREASES BY 1. IF A COIN LANDS ON AN ACTIVE ROW, THE NUMBER OF LIVES IS RESET TO 3. WHEN THE NUMBER OF LIVES REACHES ZERO, THE BONUS ENDS AND THE SUM OF ALL COINS ON ACTIVE ROWS IS AWARDED TO THE PLAYER. THERE ARE NO LINE WINS AWARDED DURING THE BONUS. THE BONUS WILL BEGIN WITH SOME COINS STUCK ON ACTIVE ROWS. IF THESE ARE POT FILL COINS, THEY WILL BE ADDED TO THE RELEVANT POT NAME AND THE COIN WILL CHANGE TO A CASH VALUE COIN. FORTUNE BET FORTUNE BET IS AN ALTERNATIVE TO STANDARD SPINS. ACTIVATING FORTUNE BET ALLOWS THE PLAYER TO SELECT A NEW BET MULTIPLIER OF 1.2X, 1.5X OR 2X THE CURRENT BET. AT 1.2X BET: MORE BONUSES WILL BE AWARDED THROUGH INCREASED ACTIVATION VIA THE PSEUDO MECHANISM. AT 1.5X BET: MORE BONUSES WILL BE AWARDED THROUGH INCREASED ACTIVATION VIA THE PSEUDO MECHANISM AND THEY WILL ALWAYS START WITH AT LEAST 2 OF THE BONUS FEATURES ACTIVE. AT 2X BET: MORE BONUSES WILL BE AWARDED THROUGH INCREASED ACTIVATION VIA THE PSEUDO MECHANISM AND THEY WILL ALWAYS START WITH ALL 3 OF THE BONUS FEATURES ACTIVE. THE AWARDS, GAMBLE ENTRY AND BIG WIN THRESHOLDS REMAIN THE SAME AS THE BASE BET. BONUS BUY THE BONUS BUY MODE CAN BE ACTIVATED BY PRESSING THE BONUS BUY BUTTON. BONUS BUY ALLOWS THE PLAYER TO PURCHASE DIRECT ENTRY TO A BONUS FEATURE. THE PLAYER CAN CHOOSE TO PURCHASE FROM 1 OF 2 OPTIONS. MYSTERY BONUS: THE PLAYER PURCHASES DIRECT ENTRY TO A BONUS FEATURE WITH ANY COMBINATION WHERE 1, 2 OR ALL 3 OF THE FEATURES WILL BE ACTIVE. TRIPLE BONUS: NIL THE PLAYER PURCHASES DIRECT ENTRY TO A BONUS FEATURE WHERE ALL 3 OF THE FEATURES WILL BE ACTIVE. THE BASE GAME BET CAN ALSO BE CHANGED AND CHANGING THIS ALTERS THE COST OF EACH BONUS OPTION. BUYING THE FEATURE RESULTS IN A BASE GAME DELIVERING THE 3 FEATURE SYMBOLS AND THEN THE RELEVANT FEATURE POTS WILL TRIGGER THE BONUS COMBINATION PURCHASED. FEATURE AWARDS ARE DETERMINED BY THE CURRENT BET. GAMBLE THE GAMBLE CAN BE CONFIGURED TO BE ON, ON IF THE WIN IS GREATER THAN OR EQUAL TO 5X BET, OR OFF. IF THE GAMBLE IS ON, ANY WINNINGS AWARDED DURING THE GAME WILL OFFER THE GAMBLE FEATURE. IF THE GAMBLE IS ON AND IF THE WIN IS GREATER THAN OR EQUAL TO 5X BET, ANY WINNINGS GREATER THAN OR EQUAL TO 5X BET AWARDED DURING THE GAME WILL OFFER THE GAMBLE FEATURE. THE GAMBLE ENTRY TRIGGER IS BASED ON THE BASE BET BEFORE ANY BET MULTIPLIERS ARE APPLIED. WHEN THE GAMBLE IS ENTERED, TWO WHEELS ARE PRESENTED. THE LEFT WHEEL OFFERS THE CHANCE TO INCREASE THE WINNINGS BY HALF, DOUBLE THE WINNINGS, OR TRIPLE THE WINNINGS. THE RIGHT WHEEL OFFERS THE CHANCE TO WIN A BONUS ENTRY. THE WIN VALUE CAN BE ADJUSTED BY PRESSING THE '-' AND '+' ARROWS, THIS IN TURN WILL ADJUST THE WINNING AND LOSING SEGMENTS OF THE WHEEL. PRESS EITHER WHEEL DURING THE FEATURE TO GAMBLE YOUR WINNINGS AND SPIN THE POINTER. IF THE POINTER STOPS ON THE GREEN SEGMENT, THE CASH VALUE OR BONUS ENTRY IS AWARDED. IF THE POINTER STOPS ON THE RED SEGMENT, THE PLAYER LOSES THEIR WINNINGS AND EXITS TO THE BASE GAME. WHEN THE POTENTIAL WINNING AREA OF THE GAMBLE WHEEL IS AT ITS MINIMUM SIZE THE CHANCE OF SUCCESS COULD BE SIGNIFICANTLY LESS THAN THE VISUAL REPRESENTATION SHOWN. IF THE RESULT OF A WINNING GAMBLE WOULD EXCEED THE WIN LIMIT OF 2500X BASE BET OR $250,000.00, WHICHEVER IS THE LOWER VALUE, WINNINGS ARE AUTOMATICALLY COLLECTED, AND THE GAMBLE WILL BE EXITED. A MAXIMUM AWARD OF HAPPY BOOST + POT FILL + DOUBLE COINS CAN BE ACHIEVED IN THE GAMBLE FOR BONUS WHEEL. ONCE THIS IS REACHED, THE BONUS WHEEL IS DISABLED, WHILE THE PLAYER IS STILL ABLE TO GAMBLE ON THE CASH WHEEL. PRESS THE 'COLLECT' BUTTON DURING THE FEATURE TO COLLECT YOUR WINNINGS. THE EXPECTED PAYBACK FOR THE GAMBLE GAME IS 100% SPIN CHANCE SPIN CHANCE IS TRIGGERED AUTOMATICALLY IF THE CREDIT IS REDUCED BELOW THE CURRENT PRICE OF PLAY, YET IS STILL GREATER THAN ZERO. SPIN CHANCE OFFERS THE PLAYER AN OPPORTUNITY TO WAGER THEIR REMAINING CREDIT IN AN ATTEMPT AT WINNING AN ADDITIONAL BASE GAME SPIN AT THE CURRENT BET. TO PLAY SPIN CHANCE, SELECT THE 'PLAY' BUTTON TO BEGIN MOVING THE POINTER ACROSS THE METER. ALTERNATIVELY, SELECT THE 'NO THANKS' BUTTON TO EXIT SPIN CHANCE WITH THE REMAINING CREDIT. IF THE POINTER STOPS ON THE GREEN SEGMENT, A SPIN AT THE CURRENT BET IS AWARDED AND PLAYS AUTOMATICALLY. IF THE POINTER STOPS ON THE RED SEGMENT, THE PLAYER LOSES ANY REMAINING CREDIT AND EXITS TO THE BASE GAME. THE RED AND GREEN SEGMENTS ARE PROPORTIONAL TO THE AMOUNT OF CREDIT IN RELATION TO THE CURRENT BET. THE CHANCE OF WINNING A SPIN IS DETERMINED BY THIS RATIO.
