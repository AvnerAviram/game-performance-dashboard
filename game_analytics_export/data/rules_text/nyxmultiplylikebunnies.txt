Multiply Like Bunnies RTP 96.5% VOLATILITY: 13.0 GAME RULES Malfunction voids all pays and play Select Bet Click the Spin button to spin the reel. This game is played at 243 ways. All 5 reels are always in play. Ways wins contain one symbol from each adjacent reel, beginning with the leftmost reel. The same symbol, or its substitute, in a different position on the same reel pays that ways win again. Only the highest win is paid for each ways symbol combination. Ways wins are multiplied by the bet multiplier. Scatter symbols award in any position on the reels. During the base game, Wild Symbols 1, 2, and 3 are wild and substitute for all symbols except Free Games Bonus symbol. Wild Symbols 1, 2, and 3 appear only on reels 2, 3, and 4. During the base game, Wild Symbols 1, 2, and 3 appearing in any position on the reels awards a 3x, 4x, or 5x multiplier value. If more than one Wild Symbol 1, Wild Symbol 2, or Wild Symbol 3 appears, the sum of the multiplier values is awarded. FREE GAMES BONUS During the base game, 3 Free Games Bonus symbols starts the Bonus Select. The Bonus Select features 3 options: 1 – 3 Free Games Awarded with 9x, 15x, and 24x available multipliers. 2 – 7 Free Games Awarded with 5x, 6x, and 9x available multipliers. 3 – 12 Free Games Awarded with 3x, 4x, and 5x available multipliers. Free Games Bonus symbol is a scatter symbol. The Free Games Bonus begins immediately afterwards. The Free Games Bonus does not award additional free games. The Free Games Bonus ends with 0 free games remain. Free Games Wild Symbols 1, 2, 3, 4, 5, 6, and 7 are wild and substitute for all symbols Free Games Wild Symbols 1, 2, 3, 4, 5, 6, and 7 appear only on reels 2, 3, and 4. During the Free Games Bonus, Free Games Wild Symbols 1, 2, 3, 4, 5, 6, or 7 appearing in any position on the reels awards x3, x4, x5, x6, x9, x15, or x24 multiplier value.
