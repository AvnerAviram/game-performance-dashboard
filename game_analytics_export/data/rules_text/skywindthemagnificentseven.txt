The Magnificent Seven HOW TO PLAY PLAY MODE AND BALANCE • You can play the game in either FUN or REAL mode. • In fun mode, you are playing just for the fun of the score. In real mode, you are playing with money from your real money balance. • Your balance is shown in the BALANCE box both in fun and real mode. PLACING BETS • This game is played with a fixed number of active paylines. The BET is the COIN value multiplied by the number of active paylines. • To set the bet amount, use the '-' and '+' buttons. CONTINUOUS SPIN • Clicking and holding the SPIN button for continuous spinning. To stop the continuous spin session, release the SPIN button. • When CONTINUOUS SPIN is on, some win animations and sound are skipped, the reels spin faster and the game results are displayed faster. AUTOPLAY MODE • To select the number of autospins, use the AUTOPLAY options panel. • Clicking the OK button initiates the autoplay session. Clicking the CANCEL button closes the panel. • You can end the autospins session earlier by clicking the AUTOPLAY button. • The number of remaining spins is displayed on the AUTOPLAY button. TURBO MODE • When TURBO Mode is on, some win animations and sounds are skipped and the reels spin faster. • Clicking the TURBO button switches Turbo Mode on or off. PAYTABLE PAYOUTS • Winning combinations start from the leftmost reel and the symbols have to be consecutive along a payline. • The BET is the COIN value multiplied by 40. • Only the highest win pays on each payline. • Coinciding wins on different paylines are added. • The maximum win in the game has an upper limit. For more information see the TERMS AND CONDITIONS section. FREE GAMES • During FREE GAMES the reels are spun automatically using the same COIN value as per the spin that won the FREE GAMES.
