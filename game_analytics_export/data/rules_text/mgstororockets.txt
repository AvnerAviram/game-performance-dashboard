Toro Rockets About Toro Rockets® Toro Rockets® is a 3x5 reel, 243 ways slot game. Return to Player (RTP) General Wild Symbol Bonus symbol Rocket Symbol Rocket Collection Bonus Autoplay Return to Player (RTP) Return to Player (RTP) is the theoretical statistical percentage of total money bet by players in a specific game, that is paid out as winnings over time. The RTP is fixed and does not change. This game has an RTP of 95.77%. General All 243 payways are permanently enabled. Minimum bet per spin is at 20 Coins. If you have a winning combination on the payway, the win is highlighted on the reels. Each symbol can be used only once per winning combination. Highest win paid per winning combination. All wins begin with the leftmost reel and pay left to right on consecutive reels, except BONUS which pays any. For providing more information to the player in calculating the payouts, this game has a dynamic paytable. The payouts displayed in the paytable for all the symbol combinations are the final amounts in credits which is already multiplied by active stake and paid to the player. The game ends when there is no winning combination after which you can play again. Malfunction voids all pays and plays. Autoplay You can also spin a set number of times in succession without interacting with the game, using Autoplay either in base game or inside menu AUTOPLAY option. You must select a bet amount before you start Autoplay, as it plays with the current bet settings. Tap/Click Autoplay On in base game. Tap/Click the number of spins you want to play. The Autoplay session automatically starts playing. To stop Autoplay, Tap/Click Stop . You can also start the Autoplay from the option available inside menu option. Start button will remain disabled until you select any number of spins. Custom Mode Autoplay is also available with Custom Mode where some Autoplay Settings are available. Autoplay Settings Coin Size You can select a coin size before you start Autoplay, as it plays with the last bet value. Coins Bet Coins Bet is the number of coins allowed per line. Coins Bet differ between games. Total Bet Total Bet displays the Total Bet value. Total Bet = (Coin Size * Coins Bet * 20 credits). Spins You can set the number of times the slot reels spin during your Autoplay session. Win Limit Stop spins if the number of credits won on any spin exceeds or equals the amount selected. Loss Limit Stop spins if Autoplay exceeds the loss limit value with the next bet. Until Stop Continue Autoplay until you Tap/Click the Stop button or stops if Free Spins or another bonus feature is triggered.
