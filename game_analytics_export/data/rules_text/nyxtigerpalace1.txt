Tiger Palace Free Games Bonus 12 Free Games are triggered by the appearance of 3 . During Free Games, 3 are required to retrigger 12 more Free Games. The Respin Feature is not active during Free Games. Free Games end when 0 Free Games remain or after 1000 Free Games have been played. Alternate reels are used during the Free Games feature. The bet, lines played, and pays during the Free Games feature are the same as the initiating base game. Progressive Jackpots It is not technically possible to have two simultaneous wins of the same Progressive Jackpot. There are separate Progressive Jackpots per bet multiplier. Contributions made to Progressive Jackpots at a certain bet multiplier are added to Progressive Jackpots that can only be achieved by making a bet at that same bet multiplier. The following contributions are made to Progressive Jackpots on each base game spin: .12% of the bet is contributed to the Mega Progressive Jackpot. .2% of the bet is contributed to the Major Progressive Jackpot. At a 1x bet multiplier, the start-up value for each Progressive Jackpot is: Mega: $100.0 Major: $15.00 The start-up values for the Progressive Jackpots at bet multipliers other than 1x are calculated by multiplying the bet multiplier by the start-up value at a 1x bet multiplier. There is no upper limit on Progressive Jackpot values. When the Progressive Jackpot is awarded and a re-trigger of the jackpot occurs, the patron is awarded a static amount. Click on the Autoplay button and select the number of Autoplay game rounds. Autoplay session — The number of Autoplay game rounds selected. Loss Limit — set the maximum loss for the Autoplay session — if this value is met or exceeded, the Autoplay session will stop. Single Win Limit — set a win limit for the Autoplay session — if this value is met or exceeded, the Autoplay session will stop. Stop On A Progressive Jackpot (if enabled) — choose if the Autoplay session should stop / not stop if a Progressive Jackpot is won. This option will only be displayed on games offering Progressive Jackpot(s). Bet — amend the Bet value, if required. The Start Button will populate with the selected number of Autoplays when a permissible combination of settings has been selected. Certain Autoplay settings are mandatory for some jurisdictions.
