Wilds Deluxe Wilds Deluxe is a 5-reel slot game with 10 pay lines. Wins are awarded for adjacent 3, 4 and 5 of a kind symbols. Vault symbols are wild and substitute for all symbols. If a vault is included in a winning combination, the vault door opens and reveals a win multiplier. If no substitution occurs, a win of x1 bet is paid for each symbol in view. Vault symbols only appear on reel 3. Vault symbol multipliers are: x1, x2, x3 and x7. The theoretical average return to player (RTP) is 96%. LOSS LIMIT: Select an amount of money, which if your losses equal or exceed this amount, autoplay will be stopped. SINGLE WIN LIMIT: Select an amount of money, which if a win in any one spin equals or exceeds this amount, autoplay will be stopped.
