Thai Flower Untitled Document Game Introduction Thai Flower is an oriental themed five reel slot game that features free games There are 10 win lines which pay for 2 or more symbols on a win line. How to play Select your stake - If you wish to adjust this, click the STAKE/LINE '+' and '-' buttons to increase or decrease the size of your stake per line. Spin the reels - Click the SPIN button to start your game. There is an auto play function available that will start each spin automatically until either something occurs that requires your intervention or the number of set spins expires. Select your desired number of spins by clicking on the numbered buttons that are revealed when you press AUTOPLAY. Click 'Stop' to cancel auto play. Help All line win amounts are shown in the games pay table as multiples of the stake per line. These can be viewed by pressing the HELP button and are displayed as cash values in the game which change as the stake per line is adjusted. The theoretical average return to player (RTP) is 95%. This RTP represents the long-term expected payback of the game which has been calculated and verified by an independent testing company. Malfunction voids all pays and play.
