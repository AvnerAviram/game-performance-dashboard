Colossus: Hold & Win 1.Overview The goal is to obtain a winning combination on a winning way spread across the reels. Game specifications: Type Slots Number of reels 5 Number of rows 3 Number of Lines 20 (cannot be changed) Number of coins you can bet 1 - per line (cannot be changed) Progressive No Double feature No Auto spin feature Yes Wild symbols Yes Multiplier symbols Yes Scatter symbols Yes Bonus Round Yes Free s pins No Instant win No RTP Standard Play: 96.00% The game disconnects if you do not take any actions for a specific number of consecutive minutes (this number varies depending on jurisdiction); the session recovery feature will save the game progress so that the next time the game is loaded, you can continue where you left off. In the event of technical difficulties resulting in a dropped connection to the game server, the game you were playing is saved automatically and will load the next time you establish a connection to the game server. You will not be able to reconnect to the game server for the first 30 seconds after the disconnection. Note : In case of malfunction all pays and play are voided; any bet that was not yet accepted will be voided, any bet that was unsettled will be refunded. The maximum win from a single game is £250,000 2. Wild The Wild symbol replaces any other symbols, except the Gold Nuggets and Gems , to form winning combinations. Note : If no specific payouts for Wild are displayed in the pay table, a combination formed exclusively with Wild will pay just like the highest paying symbol, if that combination is the highest winning combination on a line. 3.2. Gold Nuggets Landing 5 or more Gold Nuggets will trigger the Gold Link Respins. 3.3. Gold Link Respins Landing 5 or more Gold Nuggets will trigger the Gold Link Respins feature, that will award 3 respins. During the Gold Link Respins , any Gold Nuggets that land into view will stick and will reset the respins. The Gold Link Respins feature is over once all respins have been used or all available positions are filled with Gold Nuggets . Note : The values displayed on the Gold Nuggets are shown in the selected currency/coin. 3.4. Reel Boosters During Gold Link Respins different Gems can land on the reels. Collecting 3 Gems of the same kind will trigger a booster. Reels Boosted Collecting 3 Blue Gems will unlock an extra row. The maximum number of rows you can unlock is 3. Values Boosted Collecting 3 Green Gems will increase all landed values to the next possible value (i.e 0.20 > 0.40 > 0.60 etc.) Cash Boosters can only be collected a maximum of 3 times in a respins round, once this is reached no more Cash Boosters will be awarded. Spins+ Boosted Collecting 3 Red Gems will award an extra Respin for the remaining of the round, and every time the counter is reset it will be with the boosted spin amount. Multiplier Boosted Collecting 3 Purple Gems will increase the multiplier level. The first 3 Purple Gems will award the 2x multiplier, the next 3 will award 5x multiplier and the final 3 will award 10x multiplier. You can not win any other Purple Gems after winning the 10x multiplier. Possible Values during Respins are as follows in times bet values: x1, x2, x3, x5, x7, x10, x15, x25, x30, x40, x50, x100, x250, x375, x500, x750, x1000 & x1,250. If any Gold Nuggets land or are boosted to the max amount (x1250 time total bet) no increases will be made. Any multiplier won during Respins will be applied to the final win amount at the end of the round. 3.5. Gold Digger Modifiers During the normal spins Gold Diggin' Gus has a bunch O'tools to help you find those winnings: Dynamite - turns from 3 to 5 symbols Wild; Pickaxe - can transform random symbols into Gold Nuggets; TNT Plunger - explodes and upgrades low pay symbols in view (A,K,Q,J) and transforms them into one randomly chosen high pay symbol (picture symbols) for the current spin. 4. Auto spin feature The loss limit setting allows stopping the auto spin feature automatically when the selected limit, plus your cumulative winnings, and minus your cumulative bets, is less than 0. 4.1. Desktop version Click the AUTO SPIN to open the AUTO SPIN settings window where you can set up the following: Number of auto spins : click + or - to select the number of auto spins to play. Click the ACTIVATE to close the AUTO SPIN settings window and to start the auto spin feature with the currently selected bet and auto spin setup. Click the CLOSE to close the window without starting the auto spin feature. Click the STOP AUTO SPIN button to stop the auto spin feature at any time. 4.2.
