Note: Turning Totems Video slot is referred to as Turning Totems, The Game, or Game. 1. Turning Totems is a video slot with 5 reels and 11 fixed pay lines with two way wins. The game features Thunder Reels, Sleeping Wilds, Thunder Snakes, respins and a bonus game. The theoretical return to player is 96.1%. 2. A new game round is started with the action button and plays the game with the selected bet level. 3. No bets can be altered during a game round. 4. Auto play plays the game automatically for the number of game rounds selected in the auto play page. 5. The Thunder Snake symbol turns the reel it lands on into a Thunder Reel for the rest of the game round. 6. A spin that brings the number of Thunder Reels to 3 or more awards a bonus game of 5 free spins. Other spins where reels are turned into Thunder Reels award one respin. 7. The Sleeping Wild is a symbol that acts as a wild when it lands on a Thunder Reel. The Sleeping Wild acts as a normal symbol in other cases. 8. The wild symbol substitutes for all symbols except the Thunder Snake. A payline consisting of only wilds will have higher payout, according to the pay table. 9. All respins are played with the same bet as the spin they were triggered by. 10. All game payout and win combinations are paid out according to the pay table. Only the highest win per pay line, in each direction, is paid out. 11. All wins are presented and paid in the selected currency. 12. In the event of game malfunction all affected bets and pays are rendered void. Last modification date: 8/29/2017
