Baker Street Detectives GAME RULES SELECT BET. START THE GAME BY PRESSING SPIN BUTTON. GAME HAS FIXED 50 LINES ALL WINS ARE MULTIPLIED BY BET PER LINE EXCEPT SCATTER. WINS ON DIFFERENT LINES ARE ADDED. FREE SPIN WINS ARE ADDED TO SCATTER AND PAYLINE PAYS. ONLY HIGHEST PAYING COMBINATION FOR EACH SYMBOL IS PAID PER SPIN. WINNING COMBINATIONS START FROM THE LEFTMOST REEL AND THE SYMBOLS HAVE TO BE CONSECUTIVE ALONG A PAYLINE EXCEPT SCATTER, WHICH PAYS ANY. DURING FREE SPIN BONUS, ALL THE PAYS ARE THE SAME AS THE BASE GAME AND THE NUMBER OF LINES AND THE BET PER LINE STAY THE SAME AS WHEN THE BONUS WAS TRIGGERED. MALFUNCTION VOIDS ALL PAYS AND PLAYS. METERS AND BUTTONS BET : Shows the value of bet of the current settings. WIN : Displays the win for the current or last win paid. BALANCE : Displays the balance amount of the player. MALFUNCTION: In case of any malfunction of hardware or software during the game play, all current game bets and payouts are considered void and bets affected are refunded.
