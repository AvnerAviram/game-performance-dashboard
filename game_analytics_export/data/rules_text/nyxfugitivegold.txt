Fugitive Gold RTP: 96.0% VOLATILITY: 18.7 - HIGH GAME RULES This game is played at 243 ways. Ways wins contain one symbol from each adjacent reel, beginning with the leftmost reel. The same symbol, or its substitute, in a different position on the same reel pays that ways win again. Only the highest win is paid for each ways symbol combination. Ways wins are multiplied by the bet multiplier. Scatter symbols award in any position on the reels. Wins are shown in credits unless marked as currency. FREE GAMES 3 Bonus Symbols appearing in any position on reels 2, 3, and 4 starts the Free Game Bonus. 3 Bonus Symbols award 10 free games. Bonus Symbols appear only on reels 2, 3, and 4. The Free Game Bonus uses different reels. Free games automatically play on the same ways at the same bet multiplier as the game that initiated the Free Game Bonus. The Free Game Bonus does not award additional free games. The Free Game Bonus ends when 0 free games remain. Each Wild Symbol that appears on reels 2, 3, or 4 expands vertically, and fills all positions on that reel. Wild Symbols are wild and substitutes for all symbols. Wild Symbols appears only on reels 2, 3, and 4. Malfunction voids all pays and plays. Last modification date: 4/12/2023
