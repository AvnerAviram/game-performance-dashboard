Yahtzee GAME DESCRIPTION YAHTZEE is an Instant Tap game featuring five dice. Prizes are awarded if the dice make a winning combination when rolled. HOW TO PLAY The game screen consists of five dice, six win combination panels and 18 unique prize panels. Tap the PLAY button to roll the dice. There are six win combinations in the game. Each win combination awards a different number of prizes. A maximum of 5 unique prizes can be won in a game round. Each game round consists of one roll of the five dice. END OF GAME Any winnings will be displayed at the end of the game round in the WIN meter. To play again, select the PLAY button. To change the STAKE, select the plus or minus buttons. Once the PLAY button has been selected, the STAKE will be deducted from the current balance and a new game will begin. GENERAL INFORMATION System malfunction voids all pays and all plays. This game is a game of chance. The outcome of a play in the game is pre-determined. Actions or choices made by player do not affect the outcome of the game.
