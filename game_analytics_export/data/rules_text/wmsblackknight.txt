Line pays must occur on adjacent reels, beginning with the leftmost reel. Feature Description 3 or more scattered FEATURE symbols on adjacent reels, beginning with the leftmost reel, trigger the Feature. 7 free spins are awarded! Click the Start Feature button to start the free spins. WILD only appears on reels 2, 3, and 4 and is Wild for all symbols. WILD expands to fill the entire reel and is held for remaining spins. All line pays are evaluated after the WILD expands. Each expanded WILD is evaluated as one FEATURE. The Feature cannot be retriggered. During free spins, an alternate set of reels is used. Winning combinations for these reels are identical to the base game. During Free Spins, the bet per line and the active paylines remain the same as the spin that triggered the feature. All scatter pay monetary awards have already been multiplied by the total bet. Scatter pays are added to the payline wins. Paylines Wins on multiple paylines are added together. Total line bet must be divided equally between each active payline; therefore, all total bet amounts are not available. The total bet is the number of lines multiplied by the cash bet on each line. Only active paylines can register wins. Malfunction voids all pays and plays. Last modification date: 3/27/2017
