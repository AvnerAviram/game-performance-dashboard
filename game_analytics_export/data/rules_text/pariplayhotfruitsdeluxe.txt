Hot Fruits Deluxe Game Overview RTP: 97.25% Hot Fruits Deluxe is a 3 reel, 3 row fruit slot featuring Stacked symbols and Wild substitutions. The game is played with 5 lines (fixed). Wins pay only from left to right. All wins only occur on active lines. Only the highest line win is paid on each line played. wins from different lines are added. If a win of nine same symbols is achieved, the resulting win will be doubled up. The Hot Fruits symbol substitutes every other symbol. All values in the paytable are based on the current bet. These values will be changed automatically if the bet will be changed. Game Functions NOTE: It is also possible to start a game or collect a win by pressing the SPACEBAR. Game Settings Options Additional Information Malfunction voids all pays and plays. In the info panel on the top of the game on the right side there is the game id visible. Each game has its unique game id. This game id should be used for requests to the support. In case of disconnection because of network issues or other failures the game will always continue on the last state where it was interrupted. The interrupted game must be completed within 90 days, after which it will be terminated automatically. The maximum win from a single game is £500,000
