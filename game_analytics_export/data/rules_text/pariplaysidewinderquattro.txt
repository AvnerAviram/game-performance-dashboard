Sidewinder Quattro GAME RULES Sidewinder Quattro will dazzle you with an extraordinary feature: shuffling reels! After a losing spin it is possible that all reels will swirl around and land on different positions to create winning combinations! And there’s a multiplier that increases after each shuffle, making possible next winnings even higher! In case one game is not enough, you can play 2, 3 or 4 games simultaneously in Quattro mode! SIDEWINDER FEATURE The Sidewinder Feature can be triggered on any losing spin. The reels will shuffle positions during the feature. Each shuffle of the reels increases the multiplier by one step. If a win is not produced then the feature will end. Click/Tap the SPIN or AUTOPLAY buttons to start playing. Activate MAX BET to play with the maximum allowable TOTAL BET settings. Use the – and + BUTTONS to customize your bet settings. The theoretical minimum payback percentage for this game is 96.01% . Disclaimer: If an interrupted game is not finished within 90 days, your game will be terminated. If your game gets terminated you will lose every active bet. Only the highest winning combination per active win line pays. Malfunction of the virtual machine voids all plays and payment during the period of malfunction. MAIN MENU GAME SETTINGS INTRO SCREEN The default setting for the INTRO SCREEN is ON. When set to OFF, the INTRO is not visible. ANIMATIONS The default setting for ANIMATIONS is ON. When set to OFF, special animations are not visible. Win animations and symbol animations are always visible. SPACEBAR SPIN The default setting for SPACEBAR SPIN is ON. When set to OFF, it is not possible to spin the reels using the SPACEBAR key. GAME SOUNDS The default setting for GAME SOUNDS is ON. When set to OFF, only the SOUND EFFECTS are disabled. BACKGROUND SOUNDS The default setting for BACKGROUND SOUNDS (if available) is ON. When set to OFF, only the BACKGROUND MUSIC is disabled.
