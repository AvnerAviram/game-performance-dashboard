Eggcelent Wins User Interface ‘+’ / ‘-’ buttons increase / decrease the bet value — the bet value is the total bet per game round. Starts a game round at the current bet value. Displays the autoplay options. Stops the autoplay session. The numerical value displayed within the button shows the number of autoplay game rounds remaining. Bet Deﬁnitions Current Bet: The current bet is the value that appears above . Minimum Bet: The minimum bet is the lowest possible bet value, which is selected by pressing until the bet value no longer decreases. Bet Multiplier: The bet multiplier is defined as the current bet divided by the minimum bet. Autoplay Options Autoplay will allow game rounds to be played automatically. Click on the Autoplay button and select the number of Autoplay game rounds. Additional Information The overall return to player is 96.75%. Malfunction voids all pays and plays. Wins A winning play occurs when the fox's arm steals an egg from the chicken. 3 to 14 prizes and up to 2 Multipliers can be awarded on a winning play, or a chance to win a Jackpot if any combination of 3 are awarded at the same time. Multiplier Feature If 1 or 2 of are awarded on a winning play, all prizes will be multiplied by the corresponding Multiplier. Jackpot If 3 Multipliers are awarded on a winning play, a Jackpot is awarded according to the following table: Multiplier Combination Jackpot $375.00 $250.00 $187.50 $150.00 If a Jackpot is awarded, no additional prizes can be awarded.
