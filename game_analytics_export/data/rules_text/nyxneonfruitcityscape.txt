Neon Fruit Cityscape Rules This game has an RTP of 96% You play a fixed 20 lines per spin with 1 coin per line Only the highest win on any winline is paid Winning combinations and payouts are made according to the paytable Malfunction voids all pays and plays Features This game features Wilds, x2 Wilds, and Scatter symbols Normal Wilds substitute for any basic symbols and will pay as the highest win on any winline x2 Wilds work like normal Wilds, but also double the payout of any winline they are part of Multiple x2 Wilds are cumulative Getting three x2 Wilds appearing on a winline, the payout would be the value of the winline x8 x2 Wilds only appear on reels 3, 4 and 5 x2 Wilds will only appear on reels if they are part of a winning combination. The Freespins Feature is achieved via Scatter symbols that do not need to be on winlines to activate the feature The maximum win from a single game is £250,000 Last modification date: 6/4/2020
