BetMGM Lion's Gold Stepper User Interface ‘+’ / ‘-’ buttons increase / decrease the bet value — the bet value is the total bet per game round. Starts a game round at the current bet value. Displays the autoplay options. Stops the autoplay session. The numerical value displayed within the button shows the number of autoplay game rounds remaining. Bet Deﬁnitions Current Bet: The current bet is the value that appears above . Minimum Bet: The minimum bet is the lowest possible bet value, which is selected by pressing until the bet value no longer decreases. Bet Multiplier: The bet multiplier is defined as the current bet divided by the minimum bet. Autoplay Options Autoplay will allow game rounds to be played automatically.
