7's on Fire Power Mix How to play Select your stake If you wish to adjust this, click the STAKE '+' and '-' buttons to increase or decrease the size of your stake. Spin the reels Click the SPIN button to start your game. Info All line win amounts are shown in the games pay table as multiples of the total stake. These can be viewed by pressing the HELP button and are displayed as cash values in the game which change as the stake per line is adjusted. Winlines The default win line selection is 5 lines which is fixed for every game. WILD 7's The WILD 7’s is a wild symbol and substitutes for all symbols. The WILD 7’s can appear on any reel. When all reels have come to a stop any WILD 7’s transfer to the same position on all other reels. In the event that a WILD 7 is already present then it will be assigned a win multiplier based on the number of WILD 7’s that transfer into that position. Base Game Awards The awards can be viewed in the game using the Help button. Any wins will be automatically credited to your account.
