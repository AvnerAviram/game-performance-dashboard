Gummy Galaxy Gummy Galaxy is a 5x3, 10-payline videoslot which pays left to right, right to left or in the middle, and which features multiplying wilds which can stack on three of the five reels. The reels have 9 symbols, 8 regular symbols and the Wild symbol. The player is paid for matches of 3 or more symbols landing on adjacent reels. Each payline is evaluated and paid from left to right, or right to left, or in the middle of the reels depending on which of these provides the largest possible payout. Only the highest matching combinations per winning line is paid. Simultaneous or coinciding wins on different paylines are added and paid. Game pays and plays on total bet. The actual payouts equals the sum of the dynamic paytable values corresponding to each line won. Game is plays on 10 fixed paylines. The bet level is set using the BET LEVEL selectors. AUTOPLAY automatically plays the game for the selected number of rounds. GAME FEATURES Multiplying Wilds The wild exists on reels 1, 3 and 5. The Wild symbol substitutes any symbol to create the highest possible win on a payline. When landing on the reels, the Wild symbol will reveal itself to be a x2, x3 or x7 wild multiplier. All winnings with a substituting Wild Sun Gold symbol are multiplied with the corresponding multiplier. Wins with the same Wild symbol on different paylines have the same multiplier. In the case of more than one multiplier in a win, the multipliers are added together. Malfunction voids all plays and pays. Last modification date: 3/13/2024
