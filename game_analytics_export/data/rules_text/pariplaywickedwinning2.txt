Press the PLAY button to start the FREE GAMES Feature. Bet and reels played are the same as the game that triggered the feature. GAME RULES All Pays REEL POWER cost is 50 coins. Bet is 50 coins x Coin Size. All wins begin with the leftmost reel and pay left to right on adjacent reels, except scatters. Winning symbols can occur anywhere on all reels. Each symbol can be used once per winning combination. Only positions containing the winning symbol are used in determining the win for that symbol. Each winning combination is made up of only one symbol from each reel. Coinciding wins are added. Highest win paid per winning combination. Scatter wins are added to REEL POWERTM wins.
