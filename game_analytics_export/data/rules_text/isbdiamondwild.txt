Diamond Wild 1. Overview The goal is to obtain a winning combination on a winning line spread across the reels. Game specifications : Type Slots Number of reels 5 Number of rows 3 Number of lines 1 to 20 Number of coins you can bet per line 1 to 10 Progressive Yes Double feature No Autospin feature Yes Wild symbols Yes Multiplier symbols No Scatter symbols No Bonus round No Free s pins Yes Instant win No RTP 93.20% to 95.20% Jackpot Progressive The game disconnects if you do not take any actions for a specific number of consecutive minutes (this number varies depending on jurisdiction); the session recovery feature will save the game progress so that the next time the game is loaded, you can continue where you left off. In the event of technical difficulties resulting a dropped connection to the game server, the game you were playing is saved automatically and will load the next time you establish a connection to the game server. You will not be able to reconnect to the game server for the first 30 seconds after the disconnection. Note : In case of malfunction all pays and play are voided; any bet that was not yet accepted will be voided, any bet that was unsettled will be refunded. 2. How to Play Note: If you do not have enough money to play a game, the bet can be set to the minimum selectable amount through a dialog window informing about this. All winning lines pay from left to right, except Free Games symbols and Diamond symbols. Three or more Diamond symbols that appear anywhere on the reels pay according to the paytable or pay a progressive jackpot if played with maximum coins and maximum lines. Wins on different selected lines are all added. Only the highest winning combination on each selected line is paid. Diamond wins (three or more Diamond symbols anywhere in view) are added to the wins, displayed in the winnings meter and added to the game balance. 2.1. Flash version Note : On the info bar located under the main game buttons, you can view the current BET, your BALANCE and your current WINNINGS, in your currency. To play, follow these steps: Use the coin value select button to choose a coin value. The coin value select button is located in the bottom-left side of the main game screen and shows the currently selected coin value. Use the LINE BET button to choose the amount of coins you want to bet on each active line. Use the LINES button to choose the number of active lines. Note : The current bet amount, in your currency, is displayed in the BET box. Click SPIN to place the bet and spin the reels once. You can click the BET MAX button to place the maximum bet (maximum number of coins per line and maximum number of lines) for the currently selected coin value and spin the reels once. The total amount won, in your currency, according to the paytable, is shown in the WINNINGS box. For each winning combination, the amount won is the amount shown in the paytable for that winning combination multiplied by the coin value and by the line bet. 3. Special Symbols and Features 3.1. Wild The Wild symbol substitutes for all symbols except Free Games and Diamond. 3.2. Free games Three or more Free Games symbols anywhere on the reels award 8 free games with winnings doubled. Free games can be won during Free games. The winnings from the Diamond symbol are not doubled during the free games. The bet amount and lines played in the spin that triggered the free games remain the same for the awarded free games and cannot be changed until the free games are completed. 3.3. Progressive Jackpot Note : You cannot win the Progressive Jackpot during free spins. Five or more Diamond symbols appearing anywhere on the reels at max bet (maximum number of coins and lines) qualify for the Progressive Jackpot winnings. Any other type of bet (not max bet) pays according to the paytable. With every bet, the base progressive jackpot value (the progressive pool) is increased with 0.5% of that bet and with 0.25% of that bet for the 8x and 9x diamond symbol jackpots. Note : The base progressive value (progressive pool) is a value stored in Euros and it is associated to a coin value of 10 Euros (which may not be available for you to select in the game): 5x diamond symbol – 20,000.00 seed + any 0.5% increments from all bets 6x diamond symbol – 100,000.00 seed + any 0.5% increments from all bets 7x diamond symbol – 200.000.00 seed + any 0.5% increments from all bets 8x diamond symbol – 2,000,000.00 seed + any 0.25% increments from all bets 9x diamond symbol – 5,000,000.00 seed + any 0.25% increments from all bets The Progressive Jackpot counters display the progressive jackpot value for the currently selected coin value. The displayed jackpot value is proportional to the selected coin value and is a portion of the progressive pool, calculated depending on the currently selected coin value. Note : The maximum available progressive is paid when playing with the maximum available coin value. Jackpot Seed examples (in your currency): Coin value 0.01 5x Diamond 6x Diamond 7x Diamond 8x Diamond 9x Diamond Jackpot Seed Value 20 100 200 2000 5000 Coin value 0.02 5x Diamond 6x Diamond 7x Diamond 8x Diamond 9x Diamond Jackpot Seed Value 40 200 400 4000 10000 The Jackpot value is refreshed once every 3 seconds. Depending on the clients’ internet service provider as well as their hardware, server-to-client delays will vary from player to player and from message to message. Between two successive refreshes of the Progressive Jackpot counter, it is possible for two or more players to win the jackpot in close succession. In this case, the first player that wins the Jackpot is awarded the Progressive Jackpot amount in full, while the subsequent winner is awarded the minimum guaranteed Progressive Jackpot, as well as 0.5% of the bets placed between the two successive Jackpot wins. 4. Autospin feature The loss limit setting allows stopping the autospin feature automatically when the selected limit plus your cumulative winnings minus your cumulative bets is less than 0. Note : The autospin feature does not automatically start free games – player input is required. 4.1. Flash version Click the AUTO SPIN button to open the AUTO SPIN SETTINGS window, where you can set up the autospin feature. On the AUTO SPIN SETTINGS window, you can set up the following: Loss limit – enter a number for the loss limit value or use the left and the right arrow buttons to select a number for the loss limit value Stop if WIN reaches – to enable this setting, click the button on its left side and then enter a number for the WIN value or use the left and the right arrow buttons to select a number for the WIN value; click the button on its left side again to disable it Stop if JACKPOT is won - (only for games with Progressive Jackpots or with a specific Jackpot value displayed) - to enable this setting, click the button on its left side; click the button on its left side again to disable it Number of AUTO SPINS to play – use the left and the right arrow buttons to select the number of autospins Click START to close the AUTO SPIN SETTINGS window and start the autospin feature with the currently selected bet and number of autospins or click CANCEL to close the AUTO SPIN SETTINGS window without starting the autospin feature. Click the STOP xx/yy button to stop the autospin feature at any time; xx represents the remaining number of autospins and yy represents the number of autospins to play that you selected on the AUTO SPINS SETTINGS window. 5.
