Hyper Burst Welcome to Hyperburst Video Slot with Hyper Respins, Hyper Mode, Multipliers and Colossal symbols. ABOUT THE GAME Hyperburst is a 6x4 Video Slot with 25 paylines that pays out when 3 or more identical symbols land on a payline starting from the leftmost reel. This slot features 4 high pay symbols and 4 low pay symbols. There is a wild symbol that substitutes all regular symbols and has its own payout. All symbols can land as colossal symbols up to 4x4 in size. Hyper Respins Winning on a payline triggers a respin where 2 center reels have 2x2 colossal symbols. Winning in the first respin awards another respin with 3x3 symbols. Winning the second respin (3 total wins in a row) awards Hyper Mode. Hyper Mode Hyper Mode is triggered by winning on 2 respins in a row (3 spins in total). It starts with an x2 multiplier which can be increased up to x10 by winning consecutive respins. Hyper Mode features 4x4 colossal symbols on the center reels. Hyper Mode respins are retriggered by winning. Wilds Wilds can substitute for any regular symbol. Wilds have their own payouts which are listed in the paytable. If the wild’s payout is more than the possible substitution, then the wild does not substitute for the regular symbol instead it pays out itself. HOW TO PLAY Select your coin denomination. Press the Spin button to start the game. HOW TO CALCULATE PAYLINE WINS To calculate your total payline coin win, calculate all payline wins and add them together. To calculate a single payline win, count the number of identical symbols lined up in sequence on the line starting from the left most reel. If three or more identical symbols are lined up, find the coin win value in the dynamic Pay Table. There you can find the value for all symbols for 3 in a row, 4 in a row, 5 and 6 in a row. Only the highest winning combination per line is paid out. If the winning combination starts with three Wilds, there can be many ways to interpret the win. In this scenario, only the highest win combination is paid out. Calculate the total win in the home currency by multiplying the total coin win with the coin value.
