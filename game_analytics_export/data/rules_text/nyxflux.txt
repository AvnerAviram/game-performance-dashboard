Flux Untitled Document Game Rules Note: Flux video slot is referred to as Flux, The Game or Game. Flux is a video slot with 5 reels and 15 fixed pay lines. The game features bonus games with extra symbol rows and pay lines active two ways. The theoretical return to player is 96,2%. The game offers up to 15 bet levels for the selected currency. A game round is started with the action button and plays the game with the selected bet level. Auto play plays the game automatically for the number of game rounds selected in the auto play page. All game payout and win combinations are paid out according to the pay table. All wins are presented and paid out in the selected currency. Only the highest win per pay line is paid out, if in sequence from far left to right in combinations according to the pay table. Two of the bonus games have pay lines both ways which could result in additional win combinations from far right to left. The wild symbol substitutes for all symbols except the bonus symbol. The bonus symbol is a scatter symbol that grants one of the bonus games if three (3) or more bonus symbols are won. The number of bonus symbols won correlates to the number of free spins granted in the bonus games according to the pay table. The game features three different types of bonus games; free spins with additional symbol rows, free spins with pay lines both ways and free spins with additional symbol rows and pay lines both ways. The bonus game is played with the same bet as the spin that triggered the bonus game. No bets can be altered during a game round. In the event of game malfunction all affected bets and pays are rendered void. Last modification date: 3/28/2017
