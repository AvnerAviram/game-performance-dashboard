Respin Mania Mega Reels PLAY MODE AND BALANCE You can play the game in either FUN or REAL mode. In fun mode, you are playing just for the fun of the score. In real mode, you are playing with money from your real money balance. Your balance is shown in the BALANCE box both in fun and real mode. PLACING BETS This game has 50, 60, 70, 80, 90 or 100 paylines, depending on the number of rows. The number of paylines starts with 50 and can increase to a maximum of 100 during the Mega Reels™ Respin feature. The BET is the COIN value multiplied by 50. To set the bet amount, use the '-' and '+' buttons. CONTINUOUS SPIN Clicking and holding the SPIN button for continuous spinning. To stop the continuous spin session, release the SPIN button. When CONTINUOUS SPIN is on, some win animations and sound are skipped, the reels spin faster and the game results are displayed faster. AUTOPLAY MODE To select the number of autospins, use the AUTOPLAY options panel. Clicking the OK button initiates the autoplay session. Clicking the CANCEL button closes the panel. You can end the autospins session earlier by clicking the AUTOPLAY button. The number of remaining spins is displayed on the AUTOPLAY button. TURBO MODE When TURBO Mode is on, some win animations and sounds are skipped and the reels spin faster. Clicking the TURBO button switches Turbo Mode on or off. PAYTABLE PAYOUTS Winning combinations start from the leftmost reel and the symbols have to be consecutive along a payline. Regular wins are multiplied by the COIN value. Only the highest win pays on each payline. Coinciding wins on different paylines are added. The maximum win in the game has an upper limit. For more information see the TERMS AND CONDITIONS section. FREE GAMES During FREE GAMES the reels are spun automatically using the same COIN value as per the spin that won the FREE GAMES. DISCONNECTION POLICY NOTE ON DISCONNECTION If you are disconnected from the Internet in the middle of the game, log in to the Casino again. Once you have reopened the game, you will be able to continue playing from the point where the game was disconnected. NOTE ON MALFUNCTIONS Malfunction voids all pays and plays.
