Ultra Stack Lion GRAPHIC STYLE/THEME ULTRA STACK LION is a video slot game themed with the wild animals from the forest. It contains stylized tiger skin. The overall colors of the game would be pertaining to various carnivores like snow leopard, Black Panther, tiger, lion etc. with native savage theme. All portion of the background should be filled with static tiger skin at all times. “ULTRA STACK LION” has a 4x5 matrix layout, with 50 win lines always active, with 3 Scatter symbols appearing on 2, 3 and 4 triggers the BIG CAT FEATURE. The BIG CAT FEATURE consists of MINI GAME and 8 FREE GAMES. SHORT GAME DESCRIPTION ULTRA STACK LION is a 5 reel video slot game themed with the wild animals from the forest and played at 50 lines. BIG CAT FEATURE triggers when 3 scattered SUN symbols appear on reels 2, 3 and 4. The feature consists of MINI GAMES and 8 FREE GAMES. During the MINI GAME, the player chooses 1 of 5 silhouettes to reveal the number of animal symbols during free games and during the FREE GAMES, additional 8 free games are added when 3 scattered SUN symbols appear on reels 2, 3 and 4.
