Magic of Pandora About Magic Of Pandora Magic Of Pandora is a 5x3 reel, twenty five fixed payline slot game. Return to Player (RTP) General Scatter symbol Wild symbol Mystery Feature Base Game Feature Free Spin Bonus Accept Awarded Feature Create Own Feature Autoplay Return to Player (RTP) Return to Player (RTP) is the theoretical statistical percentage of total money bet by players in a specific game, that is paid out as winnings over time. The RTP is fixed and does not change. This game has an RTP of 96.01%. General Game has fixed 25 lines. If you have a winning combination lined up on the payline, the win is highlighted on the reels. Regular wins are created when the same symbols appear next to each other on a payline, with one of the symbols displayed on the first slot reel. One regular win is paid per line. If you have more than one combination, you are paid the win that has the highest value. Regular wins depend on your bet per line except Scatter. Bet per line = (Coin size * Coins Bet). Regular wins in paytable are displayed in credits, so if your total bet is 0.25 credits per spin, your bet per line = total bet / number of paylines i.e. 0.25 credits / 25 paylines = 0.01 credits. Your total payout is calculated as { (bet per line * credits won on minimum bet) * 100 }. For example a 5 of a kind win of spade symbol which is 0.50 credits, the total payout amount will come as { (0.01 bet per line * 0.50 credits won on minimum bet) * 100 } = 0.50 credits. For providing more information to the player in calculating the payouts, this game has a dynamic paytable. The payouts displayed in the paytable for all the symbol combinations are the final amounts in credits which is already multiplied by active bet per line and paid to the player except Scatter (refer Scatter symbol section for details). You are paid out the win amount displayed in the Paid box. If you do not have a winning combination, the game is over and you can play again. Scatter symbol is a Scatter symbol. Scatter symbol can appear during base game only. 5, 4 or 3 Scatter symbols scattered anywhere awards 16, 12 or 8 Free Spins respectively. Scatter symbol cannot be substituted by the Wild symbol. Scatter wins are multiplied by total bet and added to payline wins. Base pay of the Scatter symbol for the particular combination can be calculated as (credits won on minimum bet / number of paylines) * 100. So total payouts for the Scatter win can be calculated as (base pay of the Scatter symbol combination * total bet). For providing more information to the player in calculating the payouts, this game has a dynamic paytable. Scatter payouts displayed in the paytable are the final amounts in credits which are already multiplied by the total bet and paid to the player. If you have a Scatter win and payline win, you are paid out for both wins, as the Scatter symbol does not need to appear on a payline to win. You spin the reels. And your Scatter win would be (1.00 base pay * 0.25 credits) = 0.25 credits. Wild symbol is a Wild symbol. Wild symbol can appear on reels 2, 3, 4 and 5. Wild substitutes for all symbols except Scatter symbol. Example: Game A has the following criteria: Wild substitutes for all symbols except Scatter to complete winning combinations. Winning combination pay out from left to right, so one symbol must be displayed on the first slot reel. A symbol combination starting from second, third, fourth or fifth reel does not pay out. If there is more than one possible winning combination on a payline, you are paid out the value of the highest combination only. Mystery Feature Each reel contains a stack of mystery positions which are replaced by a game symbol except wild and scatter during each reel spin. All mystery symbols are replaced by the same game symbol for a given spin. Base Game Feature is a Pandora’s Magic Box. On any random spin, Pandora’s Magic Box may appear on the reel area and player is awarded one of the below features randomly: Wild Reel Extra wild Additional paylines All the pays are evaluated after the base game feature. Wild Reel: While the reels are spinning, one or two snakes may come from the box and turn one or two of the reels 2, 3 or 4 wild. Extra wild: Wild symbols will come out of the box and land on random position on the reels converting them into wilds. Additional paylines: While the reels are spinning, a spider may come out of the box start a web of additional green paylines. Game become 50 paylines for that particular spin. Free Spin Bonus is a Scatter symbol. 5, 4 or 3 scatter symbols scattered anywhere awards 16, 12 or 8 free spins respectively. Free spin bonus offers two options out of which the player can choose any one: Accept Awarded Feature Create Own Feature During either free spins, free spin feature can be re-triggered on any random spin. Pandora’s magic box will appear and awards 4 extra free spins. All bets played are the same as those that started the Free Spin Bonus. Free Spins wins are added to payline and Scatter wins. Autoplay You can also spin a set number of times in succession without interacting with the game, using Autoplay. You must select a bet amount before you start Autoplay, as it plays with the last bet value. Tap/Click Autoplay On. Tap/Click the number of spins you want to play. The Autoplay session automatically starts playing. To stop Autoplay, Tap/Click Stop. Custom Mode Autoplay is also available with Custom Mode where some Autoplay Settings are available. Autoplay Settings 1. Coin Size You can select a coin size before you start Autoplay, as it plays with the last bet value. 2. Coins Bet Coins Bet is the number of coins allowed per line. Coins Bet differ between games. 3. Total Bet Total Bet displays the Total Bet value. Total Bet = (Coin Size * Coins Bet * Number of paylines). 4. Spins You can set the number of times the slot reels spin during your Autoplay session. 5. Win Limit Stop spins if the number of credits won on any spin exceeds or equals the amount selected. 6. Loss Limit Stop spins if Autoplay exceeds the loss limit value with the next bet. Until Stop Continue Autoplay until you Tap/Click the Stop button or stops if Free Spins or another bonus feature is triggered. Important: This feature may not be available in the game you are playing. Keyboard Shortcut Spin/Stop Button can be selected by pressing Space bar on Keyboard.
