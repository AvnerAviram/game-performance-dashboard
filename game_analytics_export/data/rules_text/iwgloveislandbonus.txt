Love Island Bonus Introduction Reveal the 25 symbols. Find 3, 4 or 5 like symbols consecutively in a vertical, horizontal or diagonal line to win a prize! Find a pink diamond to unlock the DIAMOND BONUS game. Find 3 lucky Wheel symbols anywhere to unlock the WHEEL BONUS game. Start the game Select the stake amount, then to start the game press the PLAY button. Once the PLAY button has been selected the stake is deducted from the current balance and the game starts. End of the game At the end of the game, any winnings are displayed. To play again press the PLAY button. Once the PLAY button has been selected the stake is deducted from the current balance and the game starts. How to play At the start of the game there are heart symbols. Select each heart to reveal a symbol. Or press the REVEAL ALL button to reveal all symbols. If a pink diamond is revealed, unlock the DIAMOND BONUS game. Find 3 Wheel symbols anywhere to unlock the WHEEL BONUS game. DIAMOND BONUS GAME Select Diamonds to reveal a prize amount. Keep going until you find the COLLECT symbol. At this point all prizes revealed during the game are awarded. WHEEL BONUS GAME Spin the wheel. Win the prize the wheel stops on. Winning Wins are achieved by: Finding 3 or more like symbols consecutively in a vertical, horizontal or diagonal line. Win the corresponding prize on the prize table. DIAMOND BONUS game - win all prizes revealed during the game. WHEEL BONUS GAME - win the prize the wheel stops on. General information System malfunction voids all pays and all plays. This game is a game of chance. The outcome of a play in the game is pre-determined. Actions or choices made by player do not affect the outcome of the game. Bonus symbols collected within one game do not impact the winning odds of any subsequent games. The theoretical average return to player (RTP) is 90.96%.
