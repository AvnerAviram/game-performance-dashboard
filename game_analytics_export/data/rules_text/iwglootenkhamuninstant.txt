Looten' Khamun Instant Introduction There are 3 games to play: Game 1: Win a prize by revealing the Game 2: To win, match 3 prize amounts. Game 3: Match 2 symbols in a ROW and win the PRIZE shown. Start The Game To start the game, press the 'Play' button located below the stake amount. Once the 'Play' button has been selected the stake is deducted from the current balance and the game starts. End of the game At the end of the game the outcome is displayed on a message panel. You can play the game again at the same stake amount as the previous game by pressing the 'Play' button located on the message panel. Once the 'Play' button has been selected the stake is deducted from the current balance. Pressing the 'Set Bet' button will cause the 'Choose Bet' interface to appear where you can alter the stake. General Information System malfunction voids all pays and all plays. This game is a game of chance. The outcome of a play in the game is pre-determined.
