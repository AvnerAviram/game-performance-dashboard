50 Fortune Fruits Game Theme and Graphics 50 Fortune Fruits is a 5x4 50 lines slot game packed with wild jokers. Main Game 50 Fortune Fruits is played on 50 lines (fixed). The Jokers can be found in stacks on the reels. Jokers substitute for all symbols except stars (scatters). Feature 3, 4 or 5 scattered star symbols pay 10, 20 or 30 Free Games. During Free Games there are many more Jokers on the reels. Top Prize: 26.920 times bet per line per Free Game. Feature hit rate: ~300 Gamble Wins can be gambled in the red/black card gamble. Rules Jokers substitute for all symbols except stars (scatters). All prizes are for combinations of a kind. All prizes are on selected lines excluding scatters. Scatter wins are added to line wins. All prizes are for combinations left to right, except scatters. Highest win only paid on each line and per scatter combination. Malfunction voids all pays and plays. Return To Player The theoretical return to player of this game is 95.45%
