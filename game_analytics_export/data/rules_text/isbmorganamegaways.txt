Morgana Megaways 1. Overview The goal is to obtain a winning combination on a winning way spread across the reels. Game specifications : Type Slots Number of reels 6 Number of rows 2-7 Number of ways Up to 117,649 Megaways Number of coins you can bet 20 (cannot be changed) Progressive No Double feature No Auto spin feature Yes Wild symbols Yes Multiplier symbols No Scatter symbols Yes Bonus round No Free s pins Yes Instant win No RTP Standard play - 95.98% Morgana bet - 98.06% The game disconnects if you do not take any actions for a specific number of consecutive minutes (this number varies depending on jurisdiction); the session recovery feature will save the game progress so that the next time the game is loaded, you can continue where you left off. In the event of technical difficulties resulting in a dropped connection to the game server, the game you were playing is saved automatically and will load the next time you establish a connection to the game server. You will not be able to reconnect to the game server for the first 30 seconds after the disconnection. Note : In case of malfunction all pays and play are voided; any bet that was not yet accepted will be voided, any bet that was unsettled will be refunded. 2. How to Play Note: If you do not have enough money to play a game, the bet can be set to the minimum selectable amount through a dialog window informing about this. All winning ways pay from left to right. Winnings on different ways are all added. Only the highest winning combination on each way is paid. Winnings from Free Spins are added to the total win. The auto spin feature does not automatically start free spins - player input is required. Each spin can have up to 117,649 ways to win. Wins are awarded for left to right adjacent symbols, starting on reel 1, regardless of size. 3.2. Max Megaways During any paid spin the Max Megaways modifier can be triggered. This will happen randomly with a unique background animation and the modifier title appearing, causing the reels to land the maximum amount of Megaways available (117,649). Available on any paid spin Max Megaways can be triggered and will force the reels to a 117,649 Megaways outcome. 3.3. Wild The Wild symbol can replace any other symbols, except Scatter symbols, to form winning ways. Wild symbols can only be found on reels 2, 3, 4 and 5. 3.4. Scatter symbol Landing 3 or more scatter symbols will trigger Free Spins. Scatter symbols are not available in Free Spins. 3.5. Mystery symbol Mystery symbol transforms into any symbol except Scatter or Wild symbols. Mystery symbols transform into the same symbol. Mystery symbols are also available in Free Spins. 3.6. Free Spins Landing 3 or more Scatter symbols will trigger Free Spins. During Free Spins collect symbols to award multiplier prize, additional Free Spins and boost the minimum Megaways. Collecting all the required symbols will change the level to the next highest symbol. Any required symbol which appears in view during Free Spins is collected. Required symbols do not have to form winning combinations. Auto spin feature The loss limit setting allows stopping the auto spin feature automatically when the selected limit, plus your cumulative winnings, and minus your cumulative bets, is less than 0. 4.1. Desktop version Click the AUTO SPIN button to open the AUTO SPIN settings window where you can set up the following: Number of auto spins: click + or - to select the number of auto spins to play. Click the ACTIVATE button to close the AUTO SPIN settings window and to start the auto spin feature with the currently selected bet and auto spin setup. Click the CLOSE button to close the window without starting the auto spin feature. Click the STOP AUTO SPIN button to stop the auto spin feature at any time. 4.2.
