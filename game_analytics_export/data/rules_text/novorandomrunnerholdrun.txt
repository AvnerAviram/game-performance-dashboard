Volatility Variations Variations (%) 95 Return (%) 94.97% GAME DESIGN Main Game The main game is played on 5 reels and 15 fixed win lines. Any 3 of the same adjacent symbols on one of the 15 win lines will pay except for the Coin symbol. Star symbols are overlaid on other symbols and both symbols count. Feature 6 or more Coin symbols will start the Hold & Run feature which is a Hold & Spin feature. The goal is to collect as many coins as possible. Each coin will carry a prize value or bonus label. At the end of the feature all collected prizes and bonuses will be paid. Collected all 15 coins? Then the Grand Bonus will be paid on top! Rules Winning combinations and payouts are made according to the paytable. All prizes are for 3-of-a-kind combinations, adjacent on one of the win lines, except for the Coin symbols. Only the highest win per line in view is paid. 6 or more Coin symbols in view will start the Hold & Spin feature. Malfunction voids all pays and plays. PAYTABLE The values in the pay table must be multiplied by the number of coins bet.
