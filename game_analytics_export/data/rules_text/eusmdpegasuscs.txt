Pegasus Cash Spree About this Game This is a 5 reel 3 row video slot game. The objective of the game is to spin the reels so the symbols land in different combinations. Winning combinations are highlighted on the reels and a payout is awarded. This game includes 20 fixed paylines. This game features Cash Spree™ which can be awarded in the base game and during the Free Spins feature. Payout Rules If multiple winning combinations land on a single payline, only the highest win is awarded. Only the highest winning combination is awarded per symbol combination. The payout value is based on the winning combination formed. The total payout is the sum of all payline wins, Cash Spree™ symbols, Wheel Bonus and Free Spin symbol wins resulting from a spin. The Paytable displays the payout for each symbol combination based on the current bet. Payline Rules A payline is a pattern of adjacent symbol positions that runs in a line across the reels. Payline win payouts are based on the initial bet placed. A win is awarded when matching symbols land adjacent to each other on a payline. Payline wins run from left to right, starting from the first slot reel. This game includes 20 fixed paylines. Bet Rules Each spin played costs a total bet amount. The initial bet is the current total bet divided by the payline cost of 20. The total bet is based on the initial bet placed, multiplied by the cost of the paylines played. Wild Symbols Wild symbols can substitute for other symbols to complete winning combinations. Wild symbols can create their own winning combination when landing in a payline combination. Wild symbols can substitute for all symbols except Multiplier, Wheel Trigger, Cash Spree™ and Free Spin symbols. Mystery Selection There is a designated section on the reels that selects a symbol each spin. The Mystery Selection will change to any other symbol, except Multiplier, Wheel Trigger, and Free Spins symbols. The Mystery Selection will display as a Mystery symbol when it will reveal a Wild or Cash Spree™ symbol. Cash Spree™ Symbols Land Cash Spree™ symbols, starting on reels 1 and 2, to trigger the feature. Cash Spree™ symbols award Grand, Mega, Major, Minor, Mini, or cash values. Grand, Mega, Major, Minor, or Mini only appear on reels 4 and 5. Cash values appear on all reels. Higher cash values appear on reels 4 and 5. All Cash Spree™ symbols pay left to right on adjacent reels. Cash Spree™ symbol wins are added to payline wins. Wheel Bonus Collect Wheel Trigger symbol for a chance to trigger the Wheel Bonus. The Wheel Bonus awards 1 of 5 prizes. Prizes are Grand, Mega, Major, Minor and Mini. Prizes are divided into wedges. When the wheel stops, the wedge that the pointer points to is awarded. The graphical collect of the Wheel Trigger Symbol is for visual entertainment and does not influence awarding the Wheel Bonus. Wheel Trigger symbol cannot appear with the Mystery, Free Spins and Cash Spree™ symbols. This feature is available in base game only. This feature cannot trigger on a Free Spins trigger. Free Spins Multiplier The Free Spins Multiplier is displayed in the Free Spins Multiplier meter. Each Multiplier symbol that appears, the Free Spins Multiplier will increase by 0.5X. The Multiplier symbol cannot appear on Free Spins triggering spin. Multiplier symbol cannot appear on Free Spins or Mystery symbols. The maximum Free Spins Multiplier is 10X. The Free Spins Multiplier will reset between 1X - 4X after Free Spins are complete. Free Spins 3 Free Spin symbols scattered anywhere on the reels triggers Free Spins Wheel. Each wedge of the Free Spins Wheel contains a number of free spins. The multiplier applies to all wins in the Free Spins except Free Spins Wheel, Grand, and Mega. When the wheel stops, the wedge that the pointer points to is awarded. Select the Spin button to start the Free Spins. 3 Free Spin symbols scattered anywhere on the reels retriggers Free Spins. The number of free spins awarded on a retrigger is equal to the initial trigger. Free Spin symbols appear on reels 2, 3, and 4. Wins awarded during this feature are added to base game wins. All bets played are the same as the spin that triggered this feature. All lines played are the same as the spin that triggered this feature. Game Controls Credits: Displays the current balances. Free Games A Free Games offer is a number of free game spins awarded to a player. Each offer has a unique reference number. The bet configuration is set by the game managers and cannot be changed. Wins are added to the cash or bonus balance, depending on the offer. An offer can be played now, played later, or discarded. Select Play Now to begin playing a Free Games offer. Only one Free Game offer can be played at a time. Select Play Later to postpone the offer. A postponed offer will be available again when re-entering the game. Select the Bin icon to discard the Free Games. A discarded offer can only be re-opened by contacting the game managers and providing the reference number. When an offer is complete, discarded, or expired the next available offer is displayed. Each offer has an expiry date and time; if an offer is not played before this time, it will no longer be available. Select the Information icon to view bet information and the offer expiry date and time. Additional Information Some settings and features may not be available in this game. Malfunction voids all pays and plays. Any changes to game rules will be conducted in accordance with regulatory requirements. In the event of a disconnection, the final game state is displayed on return to the game. The balance is updated with the amount won or lost. The results of the last game played are displayed. If a request does not reach the server before disconnection, the results of the previous game played are displayed. Any features in progress that require interaction or selection can be continued.
