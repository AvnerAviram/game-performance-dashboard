Jester Wheel About this Game This is a 5 reel 3 row video slot game. The objective of the game is to spin the reels so the symbols land in different combinations. Winning combinations are highlighted on the reels and a payout is awarded. Payline Rules This game includes 25 paylines. The number of paylines played is fixed. A win is awarded when at least 3 matching symbols land adjacent to each other on a payline. A payline is a pattern of adjacent symbol positions that runs in a line across the reels. Payline wins run from left to right, starting from the first slot reel. Payline wins also run from right to left during the JESTER MIRROR feature. Wild Symbols Wild symbols can substitute for other symbols to complete winning combinations. Wild symbols cannot substitute for other symbols to trigger the JESTER WHEEL feature. Wild symbols can substitute for all symbols except Scatters. JESTER WHEEL Feature This feature can trigger in base game only. 2 Scatter symbols anywhere on the reels trigger this feature. 1 Bonus Spin is awarded. 1 of 4 bonus features (JESTER BURST, JESTER REEL, JESTER MIRROR, JESTER TRAIL) is selected at random and played out immediately. Wild symbols cannot substitute for other symbols to trigger this feature. JESTER BURST Feature This feature can be triggered as part of the JESTER WHEEL feature by landing 2 Scatter symbols. This feature can be triggered as part of the JESTER FREE SPINS feature by landing 3 Scatter symbols. 2 - 6 Wild symbols are scattered randomly on the reels. JESTER REEL Feature This feature can be triggered as part of the JESTER WHEEL feature by landing 2 Scatter symbols. This feature can be triggered as part of the JESTER FREE SPINS feature by landing 3 Scatter symbols. 1 - 3 reels are randomly filled with Wild symbols. JESTER TRAIL Feature This feature can be triggered as part of the JESTER WHEEL feature by landing 2 Scatter symbols. This feature can be triggered as part of the JESTER FREE SPINS feature by landing 3 Scatter symbols. 1 Wild symbol is randomly placed on the reels and generates up to 4 more Wild symbols along a random path. The path consist only of horizontally or vertically adjacent positions. JESTER MIRROR Feature This feature can be triggered as part of the JESTER WHEEL feature by landing 2 Scatter symbols. This feature can be triggered as part of the JESTER FREE SPINS feature by landing 3 Scatter symbols. Reel 2 is filled with Wild symbols. Payline wins pay from both sides for the current spin. Scatter Symbols This symbol appears in base game and JESTER FREE SPINS. In base game this symbol only appears on reel 3 and 5. In JESTER FREE SPINS this symbol only appears on reel 5. 2 Scatter symbols anywhere on the reels trigger the JESTER WHEEL feature. 3 Scatter symbols anywhere on the reels award 8 JESTER FREE SPINS. 1 Scatter symbol anywhere on the reels in JESTER FREE SPINS triggers the MULTIPLIER WHEEL feature. This symbol cannot be substituted by Wild symbols. This symbol does not have a payout. JESTER FREE SPINS 3 Scatter symbols anywhere on the reels trigger this feature. 8 JESTER FREE SPINS are awarded. JESTER FREE SPINS are played with 1 of 4 randomly selected bonus features (JESTER BURST, JESTER REEL, JESTER MIRROR, JESTER TRAIL). The randomly selected bonus feature is guaranteed to trigger on every JESTER FREE SPIN. JESTER FREE SPINS are played with the MULTIPLIER WHEEL feature. 1 Scatter symbol anywhere on the reels triggers the MULTIPLIER WHEEL feature. All bets played are the same as the spin that triggered this feature. All lines played are the same as the spin that triggered this feature. JESTER FREE SPINS play out automatically. JESTER FREE SPINS cannot be retriggered. MULTIPLIER WHEEL Feature This feature can trigger in JESTER FREE SPINS only. 1 Scatter symbol anywhere on the reels triggers this feature. 1 of 6 multipliers (x2, x3, x4, x5, x6, x10) is selected at random and applied to the winnings of the current spin. Game Controls Some settings and features may not be available in this game. Balance: Displays the current balances. Credits: Displays the current balances. Total Bet: Displays the total bet amount per spin. Lines: Displays the amount of played lines per spin. Win: Displays the most recent win. Spin button: Starts a reel spin. Stop button: Stops the current reel spin. This does not influence the results of the spin. Play button: Continues the game. Autoplay start button: Starts Autoplay session. Autoplay stop button: Ends the current Autoplay session. Game Menu button: Displays the game settings and game payout table. Bet Decrease button: Decreases the current bet down to the next lowest available bet. Bet Increase button: Increases the current bet up to the next highest available bet. Fast Play Active Indicator: Indicates when Fast Play is active. Autoplay button: Displays Autoplay options. Bet Rules Each spin played costs a total bet amount. The initial bet is the current total bet pided by the number of paylines played. Bet Settings Some settings and features may not be available in this game. Max Bet button: Sets the current bet to the maximum value. Bet Amount: Sets the current bet value. Quick Bet buttons: Sets the current bet to the selected value. Payout Rules Payline win payouts are based on the initial bet placed. If multiple winning combinations land on a single payline, only the highest win is awarded. The Paytable displays the payout for each symbol combination based on the current bet. The win up to value displayed is rounded down, and is approximately the highest possible win achievable on a single spin. Only the highest winning combination is awarded per symbol combination. Return to Player The Return to Player (RTP) is the theoretical statistical percentage of the total money bet in a game that is paid out as winnings over time. This game has an RTP of 96.49%. Additional Information Some settings and features may not be available in this game. The balance is updated with the amount won or lost. The results of the last game played are displayed. Any features in progress that require interaction or selection can be continued. In the event of a disconnection, the final game state is displayed on return to the game. Malfunction voids all pays and plays.
