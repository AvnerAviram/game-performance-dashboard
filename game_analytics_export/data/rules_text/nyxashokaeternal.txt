Ashoka Eternal Game description Ashoka Eternal is a cluster game with 5 columns by 7 rows with dropping symbols. The grid can expand up to 7 columns by 9 rows. The game comes with a multiplier wheel, multipliers, redrop feature, big symbols, blocker symbols and charged Wild symbols. Wining symbols adjacent to a blocker removes the blocker. Clear away all blockers and the grid will expand in 3 levels. First level 5x7 symbol grid, next level 7x7 symbol grid and last level 7x9 symbol grid. Multiplier wheel has 12 positions. Each win turns the wheel one position. When a multiplier activation symbol is at the top of the wheel, this activates the global multiplier for the next symbol drop. The wheel locks on the position when no more cluster wins appear. Clearing all blockers in level 3 will make the wheel stop and activate the global multiplier for the rest of the game round. The bonus game is triggered by 3 active bonus symbols to run 8 free drops. one new active bonus symbol in the bonus game adds 2 additional free drops. The super bonus game has double the chance to activate the global multiplier. Game view Bet amount and balance are always displayed in the info bar at the bottom of the game view in the currency that the player has chosen. This information is always available throughout the gaming session. Winnings are displayed in the selected currency in the middle of the info bar at the bottom of the game view when playing in landscape mode, and at the top of the game view when playing in portrait mode. Paylines Ashoka Eternal is a cluster game where a cluster of 5 or more identical symbols which are either vertically or horizontally connected triggers a win. The charged Wild symbol is a Wild symbol charged to last for up to 3 wins. It is removed after the last winning combination. The grid can expand up to 7 columns by 9 rows. The minimum bet is $ 0.20, the maximum bet is $ 100.00. Each bet is 100 coins. Wins are paid for a cluster of 5 or more identical symbols connected vertically or horizontally. A game round starts on the selected bet level when the player hits the Spin button. The game can be played automatically by activating Auto Spin. Hold down the Spin button or use the Auto Spin button to set the number of game rounds to be played automatically. Winnings and winning combinations are paid out according to the Paytable. Winnings are presented in the player's currency or in coins. The player can change preference in Settings. Winnings are presented in the player's currency. The game is played in two modes; normal mode and bonus mode with free drops. The Wild symbols substitute for any paying symbol. A winning combination of only Wild symbols pays as the highest paying symbol. A game round starts with a symbol drop with blockers above and below symbols for each column. Winning symbols and all non winning symbols of the same type as the winning symbols is removed from the grid. Any blocker symbol adjacent to a winning symbol are removed, increasing the numbers of symbols in play. The grid fills up with new symbols. Any new winning combinations are paid and the Avalanche repeats until no more wins. Removing all blocker symbols in play increases the grid to 7x7 symbols with a new set of blockers on column 1 and 7. Removing all blockers on column 1 and 7 increases the grid to 7x9 and a new set of blockers are added to the top and bottom row. Multiplier blocker symbols adds its value to the global multiplier when adjacent to a winning symbol. The multiplier wheel has 12 positions. Activation symbols are distributed on the wheel. To activate the global multiplier the wheel needs to lock into a position where a multiplier activation symbol is in the top position. The wheel turns one step to the left after each win. If new winning combinations appear the wheel continues to rotate to the left. This repeats until no more wins. Clearing all blockers in level 3 will make the wheel stop and activate the global multiplier for the rest of the game round. Big Wild symbols pay as the amount of regular 1x1 Wild symbols they cover. I.e. a 2x2 Wild symbol is 4 regular Wild symbols and a 3x3 Wild symbol is 9 regular Wild symbols. The charged Wild symbol is a Wild symbol charged to last up to 3 wins. It is removed after the last win. If a big symbol drop ends with empty cells below it, it will replicate and fill the empty cells with regular 1x1 sized symbols of the same type as the big symbol. Redrop symbol selects one payout symbol type and keep these symbols sticky during one redrop. The redrop feature activates when no more winning combination appears in view. 3 active bonus symbols triggers the 8 free drops bonus game. Inactive bonus symbol will become active when adjacent to a winning symbol. Charged Wild symbols remain when entering the bonus game and between free drops. Bonus symbols are persistent between free drops. The bonus game starts with a new set of symbols in the grid. To trigger the super bonus game one of the bonus symbols needs to be a super bonus symbol. In the super bonus game the multiplier wheel has double the chance to activate the global multiplier. One new active bonus symbol in the bonus game adds 2 additional free drops. Winnings are paid out at the end of a game round. The maximum total win in a game round is capped at 25 000x the placed bet. A game round is immediately ended when the win cap is reached, and the wins collected so far will be paid out. No bets can be changed during a game round. The game will be capped at 25 000x the placed bet. The theoretical payout (RTP) is 94.0%. In the event of malfunction all bets and wins are void.
