Dragon's Blessings RTP: 96.0% VOLATILITY: 9.5- VERY LOW GAME RULES This game is played at 576 ways. Ways wins contain one symbol from each adjacent reel, beginning with the leftmost reel. The same symbol, or its substitute, in a different position on the same reel pays that ways win again. Ways wins are multiplied by the bet multiplier. Scatter symbols award in any position on the reels. Wins are shown in credits unless marked as currency. Malfunction voids all pays and play. BASE GAME Wild Symbols are wild and substitutes for all symbols except Free Games Bonus Symbol. Wild Symbols appear only on reels 2, 3, and 4. FREE GAMES 5 or more Free Games Bonus Symbols appearing in any position on reels 1, 2, 3, 4, and 5 starts the Free Game Bonus. Free Games Bonus Symbols awards 6 free games per winning combination. Free Games Bonus Symbol is a scatter symbol.[FGWD1] is wild and substitutes for all symbols except [FGBO1]. Free Games Wild Symbols appear only on reels 2, 3, and 4. The Free Game Bonus uses different reels. Free games automatically play on the same ways at the same bet multiplier as the game that initiated the Free Game Bonus. 5 or more Free Games Bonus Symbols appearing in any position on reels 1, 2, 3, 4, and 5 awards 6 additional free games per winning combination. Free Games Bonus Symbols is a scatter symbol. The Free Game Bonus ends when 0 free games remain or when 300 free games have been played. Last modification date: 1/24/2023
