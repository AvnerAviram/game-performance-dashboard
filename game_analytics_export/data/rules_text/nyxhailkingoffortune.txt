Hail King of Fortune RTP 96.5% GAME RULES Malfunction voids all pays and play Select Bet Multiplier Click the Spin button to spin the reel. This game is played at 1024 ways. Ways wins contain one symbol from each adjacent reel, beginning with the leftmost reel. The same symbol, or its substitute, in a different position on the same reel pays that ways win again. Only the highest win is paid for each ways symbol combination. Ways wins are multiplied by the bet multiplier. Scatter symbols award in any position on the reels. Wild Symbol is wild and substitutes for all symbols except the Free Games Bonus symbol. Wild Symbol appears only on reels 2, 3, and 4. FREE GAMES BONUS 3 Free Games Bonus symbols in any position on reels 2, 3, and 4 starts the Bonus Select. Awards 1 Pick. Free Games Bonus symbol is a scatter symbol. Free Games Bonus symbol only appears on reels 2, 3, and 4. Bonus Select features 3 options 1 – 3 Free Games Awarded at 11x, 18x, and 28x multipliers 2 – 8 Free Games Awarded at 5x, 8x, and 11x 3 – 18 Free Games Awarded at 2x, 3x, and 5x Bonus Select ends when 0 picks remain. The Free Games Bonus begins immediately afterwards. The Free Games Bonus is triggered during the Bonus Select. Free Games Wild 1, 2, 3, 4, 5, 6, and 7 symbols are wild and substitute for all other symbols Free Games Wild 1, 2, 3, 4, 5, 6, and 7 symbols appears only on reels 2, 3, and 4. During the Free Games Bonus Free Games Wild 1, 2, 3, 4, 5, 6, or 7 symbols appearing in any position on the reels awards an x2, x3, x5, x8, x11, x18, or x28 multiplier value, respectively. If more than one Free Games Wild 1, 2, 3, 4, 5, 6, or 7 appears, the sum of the multiplier values I awarded. The Free Games Bonus uses different reels. Free games automatically play on the same ways at the same bet multiplier as the game that initiated the Free Games Bonus. The Free Games Bonus does not award additional free games.
