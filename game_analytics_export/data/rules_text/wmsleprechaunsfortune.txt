Leprechauns Fortune Base Game WILD FEATURE WILD substitutes any other symbols except FEATURE. Line pays must occur on adjacent reels, beginning with the leftmost reel. Winning scatter pay combinations pay in any position on any reel. Only highest winner paid per winning combination. Paytable reflects current bet configuration. Feature Description Three scattered FEATURE symbols in any position award 10 free spins. Bets staked and lines played are the same as the game that starts the feature. WILD appearing during free spins stays in the window and substitutes any other symbol except FEATURE. FEATURE can be triggered again during the feature once, with 5 free spins awarded. FEATURE only appears on reels 1-3. WILD only appears on reels 2-5 Scatter pays are added to payline wins. PAYLINES Wins on multiple paylines are added together. Total line bet must be divided equally between each active payline; therefore, all total bet amounts are not available. The line bet amount is added for every 2 lines. Only active paylines can register wins. Malfunction voids all pays and plays. Last modification date: 1/31/2017
