Trinity Pots Rising Wilds PAYTABLE Only the highest winner paid per winning combination. REEL WAYS™ pays must occur on adjacent reels, beginning with the far-left reel. All wins on multiple ways are added together. The base game contains up to 243 ways to win. Winning pays per symbol. All pays in credits. All REEL WAYS™ pays are multiplied by the bet multiplier. WILD SYMBOL The WILD symbol is wild and substitutes for all symbols except for the element coin symbols.The symbol may appear on base game reels 3, 4 and 5. In the base game wilds are persistent and move up 1 reel position for subsequent spins. If a WILD symbol appears over an element coin at the end of a spin, that position acts as both a WILD and an element coin. In Free Spins Feature, the WILD symbol appears on reels 2, 3, 4 and 5, but is not persistent. ELEMENT COINS Each ELEMENT COIN that lands during the base game will fly into the corresponding colored pot. Any ELEMENT COIN may trigger the corresponding pot. More than one pot can be triggered on a single spin. If pot(s) are triggered the corresponding feature will start. The accumulation of ELEMENT COINS over time in the RED, GREEN and BLUE pots have no effect on triggering the JACKPOT FEATURE or FREE SPINS FEATURE. JACKPOT FEATURE The JACKPOT FEATURE is a scripted player pick feature. The choices the player makes are for entertainment purposes only. Any RED ELEMENT COIN landing in the RED pot may trigger the JACKPOT FEATURE. 15 coins will be presented, and Gems Symbols are revealed by clicking on them. Coins will reveal Gems Symbols or the UPGRADE symbol. Once 3 matching Gems Symbols have been revealed, the corresponding JACKPOT is awarded. If 3 UPGRADE symbols are revealed before a JACKPOT is awarded, the JACKPOT won is upgraded to the next level. The feature ends when one JACKPOT is awarded to the player. FREE SPINS Only red coins are available during the Free Spins Feature. During the Free Spins Feature, if red coin lands on the screen, it will award either 1-2 additional spins, a credit prize, MINI or MINOR. Please see internal help pages for details of credit prize. Additionally, 1 or more red coin that lands on the screen, may also award the Jackpot Feature. Winning combinations for these reels are identical to Base Game except blue Coins and green Coins do not appear. The Free Spins Feature contains up to 243 ways to win. Only the highest winner paid per winning combination. REEL WAYS™ pays must occur on adjacent reels, beginning with the far-left reel. All wins on multiple ways are added together. All REEL WAYS™ pays are multiplied by the bet multiplier. An alternate set of reels is used during the Free Spins Feature. REEL WAYS™ Winning combinations may contain only one symbol from each adjacent reel, beginning with the leftmost reel. Like symbols, or their substitute, on a single reel are not part of the same winning combination but may create separate winning combinations. TURBO MODE The Turbo Mode button is displayed on the bottom right of the reels with an "ON" and "OFF" option. Turn ON Turbo Mode button to speed up each spin, it can be turned OFF at any point during the base game. Activating Turbo Mode does not influence the results of the spins.
