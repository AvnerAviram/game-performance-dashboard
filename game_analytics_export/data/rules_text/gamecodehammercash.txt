Hammer Cash Game Rules Hammer Cash™ is a 5-reel, 3-row video slot featuring Free Spins Feature, Hammer Feature and the Hammer Booster Feature. Base Game RTP is 96.19%. Buy Free Spins RTP is 96.13%. All RTPs represent the long term theoretical payback. Minimum bet: $0.25. Maximum bet: $100.00. The game is played with 243 Ways except for the Free Spins Feature and Hammer Feature. Number of Ways in play in Free Spins Feature depends on the feature level. Each winning combination is made up of only one symbol from each reel. All wins pay left to right, beginning with the leftmost reel. Each symbol can be used only once per winning combination. Highest win only for each winning combination. Wins on different combinations are added. Free Spins Feature and Hammer Feature wins are added to base game wins. All wins shown in currency. Jackpot Prizes are proportional to the selected bet and displayed as such. If both 'of a kind' and 'any combination of' winning rules would apply, only the higher paying combination is paid. Game outcome is random and cannot be influenced by actions of the player. Malfunction voids all pays and plays. Hammer Feature Hammer Feature is initiated with a Hammer Symbol landing on both the leftmost and rightmost reel in base game. Hammer Feature awards a random Jackpot prize instantly. Possible awards are 2, 3, 5, 10, 15, 20, 25, 30, 35, 40, 50, 75, 100, and 500 times base bet and the Grand Jackpot, which is 2000 times base bet. Free Spins Feature Free Spins Feature is initiated by 3 scattered Free Spins symbols landing on reels 2, 3 and 4 counting from the leftmost reel. Free Spins Feature includes: Alternate sets of reels Expanding reels, up to maximum of 10 rows. Collectible Blitz symbols 15 possible Jackpot awards, including the Grand Jackpot. The feature is initiated on the Jackpot level achieved for bet level played in base game, as indicated on the Jackpot tower. The player is awarded 3 Free Spins on the start of the feature. Free Spins features both regular ways wins and collectible Blitz symbols. Payouts remain the same as for the base game. Blitz counter is set to 0 at the beginning of the feature. Collecting enough Blitz symbols immediately upgrades the current Jackpot to next level and resets the number of remaining spins to 3. Blitz symbols exceeding the number required for upgrade are passed over to the next level. The size of the reels and number of Ways in play increase along with the current Jackpot level. Reaching the top Jackpot level awards the Grand Jackpot immediately and ends the feature. If no Free Spins remain, the reached Jackpot prize is awarded along with all the Ways wins collected during the feature. Jackpot level is reset to 2x bet for the current bet level on ending the Free Spins Feature. Jackpot level is independent for each available bet level and persisted on exiting the game in real play mode. Possible Jackpot Prizes are 2, 3, 5, 10, 15, 20, 25, 30, 35, 40, 50, 75, 100, 500 times base bet and the Grand Jackpot - 2000 times base bet. Hammer Booster Feature Any base spin resulting in 1 or 2 Free Spins symbols landing has a chance of upgrading the current Jackpot to the next level. 2 Free Spins symbols landing have a higher chance of upgrading the Jackpot level. Upgrade is performed for the current bet level only. Mixed-Ways™ Hammer Cash™ is utilizing Mixed-Ways™ mechanics, allowing both 'of a kind' wins and 'any combination of' wins at the same time. All winning combinations which consist of at least 2 different 'any combination of' symbol types are added to regular 'of a kind' winnings. Combinations of the symbols of the same type are not treated as 'any combination of' wins. Game Operation Spin Button: click to initiate a game round at the currently selected bet. Clicking the button during spins, feature summaries and multiplier animations fast-forwards them. Spacebar can be used in the desktop version of the game to perform spins. Bet Button: click to select the base bet. Press ‘+’ to increase the base bet. Press ‘-’ to decrease the base bet. Autoplay Button: Click to open the autoplay menu. Turbo Button: click to enable the Turbo Mode Spins will be performed more rapidly in the Turbo Mode. Turbo mode applies to base game, Free Games and the Gold Gold Gold™ feature. Click again to disable the Turbo Mode. Pays Button: click to open the Pays section. Settings Button: click to access Game Settings and the detailed Game Rules. Buy Free Games button: Allows triggering the Free Games Feature instantly for an additional fee. Buy Bonus button: Allows triggering the Gold Gold Gold™ Feature instantly for an additional fee. Buy Free Spins Feature Player can trigger a single spin on an alternate set of reels which guarantees triggering Free Spins. Feature can be started by clicking the Buy Free Spins button and confirming the choice. Triggered Free Spins are played on random starting Jackpot level. Current Jackpot level on the triggering bet does not affect the initial Jackpot level for Buy Free Spins. Cost of Buy Free Spins is 80 times bet. The cost is deducted from the available balance. Triggered Free Spins feature is played for the base bet that the Buy Bonus is initiated for. Jackpot levels for the triggered Free Spins feature are the same as for regular Free Spins Feature. Buy Free Spins is available only if there’s no game round currently in progress. Autoplay Settings Number of spins: the number of auto spins to perform. Autoplay can be stopped at any moment. Stop on any feature: If enabled, the autoplay will be stopped upon hitting a bonus feature, depending on the game played. Loss limit: The autoplay will be stopped if the total loss during autoplay exceeds the selected value. Loss is calculated as the difference between bets placed and the total win collected during the autoplay session. Stop on single win above: The autoplay will stop if a single game win exceeds the selected value during the autoplay. Available autoplay settings depend on the jurisdiction.
