Double Shamrock Double Shamrock Double Shamrock® is a 3x3 game with 5 lines that are always active. The game is centered around an Irish-themed aesthetic and features scaling jackpots as the bet per line increases. The "Double Shamrock" twists traditional gameplay by incorporating unique "Double" symbol multipliers. Jackpots are triggered when any three “Double” 7s appear on a payline. GAME FEATURES Jackpots Feature Triggered when any three “Double” 7s land on a payline. 3 Gold “Double” 7s award the highest jackpot 3 White “Double” 7s award the 2nd tier jackpot 3 Black “Double” 7s award the 3rd tier jackpot Any 3 mixed “Double” 7s award the 4th tier jackpot Jackpots scale up with the Bet Per Line Double Feature Each 7 symbol may have the word “Double” displayed over it. If three “Double” symbols appear on a payline, a jackpot is won. If two “Double” symbols appear on a winning payline, the win is multiplied by 4. If one “Double” symbol appears on a winning payline, the win is multiplied by 2.
