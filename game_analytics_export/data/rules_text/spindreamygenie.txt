WHEEL FEATURE GAME 3 appearing during the base game awards the Wheel Feature. 1 or 2 appearing during the base game may also randomly award the Wheel Feature. 1 or more appearing during the free games awards the Wheel Feature. Once triggered, 1 spin of the wheel is awarded. The prize or bonus is awarded by the wheel pointer. The available awards are credit prizes, win multipliers, added symbols, Free games, and Bonus game. Credit values displayed on the wheel have been pre-multiplied by the bet multiplier from the triggering game. If the win multiplier is awarded, all ways win for the current base game or free game are multiplied by the awarded multiplier value. If the wheel indicator shows added symbols, the number shown corresponds to the number of symbol positions that will transform into . Symbol positions on reel 1 and symbol positions already containing cannot transform. Ways wins are evaluated after all transformations are complete. FREE GAMES BONUS The Free Games Bonus is awarded when free games are won during the Wheel Feature. Additional free games can be awarded from the Wheel Feature during the free games. Free games are played on the same ways at the same bet as the triggering game. BONUS GAME Bonus game is awarded from the Wheel Feature. Pick a card to collect a GRAND, MAJOR, MINOR or MINI. Continue selecting cards until 3 matching icons have been collected Win the prize corresponding to the 3 matching icons Only one prize can be awarded per Bonus game. The Bonus game ends after a prize has been awarded. BUY A BONUS FEATURE (WHERE AVAILABLE) The Free Games Feature can be instantly triggered from the base game by buying it for 80x current total bet. When buying a Bonus, Wheel Feature will be awarded to trigger the Free Games Feature awarding one of the randomly chosen number of Free Games: 10, 15, 20, 25, 50 or 100 Free Games. Purchased Free Games follow Free Games Feature Rules. GAME RULES Ways wins contain one symbol from each adjacent reel, beginning with the leftmost reel. The same symbol, or its substitute, in a different position on the same reel pays the Ways win again. Only highest win is paid for each way symbol combination. Wins on different played ways are added. When winning on multiple pay ways, all wins are added to the total win. Malfunction voids all pays and plays. QUICK START Game starts by setting a selected Bet amount and by pressing the Spin button. Round ends when reels stop and there are no wins, or after the win presentation if there are wins. GAME MENU Game menu contains options and settings that control some of the game features: History - Displays previous rounds from the current session Close Game - Closes the game client Sound - Controls the Sound FXs in the game (on/off) Music - Controls the Music in the game (on/off) Left Hand Mode (where available) - Controls the position of the UI on Mobile devices Vibrate (where available) - Controls the vibration on the Mobile devices (on/off) Game is paused while the menu is visible and can be closed by pressing the Close button or anywhere on the screen. BET SELECTION Game starts with a default Bet amount set for the game. Player can change the Bet amount by either: Using Plus and Minus buttons next to the Bet Amount Pressing the Bet amount where Bet Amount selector will open up Bet amount cannot be changed during a game round and if there is a win presentation happening, changing bet Amount will reset the presentation. AUTOPLAY MODE (WHERE AVAILABLE) Autoplay mode enables automatic play of game. It can be enabled or disabled at any time during the Base game. Autoplay mode is enabled by pressing the Autoplay button which allows the player to select: Number of rounds to be played automatically Loss Limit (where available) - Disables the Autoplay mode if Balance is reduced by more than Loss Limit amount. Stop on Bonus Feature (where available) - Disables the Autoplay mode when Bonus game or Free games are triggered. Stop on Win above (where available)- Disables the Autoplay mode when there is a single round win exceeding the amount entered. Autoplay button will display the remaining number of spins to be played automatically. Pressing on the Autoplay button while Autoplay mode is enabled will reset the number of Rounds and disable the Autoplay mode. MALFUNCTION Malfunction of the game voids all pays. In case the game round is interrupted, it can be restored by simply starting the game again, within the period of 90 days. After that period, the incomplete round will be void.
