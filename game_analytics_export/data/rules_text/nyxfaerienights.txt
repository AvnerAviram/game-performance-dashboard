Faerie Nights GAME RULES : You play a fixed 25 lines per spin with 1 coin per line. Only the highest win on any winline is paid. Winning combinations and payouts are made according to the paytable. Malfunction voids all pays and plays. This game features Expanding, Sticky and Transform Wilds. Scatter symbols and Symbol Multipliers. All Wilds substitute for any basic symbols and pay as the highest win on any winline. Expanding Wilds expand to cover the entire reel. Sticky Wilds persist on reels over multiple spins. Sticky Wilds also increase and decrease in number during freespins. Transform Wilds replace basic symbols on the reels with the Wild symbol. Freespins are achieved via Scatter symbols which do not need to be on winlines to activate the feature. Symbol Multipliers multiply total winnings per spin. Symbol Multipliers only appear on reels 1 and 5. Multiple Symbol Multipliers are cumulative. The Return to player (RTP) is : 96%. The maximum win from a single game is £250,000.
