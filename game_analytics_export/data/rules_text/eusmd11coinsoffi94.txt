11 Coins of Fire About this Game This is a 5 reel 3 row video slot game. The objective of the game is to spin the reels so the symbols land in different combinations. Winning combinations are highlighted on the reels and a payout is awarded. Payline Rules This game includes 25 paylines. The number of paylines played is fixed. A payline is a pattern of adjacent symbol positions that runs in a line across the reels. A win is awarded when matching symbols land adjacent to each other on a payline. Payline wins run from left to right, starting from the first slot reel. Bet Rules Each spin played costs a total bet amount. The initial bet is the current total bet divided by the payline cost of 20. Payout Rules Payline win payouts are based on the initial bet placed. The payout value is based on the winning combination formed. The total payout is the sum of all payline wins and Scatter wins resulting from a spin. If multiple winning combinations land on a single payline, only the highest win is awarded. The Paytable displays the payout for each symbol combination based on the current bet. The win over value displayed is rounded down, and is approximately the highest possible win achievable on a single spin. Wild Symbols Wild symbols can substitute for other symbols to complete winning combinations. Wild symbols can substitute for all symbols except Scatters and Golden Coin symbols. Scatter Symbols Scatters can create their own winning combination when landing anywhere on the reels. Scatters appear on reels 2, 3 and 4 only. Free Spins 3 Scatter symbols anywhere on the reels trigger this feature. 10 Free Spins are awarded. Upon entry to Free Spins, extra Wild symbols are added to the reels. 30, 60 or 100 Wild symbols are randomly added. Free Spins can be retriggered. 3 Scatter symbols retrigger Free Spins. The number of spins initially triggered is awarded. All bets played are the same as the spin that triggered this feature. All lines played are the same as the spin that triggered this feature. Free Spins play out automatically. Wins awarded during the feature are added to base game wins. Epic Strike™ Feature Golden Coin symbols can create their own winning combination when landing anywhere on the reels. 3 or more Golden Coin symbols award a cash prize. Golden Coin symbol wins are added to the payline wins. Treasure Pick feature Golden Coin symbol anywhere on the reels can trigger the Treasure Pick feature. The graphical collection of Golden Coin symbols in this game is purely for visual entertainment and as such does not influence triggering the features. The feature begins with displaying 15 items. Triggering Golden Coin symbols stay revealed, the other items are represented by chests. Chests can be opened by tapping on them. Revealing the Silver Coin symbol ends the feature. The number of Coin symbols revealed determines the size of the cash prize awarded. Missed prizes in closed chests are revealed when the feature is completed. Autoplay Autoplay allows a selected number of spins to play out automatically. Spins are played at the current selected bet. The number of spins remaining in the Autoplay session is displayed in the game. Autoplay Settings Some settings and features may not be available in this game. Spins: Sets the number of spins for the Autoplay session. Autoplay ends when these spins have played out. Total Bet: Sets the current bet value. Bet Settings Quick Bet buttons: Sets the current bet to the selected value. Bet Amount: Sets the current bet value. Game Controls Balance: Displays the current balances. Bet: Displays the current total bet. Win: Displays the most recent win. Bet button: Displays options to change the current bet. Menu button: Displays additional game options. Spin button: Starts a reel spin. Pressing the Spin button during the current spin stops the reels quicker. This does not influence the results of the spin. Autoplay button: Displays Autoplay options. Pressing the Spin button during AutoPlay ends the current Autoplay session. Home button: Exits the game. Additional Information Some settings and features may not be available in this game. Malfunction voids all pays and plays. In the event of a disconnection, the final game state is displayed on return to the game. The balance is updated with the amount won or lost. The results of the last game played are displayed. If a request does not reach the server before disconnection, the results of the previous game played are displayed. Any features in progress that require interaction or selection can be continued. Any changes to game rules will be conducted in accordance with regulatory requirements.
