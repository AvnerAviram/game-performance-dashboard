7's on Fire How to play Select your stake If you wish to adjust this, click the STAKE '+' and '-' buttons to increase or decrease the size of your stake. Spin the reels Click the SPIN button to start your game. Info All line win amounts are shown in the games pay table as multiples of the total stake. These can be viewed by pressing the HELP button and are displayed as cash values in the game which change as the stake per line is adjusted. Winlines The default win line selection is 5 lines which is fixed for every game. Base Game Awards The awards can be viewed in the game using the Help button. Any wins will be automatically credited to your account. The theoretical average return to player (RTP) is 95.12%. This RTP represents the long-term expected payback of the game which has been calculated and verified by an independent testing company. Malfunction voids pays and plays.
