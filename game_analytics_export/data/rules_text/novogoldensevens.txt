Golden Sevens Main Game Your aim is to land 5 identical symbols along a win line. Winning combinations start on the first reel to the left of the machine and run along an active win line to the last reel on the right in the regular game. To win, the symbols must lie next to each other without any other symbols coming between them. Rules All pays are for wins of left to right. All prizes are on selected lines. Highest win only paid per selected line. Line wins are multiplied by credits bet per line. Return to Player The theoretical return to player for this game is 95.10%. The maximum win from a single game is £250,000. Last modification date: 6/18/2020
