Red Hot Repeater Main Game Red Hot Repeater is played on 5 x 3 reels and 10 win lines. Your aim is to land 5 identical symbols along a win line. Winning combinations start on the first reel on the left and run along an active win line to the last reel on the right in the regular game. To win, the symbols must lie next to each other without any other symbols between them. Repeat Feature All wins will enter the repeat chance feature. The arrows advance around the trail and if they stop in a position where the repeat is lit then the initial win is awarded again. This will continue until the trail stops where the repeat isnt lit. The player then collect the total win. Multiplier Feature On a random chance after a successful repeat the player may be awarded the multiplier feature. If awarded the player advances one arrow around the board until it stops on a multiplier. This will multiply the initial value that the player entered the repeat feature on. Rules The award multiplier is multiplied with the bet per line (i.e. total bet/pay-lines). Only the highest value symbol combination is paid on each pay-line Return to Player The theoretical return to player for this game is 95.67% The maximum win from a single game is £250,000.
