Luau Loot Base Game WILD FEATURE WILD is wild for all symbols except FEATURE Paytable reflects current bet configuration. Only highest winner paid per winning combination. Line pays must occur on adjacent reels, beginning with the leftmost reel. Feature Description 3 or more scattered FEATURE symbols trigger the Feature. 5 scattered FEATURE symbols award 20 free spins. 4 scattered FEATURE symbols award 7 free spins. 3 scattered FEATURE symbols award 5 free spins. Click the "Start Feature" button to start the free spins. 2 random reels become wild on every spin! WILD is wild for all symbols except FEATURE During free spins, 3 scattered FEATURE symbols add 5 additional free spins to the amount of free spins remaining. During free spins, an alternate set of reels is used. Winning combinations for these reels are identical to the base game. During Free Spins, the bet per line and the active paylines remain the same as the spin that triggered the feature. Paylines Wins on multiple paylines are added together. Total line bet must be divided equally between each active payline; therefore, all total bet amounts are not available. The total bet is the number of lines multiplied by the cash bet on each line. Only active paylines can register wins. Malfunction voids all pays and plays. Last modification date: 1/31/2017
