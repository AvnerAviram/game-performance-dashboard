Vampire Hunters Rules This game has an RTP of 96%. You play a fixed 25 lines per spin with 1 coin per line The value of the coin is: Bet value/25 Only the highest win on any winline is paid Winning combinations and payouts are made according to the paytable Malfunction voids all pays and plays Features This game features Wilds, and both Freespins and Bonus, Scatter symbols Wilds substitute for any non-special symbols and pay as the highest win on any winline The Bonus Feature is achieved via Special(Bonus) Scatter symbols which do not need to be on winlines to activate the feature The Bonus Feature is a Pick-a-Prize game with 5 chances to win Match 3,4, 5 Special(Bonus) symbols awards the Bonus Feature The Freespins Feature is achieved via Special(Scatter) Scatter symbols which do not need to be on winlines to activate the feature Match 3,4, 5 Special(Scatter) symbols awards 10,14,18 Freespins During Freespin Feature, matching 3,4,5 Special(Scatter) symbols awards 4,6, 8 extra freespins The maximum win from a single game is £250,000 or 10,000xBet - whichever is reached first (or currency equivalent.
