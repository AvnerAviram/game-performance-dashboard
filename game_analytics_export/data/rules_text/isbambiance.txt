Ambiance How to Play Note: If you do not have enough money to play a game, the bet can be set to the minimum selectable amount through a dialog window informing about this. All winning lines pay from left to right. Wins on different selected lines are all added. Only the highest winning combination on each selected line is paid. Winnings from Scatter and free spins are added to the total win. Flash version Note : On the info bar located under the main game buttons, you can view the current BET, your BALANCE and your current WINNINGS, in your currency. To play, follow these steps: Use the coin value select button to choose a coin value. The coin value select button is located in the bottom-left side of the main game screen and shows the currently selected coin value. Note : The current bet amount, in your currency, is displayed in the BET box. Click SPIN to place the bet and spin the reels once. You can click the BET MAX button to place the maximum bet for the currently selected coin value and spin the reels once. The total amount won, in your currency, according to the paytable, is shown in the WINNINGS box. For each winning combination, the amount won is the amount shown in the paytable for that winning combination. Special Symbols and Features Wild The Wild symbol replaces any other symbols except Scatter. During each free spins round, at random, two or more symbols, transform into Wild . Scatter Scatters pay anywhere on the reels, once per spin. Scatter winnings are shown in the paytable. Each Scatter symbol that appears on the reels is collected. When you collect five or more Scatter symbols, you win a number of free spins that is equal to the number of collected Scatter symbols. You can view the current number of collected Scatter symbols, in the Flash version, next to the BET MAX button and in the Mobile version, under the Spin button. Note: For the Flash version: when you win free spins, click START to begin playing the free spins rounds; at the end of the free spins, click RESUME to return to the main game screen. During free spins, each Scatter symbol that appears anywhere on the reels, awards an additional free spin. The number of collected Scatter symbols is reset to zero after the free spins end and you can start collecting again. The number of collected Scatter symbols is also reset to zero if, before you collect the necessary minimum number of Scatter symbols to win free spins, you change the coin value. Autospin feature The loss limit setting allows stopping the autospin feature automatically when the selected limit plus your cumulative winnings minus your cumulative bets is less than 0. Note : The autospin feature does not automatically start free spins - player input is required. Flash version Click the AUTO SPIN button to open the AUTO SPIN SETTINGS window where you can set up the autospin feature. On the AUTO SPIN SETTINGS window, you can set up the following: Loss limit - enter a number for the loss limit value or use the left and the right arrow buttons to select a number for the loss limit value Stop if WIN reaches - to enable this setting, click the button on its left side and then enter a number for the WIN value or use the left and the right arrow buttons to select a number for the WIN value; click the button on its left side again to disable it Stop if JACKPOT is won - (only for games with Progressive Jackpots or with a specific Jackpot value displayed) - to enable this setting, click the button on its left side; click the button on its left side again to disable it Number of AUTO SPINS to play - use the left and the right arrow buttons to select the number of autospins Click START to close the AUTO SPIN SETTINGS window and start the autospin feature with the currently selected bet and number of autospins or click CANCEL to close the AUTO SPIN SETTINGS window without starting the autospin feature. Click the STOP xx/yy button to stop the autospin feature at any time; xx represents the remaining number of autospins and yy represents the number of autospins to play that you selected on the AUTO SPINS SETTINGS window.
