Mayan Goddess Game Rules Malfunction voids all pays and play Choose the amount you want to bet Click the Spin button to spin the reels This game is played at 720 ways and all 5 reels are always in play. Ways wins contain one symbol from each adjacent reel, beginning with the leftmost reel, or ending with the rightmost reel. The same symbol, or its substitute, in a different position on the same reel pays that ways win again. Only the highest win is paid for each ways symbol combination. Ways wins are multiplied by the bet multiplier. Scatter symbols award in any position on the reels. During the base game any win with a multiplier appearing in any position on the reels awards the displayed multiplier value. If more than one multiplier appears, the sum of the multiplier values is awarded. Wins are shown in credits unless marked as currency. Free Games bonus During the base game, 3 Free Games Bonus Symbol in any position on the reels starts the Free Games Bonus. Awards 8 free games. Free Games Bonus symbol is a scatter symbol. Different reels are used during the Free Games Bonus. Free games automatically play on the same lines at the same bet multiplier as the game that initiated the Free Games Bonus. Each free game win is multiplied by the current bonus multiplier for that game. The bonus multiplier starts at 5x. For each multiplier appearing on the reels, the bonus multiplier is increased by the value shown on that symbol. The Free Games Bonus does not award additional free games. The Free Games Bonus ends when 0 free games remain. RTP: 96.0% The maximum win from a single game is £250,000.
