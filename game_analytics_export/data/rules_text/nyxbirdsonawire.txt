Birds On A Wire Untitled Document Game Rules Note: The Birds On A Wire® video slot is referred to as Birds On A Wire®, The Game or Game. Birds On A Wire® is a video slot with 15 reels and 17 fixed pay lines. The game features dropping symbols, wilds, multipliers and bonus game. The theoretical return to player is 97,2%. The game offers up to 15 bet levels for the selected currency. A game round is started with the action button and plays the game with the selected bet level. Auto play plays the game automatically for the number of game rounds selected in the auto play page. All game payout and win combinations are paid out according to the pay table. All wins are presented and paid out in the selected currency. Only the highest win per pay line is paid out, if in sequence from far left to right in combinations according to the pay table. Symbols included in pay line wins are replaced with dropping symbols and/or existing symbols. This continues as long as there is a new pay line win. For each win on a symbol drop, the multiplier increases one level up to the maximum multiplier level. The wild symbol substitutes for all symbols except the bonus symbol. The bonus symbol is a scatter symbol that grants the bonus game if 3 or more bonus symbols are won. The number of bonus symbols won correlates to the number of free spins granted in the bonus game according to the pay table. Inwinity Spin® is a feature granting the player an unlimited amount of free spins until a pay line win is reached. The bonus game is played with the same bet as the spin that triggered the bonus game. No bets can be altered during a game round. In the event of game malfunction all affected bets and pays are discharged. Last modification date: 3/28/2017
