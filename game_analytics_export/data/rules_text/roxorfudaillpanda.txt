Fu Dai Lian Lian Panda GAME RULES All wins are shown in Coins. 50 Lines are played for a Line Cost of 88 Coins. Bet is 88 Coins x Coin Size. The SPIN button will immediately commence play. All wins on lit lines only except Bonuses and Bonus Coin Prizes. Bonus Prizes and Bonus Coin prizes are added to line wins. Payline wins pay left to right on consecutive reels from the leftmost reel. Wins on different lines are added. Only highest win is paid on any active payline. Game RTP: 95.06%. FU DAI LIAN LIAN FEATURES Whenever one or more appear there is a chance the PROSPERITY FEATURE may be triggered. Whenever one or more appear there is a chance the JACKPOT FEATURE may be triggered. Whenever one or more appear there is a chance the LONGEVITY FEATURE may be triggered. If only PROSPERITY FEATURE is triggered in the spin, the PROSPERITY FEATURE is awarded. If only LONGEVITY FEATURE is triggered in the spin, the LONGEVITY FEATURE is awarded. If only JACKPOT FEATURE is triggered in the spin, the JACKPOT FEATURE is awarded. If only PROSPERITY FEATURE and JACKPOT FEATURE are triggered in the same spin, the PROSPERITY JACKPOT SUPER FEATURE is awarded only. If only LONGEVITY FEATURE and JACKPOT FEATURE are triggered in the same spin, the LONGEVITY JACKPOT SUPER FEATURE is awarded only. If only PROSPERITY FEATURE and LONGEVITY FEATURE are triggered in the same spin, the PROSPERITY LONGEVITY SUPER FEATURE is awarded only. If PROSPERITY FEATURE, LONGEVITY FEATURE and JACKPOT FEATURE are triggered in the same spin, the MEGA FEATURE is awarded only. At the conclusion of the PROSPERITY FEATURE, LONGEVITY FEATURE or PROSPERITY LONGEVITY SUPER FEATURE, one feature may be awarded as determined by the combination of feature triggers collected. PROSPERITY FEATURE, LONGEVITY FEATURE and JACKPOT FEATURE can only be triggered once per feature. If only PROSPERITY FEATURE trigger was collected, the PROSPERITY FEATURE will be awarded. If only JACKPOT FEATURE trigger was collected, the JACKPOT FEATURE will be awarded. If only LONGEVITY FEATURE trigger was collected, the LONGEVITY FEATURE will be awarded. If only PROSPERITY FEATURE and JACKPOT FEATURE triggers were collected, the PROSPERITY JACKPOT SUPER FEATURE will be awarded. If only LONGEVITY FEATURE and JACKPOT FEATURE triggers were collected, the LONGEVITY JACKPOT SUPER FEATURE will be awarded. If only PROSPERITY FEATURE and LONGEVITY FEATURE triggers were collected, the PROSPERITY LONGEVITY SUPER FEATURE will be awarded. If PROSPERITY FEATURE, LONGEVITY FEATURE and JACKPOT FEATURE triggers were collected, the MEGA FEATURE will be awarded. Malfunction voids all pays and plays Last modification date: 8/14/2024
