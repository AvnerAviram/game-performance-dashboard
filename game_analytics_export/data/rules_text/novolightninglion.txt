Win big in this game and let this lion roar! Main Game The main game is played on 3 reels and 25 win lines. Watch the Diamonds! They are not only Wild but multiply your winnings as well. Whilst playing the main game, 3 progressive Free Games Meters will increase once in a while. The Free Games Feature will start when 3 scattered Bonus symbols of the same color appear in the reel window or when one ore more Free Games counters hits 15. Free Games Feature During the Free Games Feature you really start to win big. The games are not only for free but will be multiplied as well. 2x for the Blue Bonus, 5x for the Green Bonus and 10x for the Purple Bonus. Hit 3 scattered Yellow Bonus symbols and 5 extra Free Games will be added! Rules Wins are shown in credits. Scatter wins and line wins are added.
