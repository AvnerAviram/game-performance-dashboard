Jewel in the Crown Base Game All line pay symbols must appear on a played line and on consecutive reels. Only the highest win is paid in each line played. All line wins are multiplied by the total stake. Wins that occur on all selected lines are added to the total win. Scatter wins are awarded for symbols appearing in any position on the reels for any number of played lines and paid in addition to line wins. Malfunction voids pays and plays. Game Rules In accordance with fair gaming practices, the outcome of each and every game is completely independent. The chance of getting a particular outcome are always the same at the start of every game. At < 2.00 the expected return for this game is 94.00%. At > 2.00 the expected return for this game is 96.00% The expected return for the Big Bet game is 98.00%. This reflects the theoretical return across many plays. Any combination of wins in a single game is limited and will not exceed 250,000.00. It may not be possible to reach this limit in a single game from every stake configuration. Big Bet Game A Big Bet game consists of 5 linked spins of the reels. All wins are accumulative and are paid at the end of the last spin. Each spin uses all 13 win lines and wins are paid according to the awards table. 20.00 Big Bet. During this game the Crowning Glory trail bonus is activated. Gaining Crown symbols without triggering Free Spins adds the Crown to Crowning Glory trail. 30.00 Big Bet During this game the Crowning Glory trail bonus is activated. Gaining Crown symbols without triggering Free spins adds the Crown to Crowning Glory trail. Stacked Jewels are also active during this Big Bet game 50.00 Big Bet During this game the Super Crowning Glory trail bonus is activated.
