Cash Stax Game Introduction "This is a 5x3 reel game that allows upto 30 lines to be played. The game features a Free Spins Bonus that can re-trigger." How to play Select your stake - If you wish to adjust this, click the STAKE/LINE '+' and '-' buttons to increase or decrease the size of your stake per line. Win lines The default win line selection is 10 lines in the standard game which can be increased to 20 lines by playing at higher stakes. The default win line selection in the Big Bet game is 30 win lines. Start the game Press SPIN to initiate the game. The game contains an AUTOPLAY function that allows the player to select a quantity of games for continual play. Select your desired number of spins by choosing one of the numbered buttons that are revealed when you press AUTOPLAY. Click 'STOP' to cancel auto play. Help All line win amounts are shown in the games pay table as multiples of the total stake. These can be viewed by pressing the "i" button. Free Spins Bonus The Free Spins are triggered by 1 or more Golden O scatter symbols appearing in view on each reel. The number of Golden O scatter symbols determines the number of Free Spins awarded. Golden O scatter symbols are held during Free Spins and award further Free Spins if one or more Golden O symbols exists on each reel. The number of Free Spins awarded is determined by the number of Golden O symbols present above the reels. Achieving 20 or more free spins multiplies all remaining free spin wins by x5. The RTP ranges from 94.12% to 98.14% dependant on stake selected. The expected return for this game is 98.14% when playing the Big Bet game. This RTP represents the long-term expected payback of the game which has been calculated and verified by an independent testing company. Malfunction voids all pays and play.
