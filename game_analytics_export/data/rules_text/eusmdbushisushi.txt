Bushi Sushi About this Game This is a 5 reel 3 row video slot game. The objective of the game is to spin the reels so the symbols land in different combinations. Winning combinations are highlighted on the reels and a payout is awarded. Bet Rules Each spin played costs a total bet amount. The total bet is based on the initial bet placed, multiplied by the cost of the paylines played. Payline Rules This game includes 10 paylines. A payline is a pattern of adjacent symbol positions that runs in a line across the reels. A win is awarded when matching symbols land adjacent to each other on a payline. Payline wins run from left to right and right to left, starting from the first and last reel respectively. If multiple winning combinations land on a single payline and direction, only the highest win is awarded. The number of paylines played is fixed. Bonus Symbol 3, 4, or 5 consecutive Bonus symbols trigger Free Spins when starting from Reel 1 or Reel 5 in either win direction. Only the highest Bonus symbol combination is awarded for each win direction. Bonus symbols pay in any position on the reel. Bonus symbol wins are multiplied by the total bet. This symbol appears in both base game and Free Spins. Wild Symbols Every Wild that lands will expand to form a Wild Reel. Adjacent reels of expanded Wilds form a larger image with the same functionality. Wild symbols can substitute for all symbols. Wild symbols can substitute for Bonus symbols to trigger Free Spins. This symbol appears on reels 2, 3 and 4 only. Wild symbols do not have a payout. This symbol appears in both base game and Free Spins. Payout Rules Payline win payouts are based on the initial bet placed. The total payout is the sum of all payline wins and Bonus symbol wins resulting from a spin. The Paytable displays the payout for each symbol combination based on the current bet. Free Spins Number of Free Spins awarded is dependent on the number of reels involved in the Bonus symbol win. 3, 4, or 5 consecutive Bonus symbols trigger Free Spins when starting from Reel 1 or Reel 5 in either win direction. Free Spins can be retriggered. All bets played are the same as the spin that triggered this feature. All lines played are the same as the spin that triggered this feature. Wins awarded during the feature are added to base game wins. Free Spins are played on a different reel set from the base game. Autoplay Autoplay allows a selected number of spins to play out automatically. Spins are played at the current selected bet. The Autoplay start button starts the Autoplay session based on your selected options. The number of spins remaining in the Autoplay session is displayed in the game. If any condition is met to stop the session, all AutoPlay settings return to default. If you are disconnected, AutoPlay will stop. All AutoPlay settings return to default when the game is reloaded. Autoplay Settings Spins: Sets the number of spins for the Autoplay session. Autoplay ends when these spins have played out. Loss Limit: Sets a loss limit for the Autoplay session. Autoplay ends if the cost of the next spin would decrease the balance by this amount. Single Win Limit: Sets a win limit for the Autoplay session. Autoplay ends if a single win reaches this amount.
