Game Engine : Clone - ThunderingBuffalo Game Features 3-4-5-4-3 reels configuration 720 ways Stacked Wilds Re-Triggerable Free Games Bonus Explanation of Features Stacked Wilds Stacked Wilds refers to a game where some or all of the reels have 3 or more wild symbols appearing consecutively, allowing that entire reel to be covered with wild symbols as the end result of a spin. Re-triggerable Free Games Bonus The Free Games Bonus is triggered when five or more Bonus Symbols land in any position on five consecutive reels. The Free Games Bonus is automatically played on the same ways at the same bet multiplier as the game that initiated the Free Games Bonus. Additional free games may be awarded when five or more Re-Trigger Bonus Symbols in any position on five consecutive reels. The maximum number of free games played is 240.
