Big Red 5 free games are won for each lit line where one or more Kangaroo substitutes in a win. Press the PLAY button to start the REPEAT WIN FREE GAMES FEATURE. During the REPEAT WIN FREE GAMES FEATURE, each Tree that appears awards a REPEAT WIN. A REPEAT WIN awards the player the line pay that triggered the REPEAT WIN FREE GAMES FEATURE. When more than one REPEAT WIN FREE GAMES FEATURES are awarded, the triggering line pay for each REPEAT WIN FREE GAMES FEATURE will be shown on screen. A maximum of 3 REPEAT WIN FREE GAMES FEATURE triggering line pays will be shown at any one time, although all REPEAT WIN FREE GAMES FEATURES triggered will be awarded. REPEAT WIN FREE GAMES FEATURE can be triggered again during the REPEAT WIN FREE GAMES FEATURE. Bet is the same as the game that triggered the REPEAT WIN FREE GAMES FEATURE. GAME RULES 10 lines are played for line cost of 10 Coins. Bet is 10 Coins x Coin Size. All wins on played lines only except scatters Wins on different lines are added. Highest win only on each line. Highest win only for scatters. Scatter wins and Repeat wins are added to line wins. RTP: 95.13%. The RTP is the amount we pay out to players relative to the amount of wagering on the game.
