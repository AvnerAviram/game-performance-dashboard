Viracocha 1. Overview The goal is to obtain a winning combination on a winning line spread across the reels. Game specifications : Type Slots Number of reels 5 Number of rows 3 Number of lines 25 Number of coins you can bet 25 (cannot be changed) Progressive No Double feature No Auto spin feature Yes Wild symbols Yes Multiplier symbols No Scatter symbols Yes (Bonus Symbol) Bonus round Yes Free s pins Yes Instant win No The game disconnects if you do not take any actions for a specific number of consecutive minutes (this number varies depending on jurisdiction); the session recovery feature will save the game progress so that the next time the game is loaded, you can continue where you left off. In the event of technical difficulties resulting in a dropped connection to the game server, the game you were playing is saved automatically and will load the next time you establish a connection to the game server. You will not be able to reconnect to the game server for the first 30 seconds after the disconnection. Note : In case of malfunction all pays and play are voided; any bet that was not yet accepted will be voided, any bet that was unsettled will be refunded. 2.1. Wild Stacked Wilds can only appear on reels 4 and 5, and substitute for all symbols except the Bonus trigger. 3.2. Free Spins Free Spins are triggered by Bonus symbols on reels 1, 3, and 5. On the initial stage, vases are selected until 4 matching coins are revealed. Matching four identical coins awards one of the following Bonus rounds : 4 gold coins = 10 Free Spins with 4 random Wilds 4 silver coins = 8 Free Spins with 3 random Wilds 4 bronze coins = 6 Free Spins with 2 random Wilds Depending on the bonus, 2, 3 or 4 additional Wilds will be placed randomly on the reels at the beginning of each Free Spin . More Free Spins can be won during the bonus. The maximum number of Free Spins that can be won is 150. If the Bonus is retriggered, a new picker round will take place when the current Free Spins end. If a Wild appears over a Bonus symbol , it will count as a Bonus symbol as well as contributing to line wins. A line win of 5 Wild symbols will pay the equivalent of 5 of the top paying symbol. 3 or 4 Wild symbols on an active payline will pay according to the next symbol in the payline. The main game and Free Spins reels do not have the same symbol composition. 4. Other game buttons 4.1. Auto spin feature The loss limit setting allows stopping the auto spin feature automatically when the selected limit, plus your cumulative winnings, and minus your cumulative bets, is less than 0. 5.1. Desktop version Click the AUTO SPIN button to open the AUTO SPIN settings window where you can set up the following: Number of auto spins : click + or - to select the number of auto spins to play. Loss limit : click + or - to select the value of the loss limit; Stop if a win reaches : click + or - to select the value of the maximum win; Stop if JACKPOT is won (only for games with Progressive Jackpots): to enable this setting, tick the checkbox ; to disable it, remove the tick.
