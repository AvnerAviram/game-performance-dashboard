Fu Bao Meow User Interface ‘+’ / ‘-’ buttons increase / decrease the bet value — the bet value is the total bet per game round. Starts a game round at the current bet value. Displays the autoplay options. Stops the autoplay session. Exits the autoplay menu. Starts the autoplay session. The numerical value displayed within the button shows the number of autoplay game rounds remaining Bet Definitions Current Bet: The current bet is the value that appears above Minimum Bet: The minimum bet is the lowest possible bet value, which is selected by pressing until the bet value no longer decreases. Bet Multi plier: The bet multi plier is defined as the current bet divided by the minimum bet. Autoplay Options Autoplay will allow game rounds to be played automatically. Click on the Autoplay butt on and select the number of Autoplay game rounds. Additi onal Information The overall return to player is 96.76%. Malfunction voids all pays and plays. Wins A winning play occurs when at least 1 of the 3 beckoning cat's arm lowers during the game round. 3 to 17 Currency Wins are awarded on a winning play. Multiplier Feature If 2 or 3 cat arms lower on any winning play, the Multi plier Feature is activated. If 2 cat arms lower on any winning play, a x3 will appear and all Currency Wins will be multiplied by the multiplier. If 3 cat arms lower on any winning play, a x15 will appear and all Currency Wins will be multiplied by the multiplier. Last modification date: 2/22/2023
