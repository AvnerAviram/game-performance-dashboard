Main Game & Win Ways™ Smashing Sevens™: Win Ways™ is played across a vertical reel set and a horizontal extra reel. The main reel set consists of 6 vertical reels. The maximum number of symbols for reels 1 and 6 are 7. For reels 2-5 the maximum number of symbols are 6. The horizontal extra reel always has 4 symbols. Each of these extra symbols completes the reel directly below (2nd, 3rd, 4th and 5th). A winning path (Win Ways™) is created when a combination of identical symbols starting on the leftmost reel appears. The number of Win Ways™ is up to 117649 for each spin. Winnings are awarded from left to right for adjacent symbols, except Bonus symbols. The Wild symbol replaces all symbols except the Bonus symbol and only appears on the Extra Reel. Reel Refill™ Feature All winning symbols (except Bonus symbols) included in a winning combination will explode and are then replaced by new symbols. The new symbols fall from above into the main reel set and slide in from the right for the Extra Reel. This exciting process is repeated until no symbols are included in a winning combination. Free Games & Win multiplier 3 or more Bonus symbols in any position trigger Free Games with a potentially unlimited win multiplier. The number of Free Games depends on the number of Bonus symbols that triggered the feature. All wins during Free Games are multiplied by the win multiplier. The win multiplier starts at one and increases by one after each Reel Refill™. Rules All prizes are for combinations left to right for adjacent symbols, except Bonus symbols. 3 or more Bonus symbols in view trigger Free Games. Free Games can be won again during Free Games. Free Games are played with the same bet as the spin which triggered them. In Free Games, Bonus symbols can appear only on the Extra Reel. Wild Symbols only appear on the Extra Reel. Malfunction voids all pays and plays.
