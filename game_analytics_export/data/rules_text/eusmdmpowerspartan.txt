Magnificent Power Spartan About this Game This is a 5 reel 4 row video slot game. Up to 6 reels, 4 rows can be activated. The objective of the game is to spin the reels so the symbols land in different combinations. Winning combinations are highlighted on the reels and a payout is awarded. This game includes 1024 ways. Up to 4096 ways can be activated. Payout Rules If multiple winning combinations land on a single way, only the highest win is awarded. Only the highest winning combination is awarded per symbol combination. The payout value is based on the winning combination formed. The total payout is the sum of all way wins resulting from a spin. Way win payouts are based on the initial bet placed. The Paytable displays the payout for each symbol combination based on the current bet. Way Rules A way is 1 position on each reel beginning with the leftmost reel and pays left to right on adjacent reels. Only positions containing a winning symbol are used to determine the way win for that symbol. When 5 reels are available, the game has 1024 ways. When all 6 reels are unlocked, the game has 4096 ways. Only the highest win is awarded per way. Bet Rules Each spin played costs a total bet amount. The initial bet is the current total bet divided by the way cost of 20. The total bet is based on the initial bet placed, multiplied by the cost of the ways played. Wild Symbols Wilds only substitute for High and Low Paying symbols. Wild symbols can substitute for other symbols to complete winning combinations. Wilds only appear on reels 2, 3, 4 and 5. These symbols do not have a payout. Random Multiplier Wild Random Multiplier Wild is Wild and will be 1x, 2x or 3x. The Multiplier on the Wild is the number of ways the Wild position is worth. Magnificent Power™ Feature 1 or more Wilds on screen will collect all Coins. Collect 30 Coins to upgrade the base game Wild to Random Multiplier Wild. Wild upgrade will occur at the beginning of the next game. Collect 60 Coins to unlock Jackpot Spin feature, Wild Spin feature, Bonus symbol, and the 6th reel. Jackpot Spin Feature, Wild Spin Feature, Bonus symbol, and the 6th reel unlock will occur at the beginning of the next game. Coins may appear on any High and Low Paying symbols. Falling Wild Feature After a Wild lands on the reels, all wins are awarded for that spin. If the Wild landed on the top 3 rows of the reels, the Wild will move down one position and all the reels will respin. The respin has special reels with High Paying and Low Paying symbols. All wins will be awarded after each respin. The Falling Wild feature will continue until the Wild lands on the bottom row. The Falling Wild will be a Random Multiplier Wild if the feature is unlocked. Coins are available to collect during the respin until the 6th reel is unlocked. The following features are not available during the Falling Wild Feature: Bonus symbol, Jackpot Spins and Wild Spins. Bonus Symbols Bonus symbols appear in base game only. Bonus symbol can trigger either the Jackpot Spins or the Wild Spins, but not both. Bonus symbol can increase the Jackpot Spins by 1 and the Wild Spins by 1. If the Bonus Symbol does not trigger a feature, it will accumulate. Jackpot Spins The number of spins in the Jackpot Spins will be awarded. Jackpot Spins have special reels with Jackpot symbols, Jackpot Extra Spin symbols, and blank positions only. Jackpot Extra Spin symbol will increase the total number of spins by 1. The feature pays at the end of the feature when no spins are remaining. The number of Jackpot and Jackpot Extra Spin symbols accumulated during the spins determines the Jackpot award. The Jackpot Spins meter will reset to 3 after the Jackpot Spins are complete. If the Jackpot Spins and the Falling Wild feature trigger in the base game on the same spin, the Falling Wilds feature will complete before entering the Jackpot Spins. The number of reels are the same as the spin that triggered the feature. This feature can trigger in base game only. All bets played are the same as the spin that triggered this feature. Wild Spins The number of spins in the Wild Spins meter will be awarded. Wild Spins have bonus reels with Wild symbols, highest paying High Paying Symbol and blank positions only. Falling Wild feature is available during the Wild Spins. When the Random Multiplier Wild is unlocked, Wild will be a Random Multiplier Wild that always awards a 2x or 3x. The Falling Wilds feature does not affect the number of spins remaining. The feature ends when there are no more spins and the Falling Wild feature has completed. If the Wild Spins and the Falling Wild feature trigger in the base game on the same spin, the Falling Wilds feature will complete before entering the Wild Spins. All bets played are the same as the spin that triggered this feature. All ways played are the same as the spin that triggered this feature. This feature can trigger in base game only. Autoplay Autoplay allows a selected number of spins to play out automatically. Loss Limit: Sets a loss limit for the Autoplay session. Autoplay ends if the balance decreases by this amount. Spins are played at the current selected bet. Spins: Sets the number of spins for the Autoplay session. Autoplay ends when these spins have played out. The number of spins remaining in the Autoplay session is displayed in the game. This feature is available in base game only. Game Settings Quickspin: Enables Quickspin to resolve spins faster. This does not influence the results of the spin. Left-Handed Mode button: Enables left-handed mode. Music: Enables game music. Sounds: Enables game sounds. Startup Screen button: Enables game advertisement on game load. Bet Settings Bet Value buttons: Sets the current bet to the selected value. Max Bet button: Sets the current bet to the maximum value. Free Games A Free Games offer is a number of additional game spins awarded to a player. Each offer has a unique reference number. The bet configuration is set by the game managers and cannot be changed. Wins are added to the cash or bonus balance, depending on the offer. An offer can be played now, played later, or discarded. Select Play Now to begin playing a Free Games offer. Only one Free Game offer can be played at a time. Select Play Later to postpone the offer. A postponed offer will be available again when re-entering the game. Select the Bin icon to discard the Free Games. A discarded offer can only be re-opened by contacting the game managers and providing the reference number. When an offer is complete, discarded, or expired, the next available offer is displayed. Each offer has an expiry date and time; if an offer is not played before this time, it will no longer be available. Select the Information icon to view bet information and the offer expiry date and time. Additional Information Some settings and features may not be available in this game. Malfunction voids all pays and plays. Any changes to game rules will be conducted in accordance with regulatory requirements. In the event of a disconnection, the final game state is displayed on return to the game. The balance is updated with the amount won or lost. The results of the last game played are displayed. If a request does not reach the server before disconnection, the results of the previous game played are displayed. Any features in progress that require interaction or selection can be continued.
