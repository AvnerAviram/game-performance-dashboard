Blackbeard's Compass Game rules • This game has an RTP of 95.9% • You play a fixed 25 lines per spin with 1 coin per line • The value of the coin is: Bet value/25 • Only the highest win on any winline is paid • Winning combinations and payouts are made according to the paytable • Malfunction voids all pays and plays • This game features Transformation symbols, Bonus Scatter symbols and Freespins Scatter symbols • Each spin all Transformation symbols transform into the same random non-special symbol • In the case of a losing spin the compass animation is skipped and just shows the transformed symbols on the reels. • During Freespins Transformation symbols persist on the reels until the end of freespins • The Freespins Feature is achieved via Scatter symbols which do not need to be on winlines to activate the feature • Freespins Scatter symbols only appear on reels 1, 3 and 5 • The Bonus Feature is achieved via Scatter symbols which do not need to be on winlines to activate the feature • The Bonus Feature is a trail game with up to 7 chances to win • Bonus Scatter symbols only appear on reels 2,4 and 5 The maximum win from a single game is £250,000 Last modification date: 6/4/2020
