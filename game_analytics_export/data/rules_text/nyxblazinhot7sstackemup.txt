Blazin' Hot 7s Stack Em Up How To Play This game starts with one 3x3 set of reels; wins are paid for 3 of the same symbol from left to right, and on any of the 5 active paylines. Every time a win is paid, the winnings symbols are removed and the symbols above cascade down, with new symbols filling any empty positions. If any wins are present, these are paid and the symbols cascade again. Each time a cascade occurs, one level on the Free Spins Ladder is added, and if this ladder reaches a position indicating a number of Free Spins, the Free Spins Bonus is awarded after the conclusion of the cascades. If the cascades finish without triggering Free Spins, the Free Spins Ladder is reset to the first position. Stack ‘Em Up Each STACK EM UP symbol in view on an active set of reels will unlock another set of reels. The STACK EM UP symbols is then removed, and the symbols above will cascade down, with new symbols filling empty positions. Any wins present are paid, and any further STACK EM UP symbols are actioned. If there are no further wins or STACK EM UP symbols, all unlocked reel sets will then be locked again, ready for the next game. Free Spins Bonus The Free Spins bonus is triggered with the displayed number of spins on the Free Spins Ladder. This will then count down with each Free Spin played. If Free Spins was triggered with all 4 reel sets unlocked, any further STACK EM UP symbols will add a win multiplier. The maximum value of the multiplier is x10, and Free Spins cannot retrigger. Free Spins are played with the same total bet, awards and winning combinations as the base game which triggered the bonus. Free Spins will end when there are no Free Spins remaining. Free Spins will end when the maximum win has been achieved. Return To Player The theoretical return to player of this game is 96.06% Malfunction Voids all pays and plays.
