Golden Roar Boom Main Game Golden Roar BOOM is played on 5 reels, each 3 rows high, with 10 fixed win lines. All winning combinations run from left to right, except Boom Bonus (Scatter) symbols. The Wild symbol (Gold Dancer) substitutes for all symbols except for Scatter symbols and doubles the win when substituting. 6 or more Boom Bonus symbols trigger the Golden Roar BOOM Feature with 6 or more initial Boom Spins respectively. Game Features Golden Roar BOOM Feature The Boom Feature is played on 7 reels with each having 5 individual spinning positions. The Feature starts with 2 Boom Pays on reels 3 and 5 and the number of initial Boom Spins is the same as the number of triggering Boom Bonus symbols. After each spin all new symbols are held and recombined automatically in order to ensure the best Boom Pay possible. If you have luck on your side, you might be awarded with 3 Grand symbols during the Feature, and if this happens, the Grand Jackpot is yours!
