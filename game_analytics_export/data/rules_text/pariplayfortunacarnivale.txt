Fortuna Carnivale GAME RULES A 5-reel, 3-row video slot with 25 paylines. All symbols pay from left to right on adjacent reels starting from the leftmost reel,except Instant Win symbols. Winning combinations and payouts are calculated according to the game's paytable. Only the highest combination is paid per payline. Wins on multiple paylines are summed into a total win amount. The SPACE key can be used to start the slot's spin. Malfunction voids all pays and plays. The theoretical payout of the game (RTP) is 96.00%. Set Bet: This screen shows the number of paylines available in the game, and the current total bet. The + or - buttons are used to change the values and set the bet. Settings: Sound Effects - toggles the game's sound effects on/off Music - toggles the game's background music on/off Game Rules - opens the game rules screen Deposit - opens the deposit page Lobby - exits the game back to the portal lobby Autoplay (when available) The “Number of Autospins" slider is used to select the required Autospins. Stop conditions can be set to control when the Autospins will stop: “If cash decreases by” - Autospins will stop if player's balance decreases by the entered value“If cash increases by” - Autospins will stop if player's balance increases by the entered value “If single win exceeds" - Autospins will stop if a single round win exceeds the entered value “On any win” - if selected, Autospins will stop after a winning round “On bonus" - if selected, Autospins will stop when the Free Spins bonus is won (this option is selected by default Last modification date: 1/22/2025
