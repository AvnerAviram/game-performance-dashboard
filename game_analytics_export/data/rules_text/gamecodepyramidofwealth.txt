Pyramid of Wealth Game Rules Pyramid of Wealth is a 5-reel , 3-row video slot featuring BONUS SPINS FEATURE and PICK BONUS FEATURE. Base Game RTP is 96.20%. RTP for Buy BONUS SPINS FEATURE is 96.32%. RTP for Buy PICK BONUS FEATURE is 96.10%. All RTPs represent the long term theoretical payback. Minimum bet: $0.30. Maximum bet: $100.00. The game is always played with 243 Ways except for the BONUS SPINS FEATURE, which may be played with up to 3087 Ways. Each winning ways combination is made up of only one symbol from each reel. All wins pay left to right on adjacent reels beginning with the leftmost reel. Each symbol can be used only once per winning ways combination. Highest win only for each winning combination. BONUS SPINS FEATURE and PICK BONUS FEATURE wins are added to base game wins, if any. Wins on different combinations are added. Displayed on the WILD symbol, credit prize values are proportional to the selected bet. Displayed Mega, Grand, Major, and Minor Jackpot meters are currency values and are proportional to the selected bet. All jackpots are non-progressive. Paytable values are displayed for the currently selected bet. All wins shown in currency. The game outcome is random and cannot be influenced by the actions of the player. Malfunction voids all pays and plays. WILD SYMBOLS There are multiple types of WILD symbols in the base game and the BONUS SPINS FEATURE. WILD and WILD WITH CREDIT PRIZE symbols appear in the base game and the BONUS SPINS FEATURE on reels 2,3,4 and 5. WILD WITH JACKPOT PRIZE symbols appear in the base game and the BONUS SPINS FEATURE on reels 3, 4 and 5. EXPANDING WILD symbol appears in the BONUS SPINS FEATURE on reels 2,3 and 4 only. Each WILD symbol substitutes for all symbols, except the Bonus symbol and other wilds. WILD symbols substitute for any symbol that is a part of the winning combination. WILD WITH JACKPOT PRIZE symbol awards Jackpot prize associated with it when it appears as part of a winning combination. WILD WITH JACKPOT PRIZE symbol is linked to one of the following Jackpot prizes: Grand, Major, or Minor. Each WILD WITH JACKPOT PRIZE carries only one of these Jackpot values. WILD WITH CREDIT PRIZE awards its linked credit value when it is a part of a winning combination. 1 credit equals 1 cent of the currency in play. EXPANDING WILD behaves as a wild symbol after expanding the reels. BONUS SPINS FEATURE 3 Bonus symbols on reels 1,3 and 5 in base game award the BONUS SPINS FEATURE. The BONUS SPINS FEATURE starts with 10 Bonus Spins. Landing 3 Bonus symbols on reels 1,3 and 5 awards 5 additional Bonus Spins. The BONUS SPINS FEATURE starting reels layout is always 5 reels and 3 rows. When an EXPANDING WILD symbol lands on reel 2,3 or 4, it always expands the reel on which it appears by +1 symbol space, revealing an additional symbol that becomes part of the current Bonus Spin and contributes to any winning combinations, if applicable. The expanded reel layout persists for the remainder of the BONUS SPINS FEATURE. Each EXPANDING WILD symbol that lands expands reels 2, 3, and 4 further by +1 symbol space. The maximum possible reel layout is 5 reels with a 3-7-7-7-3 row structure, creating 3,087 Ways to Win.Expanded reels can only reveal one of the following symbols: WILD EXPANDING WILD WILD WITH JACKPOT PRIZE WILD WITH CREDIT PRIZE All the wins collected during the BONUS SPINS FEATURE are summed up and presented as the total win at the end of the feature and added to base game win if any. The BONUS SPINS FEATURE ends once no Bonus Spins remain. PICK BONUS FEATURE WILD, WILD WITH JACKPOT PRIZE or WILD WITH CREDIT PRIZE landing during base game spin may randomly award the PICK BONUS FEATURE. The Pyramid displayed above the reels does not represent actual chances of triggering the PICK BONUS FEATURE. The PICK BONUS FEATURE starts with a 5x5 grid containing a mix of credit values and Jackpots. Credit values are proportional to the currently selected bet. 8 pick coins are available for selection. The collection frame that gathers prizes on which it stops always starts at a fixed position marked as . Selecting a pick coin reveals an arrow, indicating the direction in which the collection frame will move across the grid. The available directional arrows are: down, up, left, right, down-right, up-right, down-left, up-left. Each time a pick coin is selected, the collection frame moves one step in the revealed direction. The prize on the grid space where the collection frame stops is collected. After each selection, all remaining pick coins are revealed, then flipped back and reshuffled before the next selection, if available. All collected prizes are accumulated during the PICK BONUS FEATURE. The collection frame can return to previously visited positions by uncovering more coins. If 10 moves are completed without exiting the 5x5 grid, the Mega Jackpot is awarded. The PICK BONUS FEATURE ends when Mega Jackpot is awarded, or selection of pick coins resulted in passing the border of the 5x5 grid. All the wins collected during the PICK BONUS FEATURE are summed up and presented as the total win at the end of the feature and added to base game win, if any. MEGA, GRAND, MAJOR AND MINOR JACKPOTS Grand, Major, Minor Jackpots are available in the base game and the BONUS SPINS FEATURE. Mega Jackpot is available only in the PICK BONUS FEATURE, along with all the other Jackpots.
