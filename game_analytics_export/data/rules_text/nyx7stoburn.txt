7's To Burn Game Introduction 7's To Burn is the hot new three reel slot game that features Hi Roller spins. How to play Select your stake If you wish to adjust this, click the STAKE '+' and '-' buttons to increase or decrease the size of your stake. Spin the reels Click the SPIN button to start your game. Info All line win amounts are shown in the games pay table as multiples of the total stake. These can be viewed by pressing the INFO button and are displayed as cash values in the game which change as the stake per line is adjusted. Winlines The default win line selection is 5 lines which is fixed for every game. Base Game Awards The awards can be viewed in the game using the Info button. Any wins will be automatically credited to your account. During the base game burning 7's are wild and substitute for red 7's. Hi Roller Pressing the Hi Roller button in the game will bring up a box allowing you to increase or decrease your stake to play 5 Hi Roller games. During the Hi Roller games the Hot 7's substitute for all symbols. The theoretical average return to player (RTP) is 95.10%. This RTP represents the long-term expected payback of the game which has been calculated and verified by an independent testing company. Malfunction voids all pays and play.
