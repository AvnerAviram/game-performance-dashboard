Bufón Fortuna GAME RULES Bufón Fortuna is a 5-reel, 3-row video slot. The game is played with 5 paylines. All reels are in play at all times. The Total Bet is set using the bet selector. Autoplay automatically plays the game for the desired number of rounds. Winning combinations and payouts are made according to the Paytable. Increasing and decreasing your Total Bet will result in the Paytable adjusting by the same proportion. All win values shown are in the currency wagered. Only the highest win per bet line is paid. The bet lines or ways wins contain one symbol from each adjacent reel, starting with the leftmost reel. Bet line wins pay in order from the leftmost to the rightmost reel. Simultaneous wins on different bet lines are added. All wins pay on bet lines only, except for BONUS, where 3 or more symbols pay in any position. Return to Player The theoretical return to player for this game is 94.35%. HOW TO PLAY Spin / Stop Autoplay Opens the autoplay configuration. Turbo Mode Enable / Disable the quick spin (also available in autoplay popup). Bet - / + Increases or decreases the Total Bet amount. WIN Displays the win for the current or last win payout step. CASH Displays the player's current balance. PAYTABLE Opens the info and paytable screens. Sound Enables or mutes the sound. Home Directs the player to the website's home URL. GAME RULES Opens the game rules. Game History Opens your rounds history. Autoplay Enables playing the game automatically for a predetermined number of rounds at the current Total Bet. How to use the autoplay feature Note. If you are disconnected while playing, all Autoplay settings will return to default when you reload the game. Autoplay Limits When the limit's conditions are met, the autoplay will stop and the relevant popup message will appear in the middle of the screen.
