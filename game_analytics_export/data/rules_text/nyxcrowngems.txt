Crown Gems Game Introduction Crown Gems is a regal themed five reel slot game offering some majestic wins and the chance of winning big! There are 10 win lines with every one paying for 3 or more adjacent symbols in any position on a win line. How to play Select your stake - If you wish to adjust this, click the STAKE/LINE '+' and '-' buttons to increase or decrease the size of your stake per line. Spin the reels - Click the SPIN button to start your game. There is an auto play function available that will start each spin automatically until either something occurs that requires your intervention or the number of set spins expires. Select your desired number of spins by clicking on the numbered buttons that are revealed when you press AUTOPLAY. Click 'Stop' to cancel auto play. Help All line win amounts are shown in the games pay table as multiples of the stake per line. These can be viewed by pressing the HELP button and are displayed as cash values in the game which change as the stake per line is adjusted. The minimum average return to player is is 95.05%. The theoretical return for the Hi Roller game is 96.08%. This RTP represents the long-term expected payback of the game which has been calculated and verified by an independent testing company. Malfunction voids all pays and play. The maximum win from a single game is £250,000
