    How to Play

    Note: If you do not have enough money to play a game, the bet can be set to the minimum selectable amount through a dialog window informing about this.

    All winning lines pay from left to right. Wins on different selected lines are all added. Only the highest winning combination on each selected line is paid. Winnings from Scatter and free spins are added to the total win.

    Flash version

    Note: On the info bar located under the main game buttons, you can view the current BET, your BALANCE and your current WINNINGS, in your currency.

    To play, follow these steps:

    1. Use the coin value select button to choose a coin value. The coin value select button is located in the bottom-left side of the main game screen and shows the currently selected coin value.

      Note: The current bet amount, in your currency, is displayed in the BET box.

    2. Click SPIN to place the bet and spin the reels once. You can click the BET MAX button to place the maximum bet for the currently selected coin value and spin the reels once.

    The total amount won, in your currency, according to the paytable, is shown in the WINNINGS box. For each winning combination, the amount won is the amount shown in the paytable for that winning combination.

    Special Symbols and Features

    Wild

    The Wild symbol replaces any other symbols except Scatter.

    During each free spins round, at random, two or more symbols, transform into Wild.

    Scatter

    Scatters pay anywhere on the reels, once per spin. Scatter winnings are shown in the paytable.

    Each Scatter symbol that appears on the reels is collected. When you collect five or more Scatter symbols, you win a number of free spins that is equal to the number of collected Scatter symbols.

    You can view the current number of collected Scatter symbols, in the Flash version, next to the BET MAX button and in the Mobile version, under the Spin button.

    Note: For the Flash version: when you win free spins, click START to begin playing the free spins rounds; at the end of the free spins, click RESUME to return to the main game screen.

    During free spins, each Scatter symbol that appears anywhere on the reels, awards an additional free spin.

    The number of collected Scatter symbols is reset to zero after the free spins end and you can start collecting again. The number of collected Scatter symbols is also reset to zero if, before you collect the necessary minimum number of Scatter symbols to win free spins, you change the coin value.

    Autospin feature

    The loss limit setting allows stopping the autospin feature automatically when the selected limit plus your cumulative winnings minus your cumulative bets is less than 0.

    Note: The autospin feature does not automatically start free spins - player input is required.

    Flash version

    Click the AUTO SPIN button to open the AUTO SPIN SETTINGS window where you can set up the autospin feature.

    On the AUTO SPIN SETTINGS window, you can set up the following:

    • Loss limit - enter a number for the loss limit value or use the left and the right arrow buttons to select a number for the loss limit value

    • Stop if WIN reaches - to enable this setting, click the button on its left side and then enter a number for the WIN value or use the left and the right arrow buttons to select a number for the WIN value; click the button on its left side again to disable it

    • Stop if JACKPOT is won - (only for games with Progressive Jackpots or with a specific Jackpot value displayed) - to enable this setting, click the button on its left side; click the button on its left side again to disable it

    • Number of AUTO SPINS to play - use the left and the right arrow buttons to select the number of autospins

    Click START to close the AUTO SPIN SETTINGS window and start the autospin feature with the currently selected bet and number of autospins or click CANCEL to close the AUTO SPIN SETTINGS window without starting the autospin feature.

    Click the STOP xx/yy button to stop the autospin feature at any time; xx represents the remaining number of autospins and yy represents the number of autospins to play that you selected on the AUTO SPINS SETTINGS window.

    Other game buttons

    Flash version

    The following buttons are available:

    • PAYTABLE - click to open the paytable where you can view:

      • information about the payouts for the winning combinations, in your currency

      • information about any existing special features or special symbols and their payouts or specific awards

      • the available keyboard shortcuts, the game lines and short game rules

    • STOP - (available only while the reels are spinning) click this button to stop the reels faster
    • SKIP - (available during winning animations) click this button to skip winning animations
    • settings window button - in the top-right corner of the main game screen, click the settings button (list symbol) to open the Settings window where you can manage sound volumes, view gameplay history or view detailed game rules

      Note: To close the Settings window you can click anywhere on the main game screen, or, in the top-left corner of the Settings window, click the return button.

    • sound button - in the Settings window, click SOUND to open the sounds and volume panel where you can use sliders to manage Winning, Game and Background sounds or the general Volume; click SAVE to save any changes you make

      Note: To quickly mute or unmute all game sounds, in the Settings window, click the button displayed on the right side of SOUND.

    • rules button - in the Settings window, click RULES to view detailed game rules in a new browser window

    • history button - (in real mode only) in the Settings window, click HISTORY to view the gameplay history

    • full screen button - in the Settings window, click FULLSCREEN to go to full screen mode; to exit full screen, click the button again or press ESC on your keyboard

    • BACK - when you are in the paytable, click BACK to return to the main game screen
