    Gus's Gold Mines

    1. Overview

    The goal is to find Bet Multipliers before uncovering a Montezuma Mine, which will end the game round.

    Game specifications:

    Type

    Mines

    Grid

    5 x 5 grid (25 squares)

    Winning symbol

    Gold Nugget

    Losing symbol

    TNT Mine

    The game disconnects if you do not take any actions for a specific number of consecutive minutes (this number varies depending on jurisdiction); the session recovery feature will save the game progress so that the next time the game is loaded, you can continue where you left off.

    In the event of technical difficulties resulting in a dropped connection to the game server, the game you were playing is saved automatically and will load the next time you establish a connection to the game server. You will not be able to reconnect to the game server for the first 30 seconds after the disconnection.

    Note: In case of malfunction all pays and play are voided; any bet that was not yet accepted will be voided, any bet that was unsettled will be refunded.

    2. How to Play

    Note: If you do not have enough money to play a game, the bet can be set to the minimum selectable amount through a dialog window informing about this.

    On the info bar, you can view the current BET, your BALANCE and your current WIN, displayed in either credit or your real currency, depending on the current user's choice (Available only for specific jurisdictions). To toggle between credit or your real currency, click in the BALANCE area (Available only for specific jurisdictions).

    2.1. Desktop version

    To play, follow these steps:

    1. Use the upper + and - buttons to select a coin value.

      Note: The total BET is displayed in either credit or your real currency, depending on the current user's choice (Available only for specific jurisdictions). To toggle between credit or your real currency, click in the BALANCE area (Available only for specific jurisdictions).

    2. Use the lower + and - buttons to select the number of mines.

      Note: The more mines you choose, the faster your winnings can grow - but gold is harder to find.

    3. Click PLAY! to start and choose a square to uncover.

      Note: Each Bet Multiplier found will increase your prize, a Montezuma Mine will end the game round.

    4. Click CASH OUT to end the round and cash out.

    To toggle between credit or your real currency, tap in the BALANCE area (Available only for specific jurisdictions). 

    3. Special Symbols and Features

    3.1. Cash Out

    Shows the current value of your prize, which you can choose to collect (Cash Out).

    After a Bet Multiplier is found, this value will increase and you can Cash Out after any successful pick.

    3.2. Remaining Picks

    Shows the number of remaining picks you are allowed to make.

    Depending on the number of Mines, you will have a set number of initial picks available.

    After each Bet Multiplier is found, the number of remaining picks will count down by one.

    Note: If remaining picks run out, you will have reached the Maximum Win.

    3.3. Maximum Win

    Shows the maximum possible win you can reach with the current grid.

    Maximum Win is dependent on the initial bet and the number of starting Mines.

    Note: If Maximum Win is reached, no more picks are available and you must Cash Out.

    3.4. Payouts

    The payout for the game is determined by a bet multiplier which increases after each successful pick.

    The multiplier is dependent on the number of winning picks found and the number of Montezuma Mines.

    3.5. Next Win

    Shows the value of the next Bet Multiplier if you choose to continue the round.

    3.6. Found

    Shows how many Bet Multipliers have been found from the total number on the grid.
